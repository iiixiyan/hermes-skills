Day11文档使用6模型结构（3个二维DP + 3个背包DP），每道真题精确映射到一个模型：

| 模型分组 | 模型名称 | 状态定义 | OD真题 | LeetCode配套 |

## 快速查询脚本

在cron模式下用inline Python快速获取指定子类型的OD真题：

```bash
cd /tmp/huawei-od-new-system && python3 -c "
import json
data = json.load(open('complete-data.json'))

# 一维DP关键词
one_d_titles = ['猴子爬山', '抢7游戏', '找终点', '跳格子', '最佳对手', 
                '工作安排', '限流', '调度', '充电桩']

for category, problems in data.items():
    for p in problems:
        title = p['title']
        tags = p.get('topics', [])
        if isinstance(tags, str): tags = [tags]
        if '动态规划' not in ' '.join(tags):
            continue
        # 按标题关键词判断子类型
        if any(t in title for t in one_d_titles):
            print(f'[一维] [{category}] {title}')
        elif any(t in title for t in ['字符串', '最短路径', '查找', '公共', '园区', '矩阵']):
            print(f'[二维] [{category}] {title}')
        elif any(t in title for t in ['MELON', '报酬', '磁盘', '背包', '数列']):
            print(f'[背包] [{category}] {title}')
        elif any(t in title for t in ['WonderLand', '士兵', '伐木']):
            print(f'[树状] [{category}] {title}')
        else:
            print(f'[未知] [{category}] {title}')
"
```
