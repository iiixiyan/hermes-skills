## 一、API类（可编程接入，无需浏览器）

### 1️⃣ 新浪竞彩API ⭐⭐⭐⭐⭐
| 项目 | 内容 |
|:----|:-----|
| **接口地址** | `https://mix.lottery.sina.com.cn/gateway/index/entry?format=json&__caller__=wap&__version__=1.0.0&__verno__=10000` |
| **必要参数** | `__caller__=wap&__version__=1.0.0&__verno__=10000` **缺一不可** |
| **欧赔** | `cat1=footballMatchOddsEuro&matchId=xxx` → **53家赔率公司**初赔+即赔 |
| **亚盘** | `cat1=footballMatchOddsAsia&matchId=xxx` → **17家公司**盘口+水位 |
| **竞彩SP** | `cat1=jczqMatches&date=YYYY-MM-DD` → 胜平负SP/让球SP/比分SP/半全场/进球数 |
| **欧赔变化** | `cat1=footballMatchOddsEuroChange&matchId=xxx&companyId=9&offerId=xxx` → 时间序列(155条+) |
| **比赛详情** | `cat1=footballMatchDetail&matchId=xxx` → 排名/天气/轮次/中立场地 |
| **技术统计** | `cat1=footballMatchStat&matchId=xxx` → 控球率/射门等 |
| **格式** | 纯净JSON，无防盗链 ✅ |
| **速度** | ⚡ 即时响应(1-3s) |
| **状态** | ✅ 已验证可用 |

**⚠️ 2026-06-19 数据格式变化**：`result.data` 现在是**list**（之前是dict），每项含matchId/homeTeamName/guestTeamName/spf/rqspf等字段。

**⚠️ 静默空数据陷阱**：缺`__version__=1.0.0` → API返回空`result.data=[]`，无报错！

### 最佳管道推荐

```
赔率数据 → 新浪竞彩API（主）+ 500彩票（补充亚盘/大小）
基本面   → 天天盈球（阵容/伤停）+ 新浪API(排名/天气)
赛果验算 → 新浪API + 竞彩官方API（需WAF绕过）
北单独有 → 59itou API (lotteryId=45)
```
