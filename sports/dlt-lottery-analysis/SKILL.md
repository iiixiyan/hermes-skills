## 八、思维模型 — 第一性原理与对抗式审查（DLT专修版）

> 以下两种思维模型是贯穿所有DLT预测与复盘的最高层方法论。每次预测前调用，作为认知框架的初始化。

### 8.1 概述

大乐透35选5+12选2的随机空间比双色球更大（总组合数≈2142万 vs 1772万），意味着相同数量的注数覆盖比例更低。第一性原理和对抗式审查对DLT尤其重要——因为它更容易陷入"覆盖面大所以应该中"的幻觉。

- **第一性原理** —— 从35选5+12选2的数学公理出发
- **对抗式审查** —— 挑战"5注覆盖25码"与"单注命中"之间的巨大鸿沟

### 8.2 第一性原理（First Principles）

#### 8.2.1 五条基本公理（与SSQ一致，但DLT版本）

| # | 公理 | DLT数学事实 | 影响 |
|:-:|:-----|:------------|:-----|
| 1️⃣ | **每期独立** | 上期05,13,22,26,32 → 下期每个号概率仍1/35 | 重号×1.3仅为覆盖策略，非预测 |
| 2️⃣ | **等概率性** | C(35,5)×C(12,2)=21425712种组合概率相等 | 所有选号方法的中奖概率完全相同 |
| 3️⃣ | **无记忆性** | 遗漏N期的号和刚出现过的号，下期概率相同 | 区间补缺(73%)是统计观察，非数学规律 |
| 4️⃣ | **覆盖 > 精度** | 5注25码覆盖=71.4%；8码大底=22.9% | 5注独立比8+3大底有更高的命中潜力 |
| 5️⃣ | **过拟合必败** | 35选5的噪声信号比更严重（更稀疏的数据） | 权重调整必须交叉验证 |

#### 8.2.3 5注独立 vs 8+3大底的第一性原理对比

| 维度 | 5注独立 | 8+3大底(168注) |
|:----|:-------|:--------------|
| 前区覆盖 | **25/35=71.4%** | 8/35=22.9% |
| 单注数量 | 5注 | 168注 |
| 成本 | 10元 | 336元 |
| 第一性原理评估 | **覆盖广度优先**：更多不同号码，更大概率含中奖号 | **注数深度优先**：更多组合覆盖，更高概率中奖 |

**第一性原理结论：**
- 如果追求「含中奖号」的概率 → 5注独立（71.4%覆盖）
- 如果追求「中奖」的概率 → 两者都不显著（即便是168注，一等奖概率也仅为168/21425712≈0.00078%）
- 真正的最优：如果预算固定，**5注独立是极高性价比的覆盖方案**（10元覆盖71.4%的号码空间）

#### 8.2.4 后区第一性原理

| 问题 | 答案 |
|:----|:-----|
| 5注后区10对能覆盖多少组合？ | 10/66≈15.2%（前提是全部不重复） |
| 后区一大一小+一奇一偶的占比？ | 约63%（12选2中满足条件的组合数） |
| 10期后区全中的概率？ | 约15%（不奇迹，但也不常见） |
| 后区邻号×1.3的真实价值？ | 邻号覆盖64个后区组合中的24个≈36%，×1.3使邻号在评分中优先但不会独占 |

**关键洞察**: 5注后区10对的理论最大覆盖是15.2%，但实际5注独立生成中后区难免有重复（因每注独立选号）。v7.3.2将后区命中率从90%→100%并非算法革命，而是从硬约束(漏掉16%组合)到软约束(更广泛覆盖)。

### 8.3 对抗式审查（Adversarial Review）— DLT专修

#### 8.3.1 DLT特有认知偏差对抗

| 你的判断 | 对抗式挑战 | 回应 |
|:---------|:----------|:-----|
| "8+3大底命中率应该更高" | **数学计算**：8+3只有168注，5注独立确实覆盖更少组合(5 vs 168)但是覆盖的号码更多(25 vs 8)。**开奖是5个号不是所有号** — 8码大底只有8个不同的数字，5注独立有25个不同数字。25个数字包含任意5个开奖号的概率>8个数字包含的概率 | 5注独立：覆盖广度胜出 |
| "200期回测≥2=71%很稳" | **随机基线**：5组随机选号（每次5注）在200期中的≥2率是多少？如果系统算法的71%和随机选号接近，说明方法没有增量价值 | DLT系统≥2=71% vs 随机基线≈**待测试** |
| "重号×1.3让命中率提升了" | **隔离验证**：移除重号增强后200期≥2率会降到多少？如果下降<1%，说明重号增强不是核心杠杆 | 重号增强Δ≈1%（已有回测数据：70%→71%） |
| "区间补缺×1.5是最强信号" | **条件概率**：上期空区→下期补号的73%概率，在完全随机数列中出现频率是多少？73%是否显著高于随机期望？ | 35选5的区间补缺≈73% vs 随机**接近，须验证** |
| "我发现了新规律" | **样本量检查**：这个"规律"在多少期内验证？≥200期？还是只在最近10期有效？ | 任何<200期的"发现"应视为短期巧合 |

#### 8.3.2 对抗式审查四问（每次预测前强制执行）

```markdown
🔍 对抗式审查（DLT版）—
1. **覆盖 vs 命中幻觉**：我的5注确实覆盖了25/35的号码，但这5注本身是否有效地覆盖了中奖所需的结构？（5注各自的5个号内部结构是否合理——奇偶/大小/区间/AC值？）

2. **8+3大底 vs 5注独立的选择逻辑**：我今天用哪种模式，依据是什么？
   - 覆盖优先 → 5注独立（追求"含中奖号"）
   - 组合优先 → 8+3大底（追求"中奖"，预算允许时）

3. **后区选号是否陷入单一策略？**
   （5注后区应覆盖至少3种来源：邻号/步长/冷号/热号）

4. **今天预测比完全随机好在哪里？**
   （如果无法清晰回答"系统方法 vs 随机选号的差值在哪"，说明需要回归第一性原理）
```

```
### 10.3 新增铁律

#### ① 重号保留铁律（防排除上期号）
> 上期所有前区号码中，**必须至少保留1个**在5注中。不得因调整大号/区间均衡而完全排除上期所有号码。

**执行方式**：选号先锁定上期1-2个重号，再围绕它们构建5注。重号优先保留在注①或注②。

#### ② 大号覆盖上限（防30+过度）
> 30-35大号区**最多覆盖2注**（非每注），其余3注应在01-29范围内合理分布。

**原因**：每期开奖最大号≤29的期次约占**40%**，强塞30+到每注大幅降低了命中概率。

#### ③ 后区覆盖多样性（防重复浪费）
> 5注后区10对号码中，**必须≥6个不重复的号码**（即允许最多4个重复对，而非8个重复）。后区候选池应在8个不同号码以上。

**示例**：
- ✅ 好: 02/07, 04/11, 03/08, 05/09, 01/10 → 10个不重复号
- ❌ 差: 02/11, 03/11, 04/11, 08/11, 02/08 → 11出现4次，仅5个不重复号

#### ④ 前区旋转集中规则（防散射）
> 5注的25码覆盖中，核心命中号码（即预测命中率最高的3-5个号）必须**集中在同一注中至少出现2个**，而非分散到不同注。

**执行方式**：
1. 从25码中标记TOP5最强信号号
2. 将这5个号尽可能分配到1-2注中（每注至少2个）
3. 剩余号再填充其余注的剩余位置

### 10.4 偏态回归核心号选取铁律（v8.7.0 新增·26076期复盘驱动）

> 源自26076期复盘：前区5/5+后区2/2全覆盖但0注中奖。根因：26075为4小1大+1奇4偶极端偏态，核心号应从"大号+奇号"回归方向选，但执行时选了"区间补缺"方向(02,08,22,29)，核心号方向错误导致散射。

| # | 偏态类型 | 判定条件 | 核心号方向 | 案例 |
|:-:|:--------|:---------|:----------|:----|
| 1️⃣ | **大小偏态** | 上期大小比(以18为界) ≥4:1 或 ≤1:4 | → **选反方向**（大偏态选小、小偏态选大） | 26075=4小1大 → 核心号选**大号**(15,20,27,28,35) |
| 2️⃣ | **奇偶偏态** | 上期奇偶比 ≥4:1 或 ≤1:4 | → **选反方向** | 26075=1奇4偶 → 核心号选**奇号** |
| 3️⃣ | **区间偏态** | 某区间(Z1-Z5)连续≥3期空窗 | → **优先补该区间**（30%权重） | Z5连续2期空窗 → 补Z5(35✅) |

**执行优先级（覆盖§10.3-④旋转集中）：**

```
Step 1: 检查上期是否有极端偏态(大小≥4:1或奇偶≥4:1)
  有 → 核心号从回归方向选取（70%权重）
  无 → 核心号从区间补缺方向选取（70%权重）
Step 2: 区间补缺 → 辅助判断（30%权重）
Step 3: 重号保留 → 至少1个上期号码
Step 4: 核心TOP3-5号集中在1-2注中
```

**26076回测验证**：若按此规则，核心号从"大号+奇号"方向选3-5个(15,20,27,28,35)，集中在同一注→至少前区≥3命中，搭配后区02+11→中奖。

### 10.5 检查清单（每次预测前执行）

```
□ 偏态回归: 上期有无大小/奇偶极端偏态？核心号方向是否正确？
