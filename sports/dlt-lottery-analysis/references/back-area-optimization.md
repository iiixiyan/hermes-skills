## 实现方式

### 评分层（`get_back_scores()`）
统计遗漏值+频率分布，基础评分。

### 邻号加权（v7.3.2新增）
```python
if self.N >= 1:
    prev_back = [int(x) for x in self.data[-1]['back']]
    for n in prev_back:
        bs[n] = bs.get(n, 0) * 1.3           # 后区重号
        for d in [-1, 1]:
            nn = n + d
            if 1 <= nn <= 12:
                bs[nn] = bs.get(nn, 0) * 1.3  # 后区邻号
```

### 配对层（`is_valid_back_pair()` → 5注）
仅保留和值6-18范围，其余全部放宽。奇偶搭配和大小搭配通过`pair_freq`评分自然体现。

### 直选层（`get_83()` → 8+3）
TOP3评分直选。8+3后区命中仅30%——3码空间太小，建议以后区5注推荐为主。
