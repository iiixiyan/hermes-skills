# 5注策略设计原理 v5.0（正式版）

## 为什么5注比1注好

大乐透35选5的随机命中率只有0.71个/期。单注再优化也难大幅超越随机。
5注策略通过 **多策略分散 + 号码覆盖最大化 + 11大技巧集成** 实现更高命中概率。

## 200期回测最佳结果

| 指标 | 值 | vs随机 |
|:----|:---:|:-----:|
| 最佳平均命中 | **1.68个** | +136% |
| ≥2号率 | **59.0%** | +372% |
| ≥3号率 | **9.0%** | +572% |
| 后区≥1号 | **93.0%** | +200% |

## 5种策略设计

### ❶ 高分优先 — 稳健型
按18+维度综合评分选出最优5个号，全维度评分覆盖所有指标。
- 候选池: 全35个号中的Top15
- 热温冷控制: 3热+1温+1冷
- 200期单注≥2: 13.5%

### ❷ 热号追击 — 进攻型
只从热度TOP12+活跃号+转换升温号里选，放弃冷号。
- 追热不追冷，"热号惯性"现象
- 排除❶已选号，保证覆盖分散
- 200期单注≥2: 12.5%

### ❸ 冷号博冷 — 博冷型
遗漏最久的号码+遗漏压力高分号+间隔超期号+复苏号。
- 基于"均值回归"原理
- 指数加权: 10期1.0→40期5.0+
- 排除❶❷已选号
- 200期单注≥2: 11.0%

### ❹ 缩水追击 — 趋势型
5期缩水池(87.2%理论覆盖)+尾数推荐+连号对+上期参考混合。
- 补缺号区号码
- 排除前3注已选号
- 200期单注≥2: 16.0%

### ❺ 覆盖补漏 — 全覆盖
前4注没覆盖到的号码按评分高低选5个。
- 最大化总覆盖（可达25+/35个号 ≈ 71%）
- 深冷号+间隔超期偏重
- 确保冷门号码不被遗漏
- 200期单注≥2: 13.5%

## 关键参数

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| period | 100期 | 热度和基础分析窗口 |
| 热温冷比 | 3:1:1 | 每注3热+1温+1冷 |
| 后区双池 | Top6高分 + Bottom6冷号 | 最大化后区覆盖 |
| 深冷阈值 | 20期+ | 指数加权起点 |
