# 大乐透11大实战投注技巧详解（v5.0）

## ➊ 间隔状态选号

**原理:** 每个号码的出现间隔存在一定周期规律。不是简单地看"遗漏了多少期"，而是看"当前遗漏与历史平均间隔的关系"。

**实现:**
- 对每个号码，收集其历史所有出现位置，计算间隔序列
- 平均间隔 = sum(间隔) / 出现次数
- 标准差 = sqrt(var(间隔))
- 超期评分 = max(0, (当前间隔 - 平均间隔) / 标准差)，上限3.0
- 分数越高表示该号越"已超期"(超过其历史正常周期)

**与普通遗漏的差异：**
- 遗漏：只看全局平均，"20期未出=一样冷"
- 间隔：每个号码有独立的周期，"20期对均隔15的号=已超期，对均隔30的号=正常"

## ➋ 冷热转换选号

**原理:** "强者恒强"或"均值回归"都不是永久的。号码在热/温/冷之间的转换信号很重要。

**实现:**
- 短窗口(近10期)频率 vs 长窗口(近50期)频率/5(归一化)
- 5种信号：
  - **冷→热(强)**: 短窗≥2次 & 长窗归一化≤0.5 → 信号+3.5
  - **升温中**: 短窗显著 > 长窗 → 信号+2.5
  - **复苏初期**: 短窗≥1 & 长窗归一化≤0.3 → 信号+3.0
  - **持续热**: 短窗≥3 & 长窗归一化≥2 → 信号+1.5
  - **降温**: 短窗0次 & 长窗≥3 → 信号-0.5

**实战用法：** "升温"和"复苏"是最强信号，代表号码正在受到"关注"，有动量。

## ➌ 上期参考选号

**原理:** 开奖号码之间存在明显的连续关系。上期号码影响下期。

**实现(多维度叠加):**

| 关系 | 得分 | 说明 |
|:----|:---:|:-----|
| 重号 | 3.0 | 上期开出的号继续出现 |
| 邻号(±1) | 2.0 | 上期号码的邻居 |
| 隔号(±2) | 1.2 | 隔一个号的跳号 |
| 钟摆(±5) | 0.8 | 大幅摆动对称 |
| 上上期衰减 | 0.6 | 上期没出的上上期号码 |
| 10期高频(≥3) | 1.0 | 近期多次出现的号 |

**统计验证(近200期):**
- 重号出现率: 53.3%
- 邻号(±1)出现率: 73.9%
- 上期参考综合评分是5注选号的重要输入

## ➍ 多期参考选号（缩水池）

**原理:** 近5期的开奖号码，去重后作为下一期候选池。这是一种经过实战验证的缩水方法。

**统计验证(近200期):**

| 池 | 平均大小 | ≥2个中奖号 | ≥3个中奖号 |
|:--|:-------:|:---------:|:---------:|
| **5期池** | **18.6个** | **87.2%** | **60.5%** |
| 10期池 | 27.1个 | 95.9% | 92.8% |

**实战用法：**
- 5期池作为主要候选池(18-19个号, 87%命中)
- 10期池作为补充(95.9%命中)
- 缺号区从池外补充

## ➎ 相同尾数选号

**原理:** 尾数(个位数)相同的号码常一起出现。每期通常1-2对同尾，极少3+。同尾通常一大一小(0-4小, 5-9大)。

**实现:**
- 分析近50期尾数对出现频率
- 追踪大尾(5-9)和小尾(0-4)的平衡
- 推荐高频配对的尾数组合
- 选号时偏好推荐尾数对应的号码

**统计:**
- 约80%期次有至少1对同尾号
- 约30%期次有2对同尾
- 同尾出现时，大小尾混合占多数(~70%)

---

## ➏ 缩小选号范围

**原理:** 综合多期缩水 + 区间分布 + 遗漏逻辑，将35选5简化为小型乐透。

**实现:**
- **5期去重池**: 35→~19个号 (87.2%含≥2)
- **缺号区补号**: 池中缺号的区域从池外选遗漏最久的号
- **10期池备用**: 当5期池候选不足时补充

**实测效果：**
- 5期池+缺号补 → 缩小至~22个号
- 覆盖87.2%的≥2命中场景
- 为5注策略提供了可靠的起始候选

---

## ➐ 连码重码

**原理:** 连号(号码连续)和重码(与上期重复)是大乐透最常见的模式。连号出现率约49%，重号出现率约53%。

**实现:**
- **连号对追踪**: 近50期高频连号对出现频率
- **连号缺失检测**: 已几期无连号 → 强制推荐连号
- **重码追踪**: 哪些号码在相邻期次重复出现
- **选号强制**: 每注优先包含连号对成员

**实战统计:**
| 特征 | 出现率 |
|:----|:-----:|
| 至少1对连号 | 49.2% (近200期) |
| 至少1个重号 | 53.3% |
| 邻号(±1) | 73.9% |
| 3期无连号后强制推连号 | 84.6%命中 |

## ➑ 冷码必不可少

**原理:** 冷码(遗漏10+期的号码)虽然出现频率低，但一旦开出可能带来高回报，且长期必然回归。

**实现(指数加权):**

| 遗漏期数 | 冷号指数 | 说明 |
|:-------:|:--------:|:-----|
| 7-9期 | 0.5 | 温冷区 |
| 10-14期 | 1.0 | 基本冷号 |
| 15-19期 | 1.5 | 深冷区 |
| 20-29期 | 2.0 + (n-20)*0.15 | 加速加权 |
| 30-39期 | 3.5 + (n-30)*0.15 | 高爆发分 |
| 40+期 | 5.0 + (n-40)*0.15 | 极高潜力 |

**实战原则：**
- 每期5注总体**确保1-3个冷码** 
- 深冷号(≥20期)在3注以上出现
- 冷号指数≥2.0的号码列入"必选"候选

## ➒ 热温冷3:1:1搭配

**原理:** 号码按近30期出现频率分为热号(近10期≥3次或近5期≥2次)、温号(近20期出现过但不够热)、冷号(近20期未出现)。每个5号注采用3热+1温+1冷的平衡布局。

**实现:**
1. 从候选池中分离热/温/冷三类号码
2. 优先选热号(最多3个)—保证命中基础
3. 选1个温号作为过渡—既不盲追热也不赌博冷
4. 选1个冷号作为"杀招"—一旦开出大幅提升价值

**后区热温冷:** 后区同样采用1热+1冷或1热+1温搭配。

**统计验证:**
- 热号中奖概率最高(动量持续)
- 冷号一旦开出回报最大
- 纯粹追热或纯粹赌冷都效果不佳
- 3:1:1在回测中是最优搭配比例

## ➓ 结伴号码

**原理:** 彩票中存在"结伴效应"—某些号码经常在上期出现后，下期跟随出现特定号码。例如上期出6号下期经常出5号，上期出20号下期经常出21号。

**实现:**
- 扫描全部历史数据，对每个上期号码统计下期跟随号码的频率
- `companion[上期号][下期号] += 1`
- 统计历史中出现≥3次的跟随关系，作为"结伴信号"
- 选号时对结伴关系中的号码额外加分(+0.5)

**典型结伴示例:**
- 前区: 6→5, 20→21 (非固定，随时间变化)
- 后区: 3→10, 4→8, 5→12 (常见配对)

**注意:** 结伴关系随数据更新而动态变化，不可一概而论。

## ⓫ 斜连号/坐号

**原理:** 号码在图表走势中常出现斜线连接和坐号重复。

**斜连号:** 跨期号码递增/递减形成斜线
- 斜二连: 上期出n-1 → 本期出n (信号+1.5)
- 斜三连: 前两期分别出n-2, n-1 → 本期出n (信号+3.0，强信号)

**坐号:** 同一号码连续2期以上出现
- 坐号(重号持续)信号+2.0

**实现:**
- 检查最近3期的号码，检测斜向关系
- 对符合斜连/坐号规律的号码评分奖励
- 斜连号在大乐透走势图中是常见模式

## 数学限制

8大技巧可以有效提升选号质量，但无法突破以下数学限制：

1. **5注×5号=25个号**: 5个中奖号散在25候选号中，分配到5组后≥2命中率上限≈94%
2. **35选5本质**: 即使评分再准，也无法让5个中奖号全部进入同一5号注
3. **≥4个命中**: 在35选5中，P(≥4)的理论最大值<1%
