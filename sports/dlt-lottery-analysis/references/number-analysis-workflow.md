# 大乐透逐号复盘分析 Workflow

## 适用场景
用户要求复盘某期大乐透开奖号码，分析每个号码与近5期/近N期的关系。

## 数据采集
1. 浏览器导航到 https://m.cjcp.cn/kaijiang/dlt/
2. 获取最新一期号码（页面顶部大号数字）
3. 用console提取历史列表数据
4. 构造近5期对照表

## 逐号分析格式

### 前区号码分析（每个号4步）
1. **近5期矩阵表**：该号在近5期是否出现
2. **分类判定**：
   - 🔥 **热号重号** = 上期直接重复
   - 🔄 **隔期回补** = 隔1-2期重复
   - 🧊 **冷号回补** = 3期以上空窗后回补
   - 🆕 **新号** = 近5期首次出现
   - 📍 **区间补缺** = 某区间连续空后回补
3. **邻号关系**：与近5期号码的差值关系
4. **区间归属**：一区(01-07) 二区(08-14) 三区(15-21) 四区(22-28) 五区(29-35)

### 后区号码分析
1. 近5期后区分布矩阵
2. 邻号/步长关系
3. 奇偶/大小分类

## 综合总结格式

| 项目 | 说明 |
|:----|:------|
| 奇偶比 | 前区奇数:偶数 |
| 大小比 | ≥18为大，<18为小 |
| 区间分布 | 5区间覆盖情况 |
| AC值 | 号码离散度 |
| 重号率 | 与上期重复号码占比 |
| 连号 | 是否有连号组合 |
| 编号类型分布 | 各号码的类型占比 |

## 输出格式
- 每个号码独立段落，用矩阵表展示近5期出现情况
- 最后综合模式总结表
- 使用emoji标记类型（🔴前区/🔵后区/🔥重号/🔄回补/🧊冷号/🆕新号）
