## 五、种(重号)+临号(邻号) 信号集成方法

在`get_voting_pool()`中，分散策略之后追加：

```python
# 种(重号)加权 — 上期号码追加分数
if len(self.data) >= 2:
    prev_nums = [int(x) for x in self.data[-1]['red']]
    for n in prev_nums:
        if n not in pool: pool[n] = {'score': 0, 'sources': []}
        pool[n]['score'] += 5
        pool[n]['sources'].append('种')
    # 临号(邻号)加权
    for pn in prev_nums:
        for d in [-2, -1, 1, 2]:
            nn = pn + d
            if 1 <= nn <= 33:
                bonus = 4 if abs(d) == 1 else 3
                if nn not in pool: pool[nn] = {'score': 0, 'sources': []}
                pool[nn]['score'] += bonus
                pool[nn]['sources'].append('邻')
```

来源字符串中会显示"种"和"邻"，方便追溯每注的号码来源。

## 八、核心教训

1. **永远先查覆盖盲区再动权重** — 分散区间缺6,15-19,26-28是100%的10期漏号根因
2. **种和临号是最强免费信号** — 不需要模型，仅凭上期数据就能抓64.7%的开奖号
3. **5法Union vs 5注是不同维度** — Union追求全面覆盖，5注追求单注组合最优
