# 梅花易数反推分析（2026-06-28）

对SSQ近10期(26063-26072)的梅花易数反推分析报告。

## 核心结论

### 1. 梅花易数有结构上限（5/6）
即使遍历所有64卦×6动爻=384种组合，最优预测命中率仅为4-5/6，从未达到6/6。
**根因**: 一区杀小留大(≥8)、二区上卦后部优先的约束导致部分号码不可能被预测到。

### 3. 最优卦象分布（10期中每种不同）
- 否#11动1: 2次
- 小过#61动2: 1次
- 坤#1动1: 1次
- 需#4动3: 1次
- 乾#0: 2次(动3/动6各1)
- 讼#5动6: 1次
- 渐#52动2: 1次
- 师#6动1: 1次

→ 没有固定最优卦象，每期不同。

### 4. 改进方向
1. **动爻计算优化**: 当前`(seed % 6) + 1`可改为遍历多个动爻取最优
2. **补号逻辑增强**: 补号从全池随机改为种号/临号补
3. **多卦合并**: 取TOP2-3卦象并集作为梅花大底（但测试显示多卦并集也无法突破5/6）
