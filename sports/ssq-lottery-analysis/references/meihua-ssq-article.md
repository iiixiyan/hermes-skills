## 文章作者方法

作者用**梅花易数**预测双色球第26066期（2026-06-11），2026年6月11日**酉时起卦**。

### 红球预测（分开起卦）

**步1：** 找太极点 = **离火**（主互变每卦都有离火 → 红球）
- 红球分三区：1-11(主卦)、12-22(互卦)、23-33(变卦)

**步2：主卦火水未济 → 一区(1-11)**
- 直读水数：1,6,11 → 杀6,1 → 取**11** ✅

**步3：互卦水火既济 → 二区(12-22)**
- 直读水数：16,21 → 下卦前/上卦后取**21** ✅

**步4：变卦火泽睽 → 三区(23-33)**
- 直读23 + 金数24,29 → 全取 **23/24/29** ✅✅✅
- 注意：兑为2，两个都要（此处易崴到）

**步5：错综卦补充**
- 综卦风火家人 → 离火直读 **5** ✅

**红球结果：5 + 11 + 21 + 23 + 24 + 29 = 全部正确！**

### 蓝球预测（另起一卦）

**起卦：** 变卦火风鼎 → 无坎卦（坎=蓝球太极点）
**错卦：** 水雷屯 → 找到坎水
**直读：** 坎下震上 → 出水数1,6,11,16 → 杀1,6(水在上卦) → 取 **11/16**
**实际：16 ✅**

## 核心算法总结（已实现到SSQ引擎）

1. **三卦→三区：** 主卦直读一区、互卦直读二区、变卦直读三区
2. **杀规则：** 下卦在前杀小，上卦在后取大
3. **错卦=五行全变：** 金克火、火克水、水克土、土克木、木克金
4. **综卦=上下颠倒：** 上卦成下卦，下卦成上卦
5. **蓝球错卦找坎水：** 无坎水则用变卦错卦的下卦五行
