# SSQ × DLT 同期关联分析报告

## 数据概况
- SSQ: 3469期 (2003-至今)
- DLT: 2889期 (2007-至今)
- 同期匹配(期号相同): 2883期
- 回测范围: 26022~26071 (近50期)

## ⚠️ 重要发现：期号≠同一天
SSQ（二四日开奖）与DLT（一三六开奖）**不在同一天开奖**，但使用**同一套期号系统**。
- SSQ 26072 (2026-06-25 周四) ↔ DLT 26071 (2026-06-27 周六)
- 差3-4天：DLT在同期号中晚于SSQ

## 🔥 核心发现

### 1. DLT前区→SSQ红球（0.80个/期）
**50期中62%的期至少有1个DLT前区号码出现在同期SSQ红球中。**
- 平均0.80个/期（DLT前区5个号，SSQ红球6个号）

### 2. 恒值对跨彩种（76%🔥）
**最强信号！50期中38期(76%)有SSQ恒值对的另一半出现在DLT数据中。**
- SSQ出12号 → 恒值对22号出现在DLT前区或后区
- SSQ出5号 → 恒值对29号出现在DLT
- 可用于反向推导：已知DLT前区号码→找其恒值配对→加入SSQ候选池

### 3. DLT前1-3期→SSQ红球隔期命中（0.81-0.91个/期）
DLT前1-3期号码平均每期有0.81-0.91个出现在当前SSQ红球中 → 不亚于SSQ自身重号率。

### 4. DLT后区→SSQ蓝球关联（24%相邻+12%同号=36%）
- SSQ蓝球与DLT后区同号: 6/50 (12%)
- SSQ蓝球与DLT后区相邻(±1): 12/50 (24%)
- 上期DLT后区→本期SSQ蓝球相同或相邻: 17/49 (35%)

### 5. DLT后区大小→SSQ蓝球（56%）
- SSQ蓝球与DLT后区首号大小一致: 56% 🔥
- SSQ蓝球与DLT后区首号奇偶一致: 48% (接近随机)

### 6. DLT断区预测（输入方向）
- DLT缺1区(01-12)时: SSQ一区40%/二区30%/三区30%
- DLT缺3区(25-35)时: SSQ一区28%/二区39%/三区33%

## 技术实现

### `_get_dlt_cross_candidates(ssq_issue)` 方法
输入SSQ期号，返回DLT衍生的候选集：
1. **最新可用DLT前区 +3分** — 50期回测62%期有效
2. **DLT前区恒值配对 +2分** — 76%期有效🔥
3. **DLT前1-3期前区 +2分** — 0.81-0.91个/期
4. **DLT后区±1邻号 +1分** — 蓝球参考

### 蓝球增强 (get_5sets)
- DLT后区同号/邻号作为注3优先选择
- DLT上期后区也纳入候选

## 集成结果
- 50期回测: 平均最佳红球2.24个, ≥3红28.0% (vs 之前27.0%)
- 红球覆盖率91%不变
- 蓝球命中率略有下降 (38%→28%)，需进一步调优

## 局限性
1. 最新SSQ开奖时对应DLT可能还未开奖 → 只能用上期DLT数据
2. DLT数据更新滞后（DLT数据比SSQ晚3-4天）
3. 如果DLT数据未加载，功能自动降级（不报错）
