# 恒值号过滤法 — 双色球红球筛选

**来源：** 知乎专栏《恒值号》(https://zhuanlan.zhihu.com/p/716124038) | **集成时间：** 2026-06-23 | **对应skill：** ssq-lottery-analysis v1.2.0

## 核心概念

红球号码范围 01-33，将最小号01和最大号33相加得34。依此类推，每两个号码组合的和都是34，称为**恒值号**。

## 应用规则

### 规则1：排除成对出现
单组恒值号的两个号码极少同时出现在同一期开奖中。选号时如果一组恒值号的其中一个已被选中，**不建议再选其配对号**。

### 规则2：近期出现压制
当某组恒值号的1个号码在**近10期内**出现过时，其配对号在当前期出现的概率降低。用作排除号。

### 规则3：区间覆盖检查
恒值号可用来检查红球在小号区(01-11)和大号区(23-33)的平衡：
- 如果已选号集中在01-11，则25-33区间应有对应恒值号配对
- 恒值号配对确保大小球区间的自然平衡

## 实战用法

### 选号后的过滤流程
```
1. 用热度/趋势/遗漏等方法初选6-8个红球
2. 用恒值号表检查：是否有成对出现（如选了13又选21）
3. 若有成对：保留出现频率更高的那个，排除配对号
4. 检查恒值号对称性：小号区和大号区是否有对应的恒值号配对
5. 最终输出过滤后的6个红球
```

### 示例
假设初选红球为：`[03, 13, 17, 21, 26, 31]`
- 检查恒值号：03-31（成对！），13-21（成对！）
- 03(热号) vs 31(冷号) → 保留03，排除31
- 13(热号) vs 21(温号) → 保留13，排除21
- 最终：`[03, 13, 17, 26, ?]` — 需补充2个号

## 注意事项
- 恒值号是过滤工具，不是选号工具 — 不能单独用恒值号来选号
- 与区间比、热度分析配合使用效果最佳
- 蓝球不适用恒值号（蓝球1-16，无固定配对关系）
