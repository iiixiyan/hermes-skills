> 本模块记录每次复盘和优化迭代中发现的系统性规律，用于指导后续分析和技能迭代。
> 实战案例库见 `references/case-studies-2026-05-25.md`（2026-05-25北单182-184逐场分析）

### 一、范式的优化：从学术模型到实战八步法

#### 1.1 北单预测的范式升级路径

| 版本 | 核心逻辑 | 问题 | 优化方向 |
|:---|:---|:---|:---|
| v4.0 | 市场校准：赔率反推λ，完全拟合市场 | 盲信市场，忽略庄家诱导 | 赔率并非总是对的 |
| v4.1 | Edge检测：模型vs市场找差价 | 仍然以赔率为起点做模型 | 应先有独立判断，再去对比市场 |
| v4.2 | 模型优先：独立评级先于赔率 | 计算复杂，实战难落地 | 模型有价值的底层逻辑 |
| **v4.3** | **双模并行：八维预测法 + 实力盘定位法** | — | **实战为王，易精不易多** |

#### 1.2 关键教训

**赔率不是指南针，是对手方。**
- 盲信"北单三赔最低=方向" = 跟着庄家走，庄家永远不会让你赚钱
- 机构开盘的核心目的是**引导资金流向**，不是告诉你真实方向
- 实力盘偏离度 + 水位 + 变盘趋势 = 解读机构意图的三要素

**学术模型 ≠ 实战收益。**
- 泊松分布、贝叶斯网络、Glicko-2等学术工具提供的是"数学上的严谨"
- 但足球赛果的不可预测性远超数学模型的假设边界
- 实战中，**基本面八维交叉** + **实力盘偏离度判断**的稳定性和可操作性远优于复杂计算

### 二、模式一：八维预测法核心经验

#### 2.1 八维权重分配（实战总结）

| 维度 | 权重 | 说明 |
|:---|:---:|:---|
| 联赛势位 | ★★★★★ | 最稳定的基本面指标，赛季中后期尤其可靠 |
| 主客战绩 | ★★★★ | 区分"主场龙/客场虫"的关键维度 |
| 近期状态 | ★★★★★ | 6轮不败/连败的影响力常大于联赛排名 |
| 交战历史 | ★★★ | 有克制关系时重要，无压制时仅作参考 |
| 相似对手 | ★★★★★ | 区分"伪强队"和"真弱队"的最佳工具 |
| 初盘指数 | ★★★★ | 判断机构态度起点，结合偏离度分析 |
| 盘中走势 | ★★★ | 临场修正信号，变盘幅度>水位变动 |
| 交叉对比 | ★★★★★ | 决定是否出手的最终过滤层 |

#### 2.2 相似对手对比的关键价值

**为什么这步最重要？** 因为它揭露了球队的"真实水平"——不太会受近期状态或运气因素干扰。

**实战规则**：
- 双方同区域战绩差异 > 30% → 差异方的真实实力与排名不符
- 662 vs 167级差异 → 强烈信号，基本面几乎可定方向
- 同时结合近期状态验证：状态差+同区域战绩差 → 放弃为佳

### 三、模式二：实力盘定位法核心经验

#### 3.1 实力盘是锚，不是答案

| 误用 | 正确用法 |
|:---|:---|
| "实力盘这么开，所以应该买这个方向" | "初盘偏离实力盘X档，机构要么在保护要么在诱导" |
| 机械套用盘口定位表 | 结合水位和变盘验证假设 |
| 用实力盘否定基本面 | 实力盘+基本面+水位三者同向才出手 |

#### 3.2 机构意图判断的三种常见陷阱

| 陷阱 | 现象 | 正确识别 |
|:---|:---|:---|
| **升盘=真看好？** | 强队浅开，临场升盘造热 | 初盘低于实力盘 → 临场升盘是**诱导买入**，应走下盘 |
| **退盘=不看好？** | 初盘深开，临场退盘示弱 | 初盘高于实力盘 → 退盘是**驱赶资金**，仍走让球方 |
| **低水=安全？** | 高水方看似打不出 | 同档球队高水让球=受热危险，真正保护的是低水受让方 |

#### 3.3 水位解读的三个层次

| 层次 | 观察点 | 判断价值 |
|:---|:---|:---|
| 第一层 | 谁高水/谁低水？ | 识别哪方受热、哪方有赔付压力 |
| 第二层 | 水位在升还是降？ | 识别资金流向和机构是否有意引导 |
| 第三层 | 变盘配合了什么题材？ | 确认是否在利用舆论/新闻造势 |

### 双模交叉验证（v4.3.2 偏重机制）

> 分歧时不盲目双选，也不硬选一边。根据 **信号强度 + 联赛特征 + 数据质量** 判定偏重方向。

#### 偏重判定规则

| 条件 | 偏重 | 操作 |
|:---|:---:|:---|
| 模式二触发**因子4b**（验证15/15） | **偏向模式二** | 以模式二为主，异常比分按因子4b方向出 |
| 模式二触发**实力盘偏离≥1档**（如客让平半→平手） | **偏向模式二** | 盘口变动可信度>基本面判断 |
| 模式二**无信号**（盘口合理+水位平衡） | **偏向模式一** | 机构未表态，以基本面为准 |
| 联赛属于**爱超/爱甲/J1/丹甲**（信号不可靠） | **偏向模式一** | 这些联赛盘口信号已熔断或降权 |
| 联赛属于**瑞典超/挪超/瑞典甲**（信号100%可靠） | **偏向模式二** | 这些联赛盘口信号验证高效 |
| 模式一H2H**压倒性压制**（≥4连胜或≥80%胜率） | **偏向模式一** | H2H魔咒因子覆盖盘口信号 |
| 模式一有**保级/争冠战意差**（赛季末5轮） | **偏向模式一** | 战意驱动往往超过盘口信号 |
| 两者都模糊（无信号+基本面接近） | **放弃** | 不出手 |

#### 双选触发条件（必须带主推方向）

**核心原则**：双选不等于无差别推荐两个选项。必须判定哪个方向是**主推**（信号更强的那一方），另一个是**辅防**（覆盖用）。

偏重判定规则（按优先级从高到低检查，命中一条即停）：

| 优先级 | 条件 | 偏重 | 说明 |
|:---:|:---|:---:|:---|
| 1️⃣ | 模式二触发**因子4b**（验证15/15） | → 偏模式二 | 因子4b为最高优先级证据链 |
| 2️⃣ | 模式二触发**实力盘偏离≥1档** | → 偏模式二 | 盘口变动可信度>基本面判断 |
| 3️⃣ | 联赛属于**瑞典超/挪超/瑞典甲**（信号100%可靠） | → 偏模式二 | 这些联赛盘口信号已验证高效 |
| 4️⃣ | 模式一H2H**压倒性压制**（≥4连胜或≥80%胜率） | → 偏模式一 | H2H魔咒因子覆盖盘口信号 |
| 5️⃣ | 模式一有**保级/争冠战意差**（赛季末5轮内） | → 偏模式一 | 战意驱动往往超过盘口信号 |
| 6️⃣ | 联赛属于**爱超/爱甲/J1/丹甲**（信号不可靠） | → 偏模式一 | 这些联赛盘口信号已熔断或降权 |
| 7️⃣ | 模式二**无信号**（盘口合理+水位平衡） | → 偏模式一 | 机构未表态，以基本面为准 |
| 8️⃣ | 以上均不符合，偏重打平 | → **放弃该场** | 不出手 |

各场景下的操作：

| 模式一 | 模式二 | 操作 |
|:---:|:---:|:---|
| 🟢 明确 | 🟢 一致 | ✅ **强确认，单选主推该方向** |
| 🟢 明确 | 🔴 反向+偏重判定命中 | ✅ **并列输出两个方向，标明偏重方向** |
| 🟢 明确 | 🔴 反向+偏重判定不命中(打平) | ⛔ **放弃**（不强行主推） |
| 🔴 无信号 | 🔴 无信号 | ⛔ **放弃** |

#### 双选输出格式（v4.4.0 含进球数预测）

```
📌 推荐：[方向A] / [方向B]  ➡ 偏重[方向A]
  ⚽ 进球数：[总进球1]球 / [总进球2]球 / [总进球3]球
  🏋️ 偏重依据：[具体原因，如"因子4b触发→偏模式二"]
  ⭐ 信心：★★★☆☆
```

**关键变化**：两个选项用 `/` 并列输出，不使用箭头区分主辅。偏重方向通过「偏重」标记后单独指明。新增`⚽ 进球数`行，按概率从高到低排列3个最可能总进球数。信心星级统一标注，不再拆分主推/辅防两档。

**重要**：偏重方向必须有明确的判定依据（来自8级偏重优先级），不可无理由偏重。

#### 精选出手原则

| 模式一（八维）| 模式二（实力盘）| 判定 | 操作 |
|:---|:---|:---|:---|
| 🟢 方向明确 | 🟢 方向一致 | **强确认** | 🟢 可重注 |
| 🟢 方向明确 | 🔴 方向相反 | **分歧→双选** | 🟡 两模相反，双选覆盖两个方向 |
| 🟡 模糊不定 | 🟢 指向明确 | **以盘口为准** | 🟡 轻注 |
| 🔴 信号矛盾 | 🔴 信号矛盾 | **放弃** | ⛔ 不出手 |
| 🟢 方向A | 🟡 防B（平局） | **取A防B** | 方向A，提示防范 |

#### 4.2 精选出手原则

```
每日筛选准则：
  1. 两模方向一致 → 加入候选池
  2. 候选池中选星级≥4星 → 保留
  3. 剩余中选水位支持最明确 → 终选
  4. 最终场次 ≤ 3 场 → 精选出手

强制跳过：
  任何一场双模分歧超过一个方向 → 直接跳过
  水位与基本面完全矛盾（如基本面碾压但盘口退到平手）→ 跳过
```

### 六、2026-05-25北单周日复盘关键发现

#### 6.1 英超：深让(-1)赛季末爆冷风险

2026-05-25复盘发现英超赛季末段（联赛剩余1-2轮）的深让(-1)场次存在系统性爆冷：

| 场次 | 让球 | 欧指信号 | 实际赛果 | 赛果 |
|:----|:----|:--------|:---------|:----|
| 曼城 vs 维拉 | -1 | 升主胜24降4(接近触发) | 1-2 | 让负✅ |
| 利物浦 vs 布伦特福德 | -1 | 升主胜22降4(接近触发) | 1-1 | 让负✅ |
| 西汉姆 vs 利兹 | -1 | 降主胜30升0 | 3-0 | 让胜✅(唯一穿盘) |

赛季末英超深让(-1)爆冷率67%(2/3)，仅西汉姆(降主胜30家极强信号)穿盘。建议赛季末英超深让(-1)场次**降权至0.7倍**，或仅在一降主胜极强(≥25降/≤1升)时才信任上盘。

#### 6.2 一致降主胜跌破有效线

一致降主胜(含走水)准确率从65%降至63%，跌破60-74%有效信号区间下限，正式降级为🔶边缘信号。主要原因：
- 西乙FC安道尔让-1输球(降主胜29/升0)
- 瑞典甲桑德维肯无让球主负(降主胜23/升1)
- 这两场共同特征：让球方(FC安道尔降主胜)或主队(桑德维肯)基本面无法穿盘

**启示**：一致降主胜信号仍需基本面验证，尤其是在西乙、瑞典甲等次级联赛。

#### 6.3 一致升主胜回暖

一致升主胜(扣除走水)准确率从63%升至65%，重回有效信号区间。3场触发全部方向正确：
- 那不勒斯(-1)让平走水
- 诺丁汉森林(0)平局
- 马尔默(-1)让负

**回暖特征**：走水率从45%降至33%(1/3)，有效性提升。

#### 6.4 瑞典联赛继续可靠

瑞典超(1场)和瑞典甲(1场)的信号继续维持高效：
- 瑞典超马尔默(-1)让负：一致升主胜✅
- 瑞典甲桑德维肯(0)主负：一致降主胜(降23升1)但实际错误❌
  - 但桑德维肯的降主胜值(23)属于临界水平(阈值20)，比之前瑞典联赛动辄29-30的极强信号弱

**修正**：瑞典联赛信号强度与降/升家数绝对值正相关，降主胜≥25才视为"强信号"，20-24视为"弱信号"需基本面验证。

#### 6.5 赛季末段特殊规律：无欲无求 vs 保级/争欧战

| 场景 | 模式一影响 | 模式二影响 |
|:---|:---|:---|
| 中游vs保级队 | 战意差+1~2档 | 实力盘参考价值下降，初盘可能偏浅 |
| 已夺冠vs中游 | 战意减2~3档 | 浅开合理，但退盘过深需警惕 |
| 保级生死战 | 战意上限+2档 | 平手/平半盘经常有先入球，冷门高发 |
| 德比/宿敌 | 不受排名影响 | 盘口通常偏深，实际走水率上升 |
