# 信号矛盾检测方法论：升降家数 vs 百家平均

> 2026-07-05 首次系统性验证（11场北单中5场存在严重矛盾）

## 核心发现

**升降家数计数（升/降各多少家）与百家平均赔率变化方向经常系统性矛盾。**

### 根因
- 升降家数 = 仅统计「改变方向的公司数量」，不统计变动幅度
- 百家平均 = 各公司赔率按交易量加权后的整体市场深度
- 矛盾场景：27家降主胜（每家只降0.01），但几家大行大幅升主胜（每家升0.10+）→ 降主家数多但百家平均仍上升

### 解决原则
**当两者矛盾时，百家平均赔率方向更可靠。** 它代表的是「钱往哪里流动」，而升降家数只代表「多少公司动了一下」。

## 阈值判定流程

```
百家变化 < 5%        → 矛盾微弱，仍以升降家数为准
百家变化 5~10%       → 矛盾显著，以百家平均方向为主
百家变化 ≥ 10%       → 硬阈值，彻底排除百家变化反方向
```

## 与亚指的关系

不依赖亚指做判断基准。亚指（升/降盘家数、高/低水）在信号矛盾时同样可能失真。直接以百家平均方向修正偏重方向。
