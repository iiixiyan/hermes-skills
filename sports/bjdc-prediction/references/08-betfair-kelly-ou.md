## 八、必发交易量分析（新增 · 数据来自盈亏Tab）

**数据来源**：59itou详情页→盈亏Tab，通过Playwright采集 `document.body.innerText` 提取。

### 8.1 核心指标

| 指标 | 含义 | 解读 |
|:----|:-----|:----|
| **必发交易量** | 比赛总交易额（港元），反映市场热度 | 交易量越大→关注度越高，大热场次需警惕反向 |
| **各选项交易量** | 胜/平/负 各自的交易量占比 | 交易量最大的选项=市场资金集中方向 |
| **庄家盈亏** | 各选项下庄家的盈亏金额 | 庄家亏损最大的选项=庄家最怕打出的结果 |
| **冷热指数** | 负值=冷（资金流入少），正值=热（资金流入多） | 冷热指数绝对值>20=极端信号 |

### 8.2 必发信号应用规则

| 信号 | 条件 | 含义 | 方向 |
|:----|:-----|:----|:----:|
| **冷热反向** | 冷热指数为负(-)但盘口信号指向该方向 | 市场资金不看好，但盘口暗示→庄家可能在诱 | 反向下盘 |
| **必发交易量异常** | 交易量≥500万港元但盘口信号与该方向一致 | 市场过热+机构引导→警惕反向 | 反向下盘 |
| **庄家盈亏差** | 某选项庄家盈亏绝对值>350万且与盘口方向相反 | 庄家在该选项亏损最大→最怕打出 | 该方向为正 |

### 8.3 必发信号与其它信号的协同

```
凯利>返还率 + 冷热指数>+20 + 盘口降水不升盘 → 极强下盘信号
凯利<返还率 + 庄家亏损最大项一致 + 升盘阻上 → 极强上盘信号
冷热指数三向均衡 + 威廉立博平赔交叉 → 平局概率极高
```

## 九、凯利指数排雷

| 凯利 vs 返还率 | 含义 | 解读 |
|:--------------:|:----|:----|
| 凯利 < 返还率 | 投资价值被认可，风险较低 | **利好信号**，机构对该结果有信心 |
| 凯利 > 返还率 | 市场资金大量涌入，该选项"过热" | **高危警告！**"大热必死"数据体现 |
| 凯利 ≈ 返还率 | 投注与机构预期吻合 | 无明显倾向性 |

**排雷规则**：
- 判断走上盘 → 检查上盘凯利是否 < 返还率
- 判断走下盘 → 检查上盘凯利是否 > 返还率
- 凯利与盘口信号矛盾 → 降低信心，建议放弃或减注

### 9.1 凯利方差（新增）

**凯利方差** = 各主流公司凯利值相对于平均凯利值的标准差，反映机构之间的分歧程度。

| 方差范围 | 含义 | 解读 |
|:--------:|:----|:----:|
| 方差 < 0.05 | 机构高度一致 | 该结果方向可信度极高，可放心采用 |
| 0.05 ≤ 方差 < 0.10 | 机构略有分歧 | 正常范围，不影响判断效力 |
| 0.10 ≤ 方差 < 0.20 | 机构分歧明显 | 该方向存在不确定性，需降低信心中介 |
| 方差 ≥ 0.20 | **严重分歧** | 机构对该方向无共识，建议放弃该场次 |

**凯利方差排雷规则**：
- 盘口信号指向某方向 + 该方向凯利方差 < 0.05 → **极强确认**
- 盘口信号指向某方向 + 该方向凯利方差 > 0.10 → **降低一档信心**
- 凯利方差最大的选项 = 机构最拿不准的方向 → 避开
- 三项凯利方差均 < 0.05（全一致）→ 比赛结果高度可预测，首选凯利<返还率的方向

**凯利vs返还率 + 凯利方差协同**：

| 组合信号 | 强度 | 含义 |
|:---------|:----:|:-----|
| 凯利<返还率 + 方差<0.05 | ⭐⭐⭐⭐⭐ | 机构一致看好，低风险 |
| 凯利<返还率 + 方差0.05-0.10 | ⭐⭐⭐⭐ | 看好但略有分歧，可选用 |
| 凯利>返还率 + 方差<0.05 | ⭐⭐⭐ | 过热但机构有共识，需结合盘口 |
| 凯利>返还率 + 方差>0.10 | ⭐⭐ | 过热+分歧，强烈避雷 |
| 凯利>返还率 + 方差≥0.20 | ⭐ | **立即放弃该场次** |

## 十、大小球独立分析

**大小球与上下盘映射**：

| 大小球变化 | 映射关系 | 逻辑 |
|:----------|:---------|:----|
| **大小球降盘 + 上盘降水不升盘** | 强下盘信号 | 进球预期降低+诱上→双杀上盘 |
| 大小球降盘 + 小盘平半高水 | 配合小盘定胆 | 进球低但上盘高水→慎选下盘 |
| 大小球升盘 | 一般不指向下盘 | 进球预期增加，上盘打出可能性较高 |
