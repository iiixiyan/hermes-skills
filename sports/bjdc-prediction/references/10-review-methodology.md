## 十六、复盘方法论

### ⚠️ 复盘核心铁律（2026-05-24 用户纠正）

**所有复盘必须以比分为唯一核心。** 盘口信号分析、基本面验证、模型信号复盘都不是并列检查项——它们围绕比分展开，服务于解释比分偏差。

```
比分是圆心 ← 盘口信号（解释为什么这个比分没命中）
           ← 基本面验证（什么因素导致比分偏差）
           ← 模型信号（定位比分偏差的源头）
```

**输出优先级（不可违反）**：
1. **比分命中率**（精确+方向+偏差分布）— 必须放在最前面
2. **比分偏差归因**（每场没命中的具体原因：数据错？模型错？漏了信号？）
3. **改进方向**（基于归因，下次怎么调）

盘口信号统计和联赛规律是**次要内容**，放在后面。每个盘口/基本面分析结束时必须回到「→ 这对比分预测意味着什么」。

### 16.1 赛后复盘四步骤

**真正复盘的四个步骤**：
1. 赛后回顾赛前分析记录，标记关键判断依据
2. 对比实际比赛过程与预判的差异点
3. 记录盘口水位变化与实际赛果的对应关系
4. 统计特定联赛/盘口的近期规律性特征，每月汇总提炼

**两大发现价值**：
- **放水期捕捉**：把赔付率进行导向运算，运用值博率找到信号——奖金比实际可能更高时=机构在放水→加大投入
- **阶段性开盘规律**：不同阶段机构对同一联赛的运营思想可能固定，掌握规律可保持较长盈利

### 16.2 复盘反馈机制（v4.0 更新·新增CLV追踪）

#### ⚠️ 信号追踪必须绑定比分

**每个信号的准确率统计最终必须回归到比分**：信号正确 ≠ 真正有用。只有当"信号触发 → 正确判断让球结果 → 对应实际比分"时，信号才真正创造了价值。

**信号追踪增强规则**：
- 信号触发的每场比赛，必须记录**实际比分**和**让球后结果**（不只是"信号对了/错了"）
- 信号准确率统计的二级指标：①信号判对方向→实际比分符合预期？②信号判对方向但比分偏差大→信号稳定性存疑
- 比分偏差≥2球的场次，即使信号方向正确，也应单独标注为"信号部分有效"

#### CLV追踪（新增 v4.0）

**闭合线价值（Closing Line Value）= 收盘赔率 vs 初盘判断方向**。CLV 是判断模型是否真正有优势的黄金标准。

**追踪规则**：
```
初盘判断方向（如：让负）
收盘时该方向赔率：下降 → CLV正（正确判断）
收盘时该方向赔率：上升 → CLV负（判断可能错误）
```

**CLV评分规则**：
| CLV变化 | 含义 | 对后续相同联赛/盘口的影响 |
|:-------|:-----|:----------------------|
| 判断方向赔率下降 >5% | CLV强正 | 同联赛同盘口后续加权重1.1倍 |
| 判断方向赔率下降 1-5% | CLV正 | 维持原权重 |
| 判断方向赔率变化 <1% | CLV平 | 无额外信息 |
| 判断方向赔率上升 1-5% | CLV弱负 | 同联赛同盘口后续降权0.9倍 |
| 判断方向赔率上升 >5% | CLV强负 | 同联赛同盘口后续降权0.7倍，需重新审视模型 |

**每场分析时的CLV引用**：
```
"📈 CLV追踪：该联赛该盘口近10场CLV+3.2%，模型方向与市场收盘方向一致→可信任"
或
"📉 CLV追踪：该联赛该盘口近10场CLV-4.1%，模型方向多次被市场收盘否定→降一档信心"
```

## 十六.3 联赛信号熔断机制（v4.1 新增 · 系统性风控）

> 取代"人工避雷"，升级为自动熔断体系。核心逻辑：信号在特定联赛的准确率低到阈值时，系统自动禁用该信号，而非依赖人工记忆。

### 三级熔断体系

| 级别 | 触发条件 | 动作 | 标记 |
|:----:|:--------|:-----|:----:|
| **🟡 预警** | 某信号在某联赛近10场准确率<50% | 该信号在该联赛**降权至0.5倍**，输出中标注⚠️ | `联赛预警` |
| **🟠 限制** | 近15场准确率<40% | 该信号在该联赛**降权至0.2倍**，强制提示"不建议使用" | `联赛限制` |
| **🔴 熔断** | 近20场准确率<30% | 该信号在该联赛**完全禁用**，分析中跳过该维度 | `联赛熔断` |

```
3. 若某信号=🟠限制 → 评分×0.2，输出标注「该信号在此联赛不可靠」
4. 若某信号=🟡预警 → 评分×0.5，输出标注「该信号在此联赛预警」
5. 若某联赛所有信号均为🔴/🟠 → 仅依赖北单赔率+基本面分析
```

### "黑洞联赛"全熔断处理

当某联赛全部5个欧指信号均触发🔴熔断时，该联赛被标记为**"黑洞联赛"**：

| 黑洞联赛 | 状态 | 替代方案 |
|:--------|:----|:--------|
| **J1联赛** | 全部熔断 | **仅用北单三赔最低方向+基本面**，不使用任何欧指信号 |

**黑洞联赛分析模板**：
```
⚠️ J1联赛：欧指信号全部熔断，仅基于北单赔率+基本面分析
→ 北单三赔最低方向 = 市场最终判断
→ 基本面（状态/伤停/战意）作为验证
→ 不参考任何欧指一致性信号
```

### 熔断解除条件

- 连续10场该联赛该信号准确率恢复至>50% → 熔断→限制
- 连续15场>60% → 限制→预警
- 连续20场>70% → 预警→正常

### 与CLV动态权重的协同

熔断机制与CLV动态权重（§十三）是两层风控：

```
第一层（粗滤）：熔断表 → 禁用级直接跳过信号
第二层（精调）：CLV系数 → 在可用信号内动态调整权重
```

> 两层独立运作。熔断基于长期准确率（20场窗口），CLV基于近期收盘赔率方向（10场窗口）。
