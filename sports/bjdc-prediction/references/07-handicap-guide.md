## 五、小盘定胆（上盘捕捉专项）

**适用范围**：平手盘（0球）、平半盘（0.25球）、受让平半（+0.25球）

**高水阻上三条件判断法**（平半盘上盘持续高水≥1.00时逐条核对）：

| 条件 | 内容 | 核心问题 |
|:----:|:-----|:---------|
| 条件1 | 上盘在基本面数据上是否处于劣势？ | 交锋记录、近况、伤病是否下盘占优？ |
| 条件2 | 上盘在大众心理名气上是否处于均等或劣势？ | 下盘知名度是否≥上盘？市场是否认为下盘"更安全"？ |
| 条件3 | 上盘是否持续高水（≥1.00），下盘是否持续低水安全区（0.80-0.94）？ | 最关键盘口形态 |

**判定原则**：三条基本都符合 → **定上盘胆**

**⚠️ 关键矛盾场景（小盘定胆 vs 欧指叠杀）**：
当小盘定胆三条件满足（指向定上盘胆），但同时出现以下情况时，**小盘定胆信号被欧指降盘双杀**：
1. **欧指**：≥15家一致升主胜（上盘方）、0家降主胜 — 欧指方向与本技能信号冲突
2. **亚指**：多家公司降盘（如半球→平半），而非升盘阻上
3. 此时小盘定胆的"阻上"假设被欧指一致性看衰+降盘趋势否定

**结论**：欧指一致性（≥15家升/0家降）+ 降盘趋势 的优先级 **高于** 小盘定胆信号。取降盘信号为主，方向改为下盘。2026-05-20复盘验证：科金博联(平半高水三条件满足)但21家升主胜+3家降盘→实际客队不败。

**小盘定胆 vs 诱上/阻上协同**：

| 场景 | 所属框架 | 信号特征 | 操作 |
|:----|:---------|:---------|:----|
| 普通阻上（中大盘） | 诱上/阻上框架 | 升盘+中高水 | 走上盘 |
| 小盘阻上（平手/平半） | 小盘定胆 | 平半+持续高水+基本面劣势 | 定上盘胆 |

## 六、盘口技术手册

### 6.1 平手盘（0球）

**不升/降盘，只变水位**：
- 开盘上盘低水 → 上盘胜出（后期升至高水则防平）
- 开盘上盘高水 → 下盘胜出或走水（水位下跌则平局居多）
- 上下盘平水开盘 → 后盘低水队胜出（平局概率大）
- 盘口水位一直不变 → 低水队胜出概率大

**升/降盘**：
- 平手升平半：开盘上盘低水则上盘胜出；高水则下盘胜出（多为平局）
- 平手变受让平半：平局概率大（主队超低水开盘例外，一般主队会输）

**0球盘特殊规律（下盘方向）**：
- 弱队主让0球：主队弱势却作为让球方 → 90%概率下盘（客队不败）
- 客队让0球后升盘至0.25：升盘为平局散热 → 主队不败

### 6.2 平半盘（0.25球）

- 主队受让平半平水开盘不变，主队水逐渐升高至高水 → 客胜
- 主队受让平半高水开盘不变，主队一直高水、客队一直低水 → 客队赢球赢盘
- 平半盘长时间超低水，临场升盘降水却维持1.00以上水位 → 客队不败，平局居多

### 6.3 半球盘（0.5球）

**盘口不变只变水位**：
- 主让半球低水 → 庄家信心足，一般不选客队
- 主让半球高水 → 庄家信心不足，一般不选主队
- 水位持平 → 分胜负，打平概率最低
- 主让半球初始上盘水位>1.85 → **下盘频出**
- (其他详见完整技术手册)

**升降盘**：
- 半一退半球高水 → 下客队
- 半一高水退半球低水 → 下主队
- 半球低水升半一高水 → 下主队
- 半球低水升半一后水位又调低 → 盘口过热，主队不胜
- 半球高水降平半低水 → 博平局
- 半球低水降平半高水 → 下客队

### 6.4 半一盘（0.75球）

**实力盘 vs 蛊惑盘**：
| 类型 | 特征 | 方向 |
|:----:|:-----|:----:|
| 实力盘 | 实力差距开半一盘合理 | 盘口高看一方易打出 |
| 蛊惑盘 | 实力差距开半一盘不合理 | 操盘手认为会爆冷 |

**水位信号**：
- 上盘持续低水时间过长 → 对上盘十分不利
- 长时间持续半一中水 → 主队最多一球小胜（诱上）
- 持续半一高水 → 可放心投注上盘，且能净胜两球

**判断五步骤**：①分析是否实力盘 → ②寻找开出半一盘理由 → ③根据近况判断哪方受热 → ④观察临场盘赔吻合度 → ⑤观察盘口水位临场表现

**高水信号（下盘）**：客队让0.75~1球且长时间高水 → 上盘难打出

### 6.5 一球及一球以上盘

- 让球队低水开盘盘口不变 → 稳胜
- 让球队高水开盘盘口不变 → 稳胜
- 让球队高水开盘后变低水 → 有平的可能
- 让球1.5球以上，以触底水位(0.80)开盘且盘口不变 → 有爆冷可能

**反常高开信号（下盘）**：实力差距不大却创历史最深盘口 → 下盘可期
