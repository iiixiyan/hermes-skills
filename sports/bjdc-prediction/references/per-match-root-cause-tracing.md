# 北单逐场根因追溯法

> v1.0 | 2026-06-28 | 从26069期6/7场错误的根因分析中提炼

---

## 为什么要逐场追溯

统计汇总（偏重命中率25%、双选覆盖率62.5%）只能告诉你"有多差"，
逐场根因追溯才能告诉你"为什么差"——每场错的根因不同，需要独立的诊断。

---

## 追溯3层法

### 第1层：预测时实际可获取的数据

从59itou详情页战绩Tab提取（这是Hermes浏览器中唯一可靠的数据源）：

| 数据项 | 提取位置 | 用途 |
|:-------|:---------|:-----|
| 排名 | 页面顶部 `世XX` / `巴乙X` / `挪甲X` | 实力基准 |
| 综合实力 | 页面底部「综合实力」主队X vs 客队X | 客观差距 |
| 近10场战绩 | 战绩Tab 前两行(胜/平/负/进失球) | 近期状态 |
| 主场/客场战绩 | 战绩Tab 第三行起 | 主客差异 |
| 近5场盘口走势 | 战绩Tab逐场亚指(赢/输/走) | **赢盘能力** |
| H2H交锋 | 两队交锋段 | 历史克制 |
| 近期赛程 | 底部赛程表 | 密集程度/战意 |
| 伤停 | 情报Tab | 核心球员缺失 |

### 第2层：预测依据 vs 实际数据的差距

对照预测时用的「依据」：

| 依据类型 | 示例 | 常见问题 |
|:---------|:-----|:---------|
| 联赛统计默认值 | "巴西乙-1盘75%" | 忽略具体排名/客场战绩 |
| 信号解读 | "造热陷阱" | 信号方向是否与基本面一致？ |
| 排名/FIFA差 | "62位FIFA差" | FIFA差≠赢盘能力！要查赢盘率 |
| 盘型判断 | "ad买客+深盘" | 深盘走水(让平)是否在双选池？ |

### 第3层：定位因果类型

| 类型 | 特征 | 修复方向 |
|:-----|:------|:---------|
| **数据遗漏** | 可采集到但没用上的数据 | 检查预测流程缺了哪步 |
| **双选结构缺陷** | 方向判断对但双选没包住 | 修改默认双选策略 |
| **指标误用** | 覆盖率被当作偏重方向 | 分开统计覆盖率和偏重准确率 |
| **规则优先级错误** | 信号规则碾压基本面 | 调整规则优先级顺序 |

## 实战模板

```markdown
### #N [主队](hcap) vs [客队]
**预测依据**：[引用预测时的依据文本]
**实际结果**：[让胜/让平/让负]

**第1层——可采集数据**：
- 排名：[主队#? vs 客队#?]
- 综合实力：[X:X]
- [主队]近10场/近X场盘口：[X胜X平X负 / X赢X输X走]
- [客队]近10场/近X场盘口：[X胜X平X负 / X赢X输X走]
- 主场/客场：[该队主场/客场战绩]
- 关键发现：[最反直觉的那条数据]

**第2层——依据 vs 数据差距**：
预测说[依据]，但实际数据中[反例数据]被忽略

**第3层——根因类型**：数据遗漏 / 双选结构缺陷 / 指标误用 / 规则优先级
```
