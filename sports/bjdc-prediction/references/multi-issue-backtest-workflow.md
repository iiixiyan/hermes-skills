# 北单多期全量回测工作流

> 用于对北单历史期次做批量数据采集→信号分析→策略验证→skill优化的完整流水线

## 流水线步骤

### Step 1: 从okooo批量获取数据

```bash
for iss in 26061 26062 26063 26064 26065 26066 26067; do
  curl -s "https://vxbf.okooo.com/kaijiang/sport.php?LotteryType=WDL&LotteryNo=$iss" \
    -H 'User-Agent: Mozilla/5.0' -H 'Referer: https://www.okooo.com/' \
    | iconv -f GBK -t UTF-8//IGNORE > /tmp/okooo_$iss.html
  sleep 0.5
done
```

### Step 2: 解析赛果

HTML关键标签：
- `fontred">X` — X=主/客/平 为赛果
- `<cite class="fontred2 font10">(-1)</cite>` — (-1)为让球数
- `gray9">3.12` — SP值

### Step 4: 按让球+联赛分组统计最优双选

key=(让球数,联赛) → 统计主/客/平频次 → 选频次最高2方向

### Step 5: 三层优化策略

1. **静态规则** — 24条主表（让球+联赛最优双选）
2. **平局保护** — 平率>30%分类强制含让平
3. **SP<1.7动态** — 极低SP方向强制纳入

### Step 6: 验证标准

- 每期覆盖率75-88%（避免过拟合）
- 无单期<70%
- 各联赛覆盖率差异<15%
- 较纯上盘基线提升>40%
