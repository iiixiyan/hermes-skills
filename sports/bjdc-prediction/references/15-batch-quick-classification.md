# 批量预测快速分类指南（实战模式）

> 用于早场/晚场批量预测场景。当需要一次性分析10~25场比赛时，按此流程快速筛分，
> 避免在每场比赛上过度分析导致时间耗尽。

## 三通道分类法

拿到一场比赛的数据后，并行检查三条通道，按结果归类：

### 通道A：欧指一致性信号强度

| 信号等级 | 判断标准 | 典型场景 |
|:--------:|:---------|:---------|
| **🔥强** | 升/降家数差值≥12 且方向一致（如胜降≥12家且胜升≤3家） | 长崎vs水户(胜降17家)、釜山vs坡州(胜降14家) |
| **⚡中** | 升/降家数差值≥7 但<12 | 水原FC vs 城南(胜降14/升3=差11) |
| **❄弱** | 升/降家数差值<7 或方向混乱 | 华城vs庆南(升6降2=差4) |

**强信号** + 其他通道支持 → 单选。**弱信号** → 倾向放弃/双选。

### 通道B：SP-Edge匹配度

| SP赔率特征 | 对应Edge含义 | 操作 |
|:-----------|:-------------|:-----|
| 主胜SP最低(最被看好) | 模型概率 vs 市场概率 ≥5% | 信号叠加→单选 |
| 主胜SP最低但Edge为负 | 市场可能高估了让球方 | 谨慎降信心 |
| 主胜SP最高(最不被看好) | Edge可能很大但方向是逆向 | 检查欧指是否一致→若一致则考虑爆冷 |
| Edge≥5%且方向与SP最低一致 | **强出击信号** | 信心+1⭐ |

**注意**：北单三赔是**让球胜平负赔率**，欧指百家是**标准胜平负赔率**。两者方向不一致不一定是矛盾——可能反映了让球数的实际影响。例如：
- 让-1球：欧指主胜低(标准主队强)，但北单"胜"SP高(让1球后很难赢)
- 让+1球：欧指客胜低(标准客队强)，但北单"胜"SP低(受让1球后容易不败)

### 通道C：亚指水位验证

| 亚指特征 | 信号含义 | 处理 |
|:---------|:---------|:-----|
| 低水(<10家高水) + 欧指一致 | 真实支持 | ✅ 强化方向 |
| 高水(>10家高水) + 欧指一致 | 诱盘嫌疑 | ⚠️ 降低信心，双选规避 |
| 高水 + 欧指反向 | 阻盘可能 | 🔍 检查基本面是否支撑 |
| 降盘(升盘数-降盘数<0) + 欧指一致 | 机构信心下降 | 📉 需调低信心 |

## 快速决策矩阵

```
        ┌────────┬────────┬────────┐
        │ 通道A强 │ 通道A中 │ 通道A弱 │
├────────┼────────┼────────┤
│ 通道B+ │  单选  │  双选偏│  双选  │
│ C支持  │        │  重    │        │
├────────┼────────┼────────┤
│ 通道B+ │  双选  │  双选  │  放弃  │
│-C中性  │  偏重  │        │        │
├────────┼────────┼────────┤
│ B-C-   │  双选  │  放弃  │  放弃  │
│ 不支持 │        │        │        │
└────────┴────────┴────────┘
```

## 批量排序优先级

处理10+场比赛时，按以下顺序安排分析精力：

1. **先处理强信号场次**（通道A强+通道B支持）：最快速给出单选方向
2. **再处理实力盘偏离明显场次**（排名差≥5位+综合实力差≥15分）：模式二输出
3. **最后处理模糊场次**（通道A弱+通道C分歧）：快速判断放弃或双选

**时间预算参考**：
- 20场比赛 → 强信号约6-8场(每场30s) + 中等8-10场(每场20s) + 模糊4-6场(每场10s) ≈ 8-12分钟

## 典型快速分类模式（实战案例）

### 模式A：让负/单选（客队强势型）
```
条件：通道A负降≥12家 + Edge≥5%(向客队) + 亚指客让低水 + 综合实力客队>55
输出：让负 → 单选
案例：36忠南vs水原三星、83安山vs天安城
```

### 模式B：让胜/单选（主队碾压型）
```
条件：通道A胜降≥12家 + 主队主场不败 + Edge≥3% + 亚指低水无升盘
输出：让胜 → 单选
案例：38长崎vs水户、41水原FCvs城南
```

### 模式C：让胜/双选（受让方优势型）
```
条件：让球数≥+1 + SP胜≤1.50 + 欧指基本支持
输出：让胜(单选) / 让胜+让平(双选)
案例：37龙仁vs大邱(让+1, SP1.36)、40金海vs金浦(让+1, SP1.15)
```

### 模式D：放弃型（信号模糊）
```
条件：通道A弱 + Edge<3% + 亚指平手盘或低波动 + 基本面均势
输出：放弃（双选覆盖示意见稍高/稍低方向）
案例：32大阪樱花vsFC东京(Edge0.4%)、81华城vs庆南(Edge0.6%)
```

### 模式E：高Edge逆势型
```
条件：Edge≥5%但方向与市场SP最低的方向相反
处理：遵循Edge方向(模型概率>市场概率)，但降1⭐信心
案例：29神户vs鹿岛(市场SP胜2.73最低,但Edge+6.3%在胜方向)
注意：需交叉检查欧指方向是否与Edge一致
```

## 结合偏重判定8级优先级

批量场景下，8级优先级简化为三档：

| 触发条件 | 操作 |
|:---------|:-----|
| 实力盘偏离≥1档(P2) 或 联赛=瑞典超/挪超/瑞典甲(P3) | **偏模式二**，客队方向优先 |
| H2H碾压(P4) 或 赛季末战意差(P5) 或 联赛=爱超/爱甲/J1(P6) | **偏模式一**，主队方向优先 |
| 以上均不命中 | 三通道分类法输出(见上表) |
