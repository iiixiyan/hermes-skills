## 三-B、基本面分析模块（新增）

> 基本面分析是盘口信号之外的第二大分析支柱。当盘口信号模糊（评分-2~+2）时，基本面起决定性作用；当盘口信号明确时，基本面用于验证或推翻信号。

### 3B.1 近10场战绩趋势判定

从「战绩Tab」采集主客队近10场战绩数据，按以下标准判定状态趋势：

| 趋势类型 | 判定标准 | 信号含义 |
|:--------|:---------|:---------|
| **🔥 爆发期** | 近3场全胜或进球≥7球 | 球队处于攻击高潮，上盘/让球方可信 |
| **📈 上升期** | 近6场积分 > 前6场积分 ≥ 2分 | 状态持续改善，值得高看一线 |
| **➡️ 平稳期** | 近6场与赛季均值差异 < 0.3球 | 无明显趋势变化，参考盘口信号即可 |
| **📉 下滑期** | 近6场积分 < 前6场积分 ≥ 2分 | 状态下滑，需降低让球方信心 |
| **❄️ 冰点期** | 近3场全败或进球≤2球 | 球队处于低谷，受让方/下盘方向可关注 |

**应用规则**：
- 爆发期 + 盘口对上盘有利 → 强上盘确认，可重注
- 冰点期 + 盘口对上盘有利 → 需警惕"机构在诱上"，降低一档信心
- 下滑期 + 盘口一致升主胜 → 基本面与盘口信号一致看衰主队，极强下盘

### 3B.2 主客场得失球分析

从「战绩Tab」提取主客战绩数据，计算主场强度指数：

```
主场强度 = 主场场均进球 ÷ 客场场均失球
客场韧性 = 客场场均进球 ÷ 主场场均失球
```

| 主场强度 | 含义 | 盘口修正 |
|:--------:|:-----|:---------|
| > 2.0 | **魔鬼主场** | 主队盘口可信度+20%，半球≈普通联赛一球效果 |
| 1.5 ~ 2.0 | 主场强势 | 主队盘口可信度+10%，无需主队降权 |
| 1.0 ~ 1.5 | 主场正常 | 按正常主场系数计算（见联赛基因表） |
| < 1.0 | 主场疲软 | 主队盘口可信度-10%，浅盘主让需谨慎 |

**应用规则**：
- 主场强度 > 1.5 + 盘口为主队让球 → 盘口合理性高，上盘可信
- 主场强度 < 1.0 + 主队却让球 → 疑为"弱队让球"信号（见§6.1平手弱队让球规则），走下盘
- 客场韧性 > 1.2 + 客队受让 → 客队客场有战斗力，下盘可信度提升

### 3B.3 伤停影响量化评估

从「阵容Tab」的预计伤停信息，按以下标准量化影响：

| 缺阵级别 | 判定标准 | 实力折损 | 盘口修正 |
|:--------|:---------|:--------:|:---------|
| **🔴 致命级** | 头号射手/主力门将/防线核心（3选2+缺席） | -30% | 若盘口未反映→强烈预警，下盘方向 |
| **🟡 重要级** | 常规主力（2名以上缺席） | -15% | 浅盘（平半以下）受影响大，深盘影响有限 |
| **🟢 边缘级** | 轮换球员/替补缺阵 | -5% | 基本不影响盘口判断 |
| **⚪ 无信息** | 阵容Tab无伤停数据或比赛未到赛前 | 0% | 按正常分析，标注"无伤停信息" |

**应用规则**：
- 致命级缺阵 + 该队为上盘（让球方）→ **盘口需降1档信心**，交叉评分中基本面项-1
- 致命级缺阵 + 该队为下盘（受让方）→ **下盘可信度再提升**，交叉评分中基本面项+1（支持下盘方向）
- 重要级缺阵但盘口无变化 → 机构认为影响不大，维持原判断

### 3B.4 战意量化评估

根据「排名Tab」的联赛排名、积分差距、赛季末段情况量化战意：

| 战意级别 | 判定标准 | 影响 |
|:--------|:---------|:----|
| **🔥 殊死战意** | 保级生死战/争冠天王山/德比战/杯赛淘汰赛 | 战意方表现上浮+20%，异常权重+1级 |
| **💪 明确战意** | 争欧战资格/保级区边缘/争直升名额 | 战意方表现上浮+10% |
| **➡️ 普通战意** | 中游无欲无求/赛季末段已无目标 | 不影响 |
| **❄️ 无战意** | 已提前夺冠/已确定降级/杯赛大幅轮换 | 该队表现下浮-15%，冷门诱因+1级 |

**赛季末段强制检查**（与football-prediction因子2联动）：
- 联赛剩余场次 ≤ 5 轮时，**必须**检查每队的当前战意级别
- 主队"❄️无战意" + 客队"🔥殊死战意" → **爆冷系数+2级**（而非通常的+1级）
- 检查结果在输出中单独标注一行："🔍 赛季末战意检查：[主队状态] vs [客队状态]"

### 3B.5 基本面与盘口信号矛盾处理规则

| 场景 | 盘口信号 | 基本面信号 | 处理规则 |
|:----|:---------|:-----------|:---------|
| ① | 极强上盘（一致降主胜+升盘） | 但基本面冰点期+致命伤停 | **盘口信号优先**（机构信息量更大），但降一档信心 |
| ② | 极强下盘（一致升主胜+降盘） | 但基本面爆发期+魔鬼主场 | **盘口信号优先**，但标注"基本面反向"提示 |
| ③ | 信号矛盾（评分-2~+2） | 基本面一方倾向明确 | **基本面优先**，按基本面方向判定 |
| ④ | 无明显盘口信号 | 基本面差距≥2档 | 按基本面方向判定，标注"基于基本面" |
| ⑤ | 盘口与基本面同向 | 两者一致 | **极强确认**，评分在原基础上+2 |

### 3B.6 交叉评分中的基本面权重提升

原交叉评分中"基本面 ±2分"过于笼统，现拆分为具体细则：

| 基本面维度 | 权重 | 得分规则 |
|:----------|:---:|:---------|
| **状态趋势** | ±1 | 爆发期/上升期→+1（支持上盘）；冰点期→-1（支持下盘）；平稳期0 |
| **主场强度** | ±1 | 魔鬼主场(+)→+1；主场疲软→-1；正常0 |
| **伤停影响** | ±1 | 致命级缺阵在上盘方→-1；致命级缺阵在下盘方→+1（支持下盘）；无信息0 |
| **战意** | ±1 | 殊死战意→+1（支持该方向）；无战意→-1；普通0 |
| **高原/联赛基因** | ±1 | 高原合理深让→+1；杯赛轮换降权→-1；无特殊情况0 |

**基本面总分范围：-5 ~ +5**（原±2提升为±5）

**当盘口信号模糊（评分-2~+2）时，以基本面评分为主：**
- 基本面评分 ≥ +3 → 该方向可轻注
- 基本面评分 ≤ -3 → 反向可轻注
- 基本面评分在-2~+2 → **建议放弃该场**（基本面和盘口均无信号）
