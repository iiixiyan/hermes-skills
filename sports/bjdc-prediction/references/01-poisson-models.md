## 三、泊松分布与三大数值模型

**泊松分布**：核心参数 λ（场均进球数），考虑主/客进攻能力、防守能力、控球率、射门次数等。由两队 λ 推导比分概率分布、大小球概率和胜平负概率。

**三大经验性模型**：

| 模型 | 核心公式/规则 | 适用场景 |
|:----:|:-------------|:---------|
| **埃罗预测法** | 主队取胜可能 = 44.8% + 0.53% ×（主队积分 - 客队积分）；差值>15分需结合近期状态 | 联赛积分制 |
| **进球率预测法** | 近6轮平均进球率差值 >0.3→押进球率高一方；0.1-0.3→关注主队；<0.1→主队不胜 | 近况分析 |
| **六场预测法** | 6场积分差≥6→高分队必胜；5分→看主客场；2-4分→高分队胜；≤1分→主场不败 | 近期状态 |

## 三-F、双变量泊松模型（Bivariate Poisson · v4.0新增）

> **学术来源**：Karlis & Ntzoufras (2003, 2009) + *"On the dependence in football match outcomes"* — 独立泊松假设在让球场景下系统性失真。

### 核心问题

独立泊松假设 P(home=x, away=y) = P(home=x) × P(away=y)，但足球比赛中两队进球**存在相关性**：

| 场景 | 相关性 | 示例 |
|:----|:------|:-----|
| **正相关 ρ>0** | 强队领先→收兵→弱队反击进球 | 强队让-2，3-0变3-1 |
| **负相关 ρ<0** | 强队全程压制→弱队零射门 | 强队让-1，2-0 |
| **无相关 ρ≈0** | 双方对攻，互不影响 | 中立场杯赛 |

**对让球盘的影响**：独立泊松在让球深盘（-1/-2）时**系统性低估让平/让负概率**。

### 双变量泊松公式

```
P(S₁=x, S₂=y) = exp(-λ₁-λ₂+λ₃) × 
    Σ_{k=0}^{min(x,y)} [λ₁^(x-k) · λ₂^(y-k) · λ₃^k] / [(x-k)! · (y-k)! · k!]

其中：
- λ₁ = 主队进球强度
- λ₂ = 客队进球强度  
- λ₃ = ρ × √(λ₁ × λ₂)，捕获进球相关性
- ρ ∈ [-1, 1] 为相关系数
```

### 相关性系数 ρ 的估计

| 信号 | ρ修正 | 依据 |
|:----|:-----:|:-----|
| 实力悬殊(λ差>1.0) | -0.15 | 强队压制，弱队难进球 |
| 实力接近(λ差<0.3) | +0.10 | 对攻战，互有进球 |
| 深让(-1/-2) + 主强客弱 | -0.20 | 让球方可能"赢球输盘" |
| 杯赛淘汰赛 | +0.05 | 双方保守，低比分 |
| 赛季末保级战 | +0.15 | 双方都会进球 |
| 德比战 | +0.10 | 情绪化比赛，进球多 |
| 高原主场 | -0.25 | 客队受海拔影响，进攻大降 |
| 默认 | -0.05 | 轻微负相关（足球天然偏向一侧压制） |

### 从市场校准 ρ

如果北单让球赔率的三项概率明确偏离独立泊松预测，可以用同样的联合校准方法估计 ρ：

```
λ₁, λ₂, ρ = argmin Σ(P_model - P_mkt)²
```

| **巴甲** | -0.10 | 主场压制明显，客场进球难 | -1让球时让平概率需上调 |
| **阿甲** | -0.08 | 防守优先联赛 | 深让防让平 |

### 对预测的影响示例

```
场景：主让-1球，λ1=2.0, λ2=0.8

独立泊松：
  P(让胜: 净胜≥2) = 41.2%
  P(让平: 净胜=1) = 24.8%  
  P(让负: ≤0)    = 34.0%

双变量泊松(ρ=-0.15)：
  P(让胜: 净胜≥2) = 38.7%  ← 下降（强队压制但可能赢不多）
  P(让平: 净胜=1) = 27.3%  ← 上升
  P(让负: ≤0)    = 34.0%  ← 不变

→ 让平概率从24.8%升到27.3%，相对提升10%！
```

## 三-G、Dixon-Coles τ 修正与 Sarmanov 族扩展（v4.0 增强）

> **学术来源**：Dixon & Coles (1997) + *"Extending the Dixon and Coles model: an application to women's football data"*

### 现有基础

football-prediction v3.5 已含 Dixon-Coles τ。本次将其**移植到 bjdc-prediction**，并针对让球盘场景增强。

### Dixon-Coles τ 修正

标准泊松**系统性低估**低比分结果（尤其是0-0和1-1），因为足球比赛的进球不是真正独立的。

修正式：
```
P_DC(x, y) = τ_{x,y}(λ₁, λ₂) × P_Poisson(x|λ₁) × P_Poisson(y|λ₂)

其中 τ 在以下情况起作用：
- x=0, y=0: τ = 1 + λ₁λ₂ρ       → 修正0-0
- x=0, y=1: τ = 1 - λ₁ρ          → 修正0-1
- x=1, y=0: τ = 1 - λ₂ρ          → 修正1-0
- x=1, y=1: τ = 1 + (1-λ₁)(1-λ₂)ρ → 修正1-1
- 其他: τ = 1                    → 不修正（高比分不受影响）
```

ρ 的经验值：≈ -0.05 ~ -0.10（足球中低比分比独立泊松预测的更多）

### 北单让球盘的 Dixon-Coles 增强

北单让球盘的**关键比分区**与标准胜平负不同：

| 让球数 | τ修正关键比分 | 原因 |
|:-----:|:------------|:-----|
| -1 | 1-0(让平), 0-0(让负), 2-1(让平) | 穿盘边缘比分 |
| -2 | 2-0(让平), 1-0(让负), 3-1(让平) | 深盘穿盘难 |
| +1 | 0-1(让平), 1-1(让胜), 0-0(让胜) | 受让方守平 |

**增强规则**：当让球数 ≥ 1 时，将 τ 修正扩展到双方 ≤ 2 球的所有组合：
```
if (handicap >= 1):
    扩展 τ 修正到 x≤2, y≤2 共9个组合
    使用 Sarmanov 族扩展公式
```

### Sarmanov 族扩展

Dixon-Coles τ 是 Sarmanov 族的特例。Sarmanov 族可以同时修正多个低比分组合：

```
P_Sarm(x, y) = P_Poisson(x) × P_Poisson(y) × [1 + Σ ω_ij × φ_i(x) × ψ_j(y)]
```

这允许我们**针对让球盘的特定关键比分进行定向修正**，而非一刀切的 τ 参数。

### 实战流程（三模融合）

```
1. 市场校准 λ_market（从北单赔率反推）
2. 统计 λ_stats（从战绩数据计算）
3. λ_final = α × λ_market + (1-α) × λ_stats
4. 应用双变量泊松（含ρ修正）→ 得分概率矩阵
5. 应用 Dixon-Coles τ 修正 → 低比分修正
6. 映射到让球盘 → 让胜/让平/让负概率
7. 结合盘口信号 → 最终判定
```
