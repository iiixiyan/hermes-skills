## 十八、进球数预测方法（v4.4.0 新增）

> 基于已有泊松分布+双变量泊松+DC τ修正框架，从得分概率矩阵推导总进球概率。

### 核心思路

bjdc-prediction 的泊松模型（§三-F 双变量泊松 + §三-G Dixon-Coles τ）已生成完整的 `P(主队=x, 客队=y)` 得分矩阵。总进球概率 = 沿对角线求和：

```
P(总进球 = k) = Σ_{x=0}^{max} P(主队=x, 客队=k-x)
```

### 四步计算流程

**Step 1：获取或计算 λ₁, λ₂**

优先使用市场校准 λ（从北单让球三赔反推，见§三-E），其次使用统计 λ（近3场×0.4+近6场×0.3+赛季×0.3）：

```python
λ₁, λ₂ = λ_market  # 市场校准优先
# 或 λ_stats（当市场校准不可用时）
```

**Step 2：生成双变量泊松得分矩阵**

```python
# 对 x∈[0,8], y∈[0,8] 共81个组合
for x in range(9):
    for y in range(9):
        P(x,y) = bivariate_poisson(x, y, λ₁, λ₂, ρ)
```

其中 ρ 从§三-F.1 联赛ρ参考表获取，或从市场校准联合拟合。

**Step 3：应用 Dixon-Coles τ 修正**

对低比分组合（x≤2, y≤2）应用 DC τ 修正：

```python
τ_00 = 1 + λ₁*λ₂*ρ
τ_01 = 1 - λ₁*ρ
τ_10 = 1 - λ₂*ρ
τ_11 = 1 + (1-λ₁)*(1-λ₂)*ρ
# 其他 x,y → τ=1
P_dc(x,y) = τ(x,y) * P(x,y)
```

**Step 4：计算总进球概率并排序**

```python
total_goal_prob = {}
for k in range(17):  # 总进球0~16（含极低概率的大比分）
    total_goal_prob[k] = 0
    for x in range(min(k, 8)+1):
        y = k - x
        if y <= 8:
            total_goal_prob[k] += P_dc(x, y)

# 归一化
sum_probs = sum(total_goal_prob.values())
for k in total_goal_prob:
    total_goal_prob[k] /= sum_probs

# 取前3高概率
top3 = sorted(total_goal_prob.items(), key=lambda x: -x[1])[:3]
```

### 进球数选三的输出规则

| 场景 | 规则 |
|:---|:---|
| **λ₁+λ₂ < 2.0** | 大概率小球局 → top3优先选 0/1/2 球，如 `1球/2球/0球` |
| **2.0 ≤ λ₁+λ₂ < 3.0** | 正常区间 → top3如实输出 |
| **λ₁+λ₂ ≥ 3.0** | 大概率大球局 → top3优先选 3+/4+/5+ 方向，如 `3球/4球/2球` |
| **让球深盘(-1/-2)** | 强队可能\\\"赢球就收\\\"→ 总进球向下修正0.5球 |
| **德比/保级战** | 情绪化比赛 → 总进球向上修正0.5球 |

### 五大核心修正因子

#### 因子G1：赛程密度（体能修正）

| 场景 | 对总进球的影响 | 修正幅度 |
|:----|:-------------|:--------:|
| 一方周中杯赛（间隔≤72h） | 该队进攻效率下降，总进球减少 | **-0.3球** |
| 双方均周中双赛 | 双方体能均不足，总进球显著减少 | **-0.5球** |
| 一方长途飞行（跨洲/跨时区≥3h） | 客场队体能消耗大，总进球减少 | **-0.2球** |
| 双方均一周一赛（正常节奏） | 不影响 | 0 |
| 联赛末段冲刺期（最后5轮） | 保级/争冠队拼尽全力，进球可能增加 | **+0.3球** |

**与让球方向的协同**：
- 上盘方双赛 → 上盘穿盘能力下降，总进球倾向于下盘方向
- 下盘方双赛 → 下盘抵抗能力下降，总进球倾向于上盘方向
- 深让(-1/-2) + 上盘方双赛 → **强烈预警让平/让负**，总进球上限降低

#### 因子G2：伤停影响（攻防修正）

| 缺阵类型 | 对总进球的影响 | 修正幅度 |
|:--------|:-------------|:--------:|
| 头号射手缺阵（该队进球≥30%由他贡献） | 该队进攻火力锐减 | **-0.5球**（从该队λ中扣减） |
| 主力门将缺阵 | 防守下降，对手进球概率上升 | **+0.3球**（对方λ增加） |
| 防线核心（中后卫/防守中场）缺阵 | 防守组织下降，总进球增加 | **+0.4球** |
| 多名主力同时缺阵（≥3人） | 球队体系崩溃 | **-0.6球**（实力差距拉大或缩小） |
| 无伤停信息 | 按正常处理 | 0 |

**双重效应**：
- 让球方头号射手缺阵 → 穿盘能力下降 + 总进球减少 → **让平/让负概率提升** ✅ 与方向推荐一致
- 让球方防线核心缺阵 → 穿盘能力下降（可能失球）+ 总进球增加 → **让平/让负**但也可能大球
- 冲突时：**伤停对方向的影响 > 对进球数的影响**

#### 因子G3：天气与场地

| 条件 | 对总进球的影响 | 修正幅度 |
|:----|:-------------|:--------:|
| 大雨/暴雨 | 球速快+场地积水，进球减少 | **-0.4球** |
| 大风(>30km/h) | 长传精准度下降，偶然性增加 | **-0.2球** |
| 高温(>30°C) | 体能消耗快，下半场节奏放缓 | **-0.3球** |
| 低温(<5°C) | 球员发挥受影响，进球减少 | **-0.2球** |
| 人工草皮（主队适应vs客队不适） | 适应方优势扩大，总进球分布偏移 | **±0.2球**（偏向适应方） |
| 高原主场（>2500m） | 客队体能迅速消耗，上半场或大球下半场缩 | **-0.3球**（下半场尤其明显） |

**注**：天气信息从 59itou 阵容Tab或外部天气API获取。无数据时不修正。

#### 因子G4：历史交锋进球模式

| H2H进球特征 | 含义 | 修正 |
|:-----------|:----|:----|
| 近5场交锋场均总进球 > 两队赛季均值+0.5 | 两队存在\\\"对攻基因\\\" | **+0.3球** |
| 近5场交锋场均总进球 < 两队赛季均值-0.5 | 两队相遇时互相克制 | **-0.3球** |
| 近5场交锋中≥3场总进球≤2 | 历史小球局 | **-0.4球** |
| 近5场交锋中≥3场总进球≥4 | 历史大球局 | **+0.4球** |
| 首次交锋或无H2H数据（如杯赛） | 无历史参考 | 按模型计算 |

**H2H进球模式 vs 联赛场均对比**：
- 若H2H进球模式与联赛场均方向一致 → 确认信号，修正生效
- 若矛盾 → 降权50%，仅作参考

#### 因子G5：大小球盘口对比（市场验证）

当能从详情页亚指Tab获取大小球盘口时（部分联赛/时段可用），将模型总进球预测与市场大小球盘口对比：

```python
# 获取市场大小球盘口（如2.5球）
market_ou = 2.5  # 从亚指Tab提取

# 模型预测的总进球期望
model_expected_goals = sum(k * total_goal_prob[k] for k in range(17))

# 判断价值
edge = model_expected_goals - market_ou

if edge > 0.5:
    # 模型显著高于市场 → 大球有价值
    大球信号 🟢
elif edge < -0.5:
    # 模型显著低于市场 → 小球有价值
    小球信号 🟢
else:
    # 模型与市场一致 → 无价值
    无显著信号 ⚪
```

**大小球市场验证的使用规则**：
- 只有能获取到大小球盘口时使用
- edge > 0.8 → 强信号，进球数预测上调一档（如2→3球作为第一候选）
- edge < -0.8 → 强信号，进球数预测下调一档（如3→2球作为第一候选）
- 无法获取大小球盘口时跳过此因子

### 进球数三步验证法

每次输出进球数前，按以下顺序验证：

```
第1步：λ模型计算 → 得到top3总进球（数学基础）
  ↓
第2步：应用G1-G4修正因子 → 调整排序（实战修正）
  ↓
第3步：G5大小球市场验证 → 终选确认（市场验证）
```

**各因子冲突时的优先级**：
```
G2（伤停）> G1（赛程）> G3（天气）> G4（H2H）> G5（市场）
```

### 实战场景示例

**场景一：德甲 拜仁(-1) vs 不莱梅**
- λ₁=2.4, λ₂=0.9 → λ₁+λ₂=3.3 → 大球基线 ✅
- 联赛基因：德甲场均3.0~3.3 → 大球联赛 ✅
- 伤停：拜仁头号射手凯恩缺阵 → G2修正：-0.5球
- 修正后：λ₁+λ₂=2.8 → top3: 3球/2球/4球
- 大小球盘口：3.0球 → 模型期望2.8 vs 市场3.0, edge=-0.2 → 无显著信号
- 最终：2球/3球/4球（注意让-1球场景下2球=让平走水）

**场景二：巴甲 弗拉门戈(-1) vs 戈亚斯**
- λ₁=1.6, λ₂=0.7 → λ₁+λ₂=2.3 → 中小球
- 联赛基因：巴甲场均2.2~2.4 → 小球联赛 ✅
- 赛程：弗拉门戈周中解放者杯 → G1修正：-0.3球
- 修正后：λ₁+λ₂=2.0 → top3: 1球/2球/0球
- 最终：1球/2球/0球（巴甲小球特征+双赛消耗=极可能小球）

### 进球数与方向推荐的协同

| 让球方向 | 进球数特征 | 验证 |
|:--------|:----------|:----|
| 让胜（穿盘） | 总进球 ≥ 让球数+2（如-1让球→总进球≥3） | 穿盘需要进球数支撑 |
| 让平（走水） | 总进球 = 让球数+1（如-1让球→总进球=2） | 走水比分锁定总进球 |
| 让负（下盘） | 总进球 ≤ 让球数（如-1让球→总进球≤1） | 下盘意味着让球方进球不足 |

**方向与进球数一致性检查矩阵**：

| 方向 \ 进球数 | 小球(0-1) | 中球(2-3) | 大球(4+) |
|:------------|:--------:|:---------:|:--------:|
| **让胜(穿盘)** | ❌矛盾 | ✅-1盘合理 | ✅-2盘合理 |
| **让平(走水)** | ❌矛盾 | ✅ | ⚠️罕见 |
| **让负(下盘)** | ✅ | ✅-1盘 | ❌矛盾 |

**处理规则**：
- ❌矛盾 → 输出中标注⚠️\"方向与进球数矛盾，建议重新验证数据\"
- ✅合理 → 正常输出
- ⚠️罕见 → 标注\"此组合概率较低，注意风险\"

### 联赛特异性进球阈值

| 联赛 | 小球阈值 | 大球阈值 | 说明 |
|:----|:--------:|:--------:|:----|
| 巴甲 | λ₁+λ₂ < 2.2 | λ₁+λ₂ > 3.0 | 巴甲天然小球，大球门槛更高 |
| 德甲/荷甲/美职联 | λ₁+λ₂ < 2.8 | λ₁+λ₂ > 3.5 | 天然大球联赛，小球门槛更高 |
| 挪超/英超 | λ₁+λ₂ < 2.4 | λ₁+λ₂ > 3.2 | 偏进攻但合理性中 |
| 意甲/法甲/葡超 | λ₁+λ₂ < 2.0 | λ₁+λ₂ > 2.8 | 偏防守，大球门槛最低 |
| J1/芬超 | λ₁+λ₂ < 2.0 | λ₁+λ₂ > 3.0 | 小球联赛，进球不易 |

**应用规则**：在判断\"大球概率高\"或\"小球概率高\"时，使用联赛特异性阈值替代通用阈值(2.0/3.0)。

### 复盘反馈：进球数准确率追踪

每次复盘时增加进球数准确率统计：

| 维度 | 统计方法 |
|:----|:--------|
| 精确命中 | 预测top3中包含实际总进球数 |
| 偏差1球 | 实际总进球与最近预测值差1球 |
| 方向命中 | 预测方向（大/小球倾向）与实际一致 |
| 完全偏差 | 实际总进球不在top3中且方向相反 |

**复盘改进**：累计≥30场数据后，按联赛统计进球数修正因子的有效性，优化各修正系数的幅度。
