# 详细单关推导输出格式（九维表+SP漂移+数据源）

## 何时使用
用户要求对选定场次做"深挖/详细分析/重新拉取实时数据/给我全部推导"时使用。

## 核心输出格式（严禁省略任何步骤）

```markdown
### #XX 🥇 主队(hcap)VS客队 → **单选方向**

**数据源：** [列表页/详情页战绩Tab+欧指Tab+亚指Tab]

| 维度 | 数据 | 判定 |
|:----|:-----|:----:|
| **①联赛势位** | [排名/积分/差] | [主/客/均势] |
| **②战意** | [保级/争冠/中游] | [级别] |
| **③主客战绩** | [主队主场NX胜] [客队客场NX负] | [判定] |
| **④近期状态** | [近N场X胜X平X负] | [判定] |
| **⑤阵容伤停** | [综合实力X:X/伤停] | [判定] |
| **⑥交战历史** | [H2H记录+主场不败/客场劣势] | [判定] |
| **⑦初盘指数** | [0盘/半球/百家赔率] | [合理/偏深/偏浅] |
| **⑧盘中走势** | [升X/降X家+高X低X家] | [方向判定] |
| **⑨战术风格** | [联赛特征] | [判定] |
| **⑩SP漂移** | [初版→临场变化 主+0.03/平-0.05/客+0.06] | [方向解读] |

**欧指信号归一化**：[升/降三线解读+一致性方向]

**数据漂移对比**：初版SP vs 临场SP → [变化趋势解读]

**核心决策链**：[九维→信号归一→SP验证→最终方向，1-3句]

✅ **单选[方向]** — [一句核心依据]
```

## 流程

1. **数据采集**（Stage 1 → 列表页获取SP、排名、让球数）
2. **数据漂移对比**（初版SP→临场SP变化，标注升/降幅度）
3. **详情页采集**（需深挖场次：战绩Tab+欧指Tab+亚指Tab）
4. **九维表填充**（以上述10维表格输出）
5. **欧指信号归一化**（升/降三线数据的综合方向判断）
6. **核心决策链**（九维→信号→SP验证→最终方向）
7. **输出**（单选方向+一句总结）

## 数据漂移分析的三种模式

| 漂移模式 | 含义 | 处理 |
|:---------|:-----|:-----|
| **主胜升+客胜降** | 资金涌入客队方，市场由主变客 | 审查客队基本面是否支撑→偏客 |
| **主胜降+客胜升** | 资金涌入主队方，市场信心强化 | 与初版判断一致→强化主队信心 |
| **平赔降(≥0.05)** | 市场防平 | 让平/走水概率上升，结合盘口判断 |
