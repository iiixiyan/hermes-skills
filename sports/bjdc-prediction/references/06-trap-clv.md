## 四、诱上与阻上识别框架

**第一层核心过滤**——先判断庄家意图，再看盘口。

| 水位+盘口变动 | 庄家意图 | 应对策略 |
|:--------------|:---------|:---------|
| **降水 + 盘口不动** | **诱上**——制造"这队稳了"假象 | 走**下盘** |
| **升盘 + 中高水** | **阻上**——让你觉得"贵了不敢买" | 走**上盘** |
| 升盘 + 水很低 | 两可，需结合基本面 | 小心观望 |
| 盘水都不动 | 放任热度积累 | 反向操作（热度→下盘，冷→上盘） |

**诱上盘——降水不升盘**：盘口不变但水位持续下降。既然这么看好，为什么不升盘？降水制造安全感，盘口不动暴露庄家并不真看好上盘。→ 走下盘。

**阻上盘——升盘中高水**：盘口升级（如半一→一球），同时水给中高。双重吓退跟注。→ 走上盘。

**三步练习法**：①看水位变化方向（降还是升）→ ②看盘口跟不跟 → ③看基本面验证

## 四-A、变盘时间节点权重体系

不同时间节点的盘口与水位变化蕴含不同的信息权重。早盘、中盘、临场的信号可信度差异显著。

### 时间节点划分

| 阶段 | 时间范围 | 权重系数 | 特征 |
|:----|:---------|:--------:|:-----|
| **早盘** | 开赛前72h-24h | 1.0（基准） | 机构初盘思维体现，反映基本面基础判断 |
| **中盘** | 开赛前24h-3h | 1.5 | 市场资金开始涌入，机构调整期，信息含量最高 |
| **临场** | 开赛前3h至开球 | 2.0 | 终极调整，包含最新资讯（首发、天气、突发伤停） |

### 各阶段信号解读规则

| 阶段 | 盘口变动 | 信号强度 | 解读 |
|:----|:---------|:--------:|:-----|
| **早盘变动** | 初盘开出后即大幅调整 | ⭐⭐ | 初步市场反应，但机构尚未充分导向，可信度中等 |
| **中盘升盘** | 盘口在资金涌入期升级 | ⭐⭐⭐⭐ | 真实看好（机构用升盘挡热），可信度最高 |
| **中盘降水不升盘** | 中盘持续降水但不升盘 | ⭐⭐⭐ | 诱上信号最可信阶段，机构希望中盘吸纳散户资金 |
| **临场剧烈变动** | 临场3h内大幅升降或翻转 | ⭐⭐⭐⭐⭐ | 包含最新资讯（首发名单、天气突变），必须重视 |
| **临场稳定** | 临场3h盘水锁定不动 | ⭐⭐⭐ | 机构对初盘判断自信，维持原有分析结论 |

### 综合应用规则

1. **多时段一致的信号 > 单时段信号**：早盘中盘临场都降水不升盘→极强诱上；仅临场降水→可能为信息驱动
2. **中盘信号权重 = 1.5×早盘信号权重**：分析时应优先关注中盘变化
3. **临场信号覆盖早/中盘**：若临场走势与早中盘方向相反，以临场为准（除非确认无重大信息变化）
4. **时间缺口警惕**：若早盘→中盘→临场之间有长时间无变化（>6h），突然变化的风险更高
5. **信号速查表中的阈值在临场阶段可放宽5-10%**（如临场降水速率加快，诱上信号门槛从"持续降水"降为"2h内快速降水"）

## 四-B、临场CLV监控与对冲策略（v4.1 新增）

> 解决赛前突发状况（首发名单、天气突变、更衣室消息）无应对体系的问题。

### 核心原则

**CLV（Closing Line Value）= 赛前1小时内赔率变化方向 → 判断市场最终立场。**

### 临场监控三步法

```
Step 1（赛前1h）：重拉北单三赔 + 欧指信号
Step 2（对比早盘）：初盘方向 vs 临场方向
Step 3（决策）：
  ├─ 方向一致 → 维持原判断，权重×1.1
  ├─ 方向反转 → CLV警报，降2档信心
  └─ 赔率大幅波动(>15%) → 触发对冲评估
```

### CLV对冲触发条件

| 触发条件 | 含义 | 操作 |
|:--------|:-----|:-----|
| 初盘判让胜，临场让胜赔率↑>10% | 市场放弃初盘方向 | 原方向减注50%，剩余50%转向反向 |
| 初盘判让负，临场让胜赔率↓>15% | 市场大幅倾向反向 | **放弃原判断**，转为反向或撤单 |
| 临场三赔剧烈震荡（任意两项±>20%） | 疑似幕后消息 | **立即撤单**，不参与 |
| 盘口完全翻转（如-1→+1） | 极端信号 | **全额反向对冲** |

### 对冲计算公式

```
原始投注 = 1000元（方向：让胜@2.50）
临场让负赔率 = 2.80
CLV警告触发（让胜赔率升>10%）

对冲金额 = 原始投注 × (初盘赔率 / 临场反向赔率)
         = 1000 × (2.50 / 2.80) = 893元

→ 补买893元让负，无论结果：保本或小亏
```

### 撤单铁规

以下情况**无条件撤单**：
- 首发名单中让球方核心球员（射手/门将）缺席
- 天气导致比赛推迟或场地条件极差
- CLV监控触发「极端波动」级别
- 盘口完全翻转
