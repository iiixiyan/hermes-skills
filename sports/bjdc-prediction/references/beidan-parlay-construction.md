# 北单串关组合构建方法论

> 2026-07-04 实战沉淀：14串1、9串1（4双选5单选）两种经典模式

---

## 一、适用场景

用户要求从北单预测中提取多场组成串关时使用。典型触发词："给我N串1"、"加一场"、「综合考虑」。

## 二、两步法流程

### Step 1：逐场分析（单场全量）

每一场必须走完完整分析链后，才能进入组合阶段：

```
基本盘 → 信号分析 → 规则链追踪 → 对抗式审查 → 🎯双选+🎯单选
```

**数据源优先级**（北单专用）：
1. **列表页SP**（让球胜平负三赔）— 三赔最低=市场真实看好方向
2. **59itou欧指Tab** — URL参数 `current_tab=odds&lotteryId=45` 直取，提取上升/降低家数
3. **新浪API** — 仅限同一比赛同时存在于竞足系统时（如世界杯）
4. **联赛特性** — 冰岛超/挪甲/巴西乙/爱超/爱甲各自的规则（见SKILL.md联赛特性表）

### Step 2：组合构建

#### 2.1 类型A：全单选串关（如13串1、14串1）

每条=1注×2元，风险最高但回报最大。

**选取规则**：
| 安心度 | 首选联赛 | 原因 |
|:------|:---------|:-----|
| 🥇 最稳 | **巴西乙** | 6月持续高效~83%，排名优势规则稳定 |
| 🥇 次稳 | **爱甲** | 6月优秀~80%，-1盘下盘规则有效 |
| 🥈 可靠 | **世界杯深盘** | +2盘71.7%让平统计+规则23 |
| 🥉 谨慎 | **冰岛超** | 0盘需含让平，无极端信号走SP方向 |
| ⚠️ 高风险 | **挪甲** | 极端波动，仅SP极低值(≤2.0)可信 |

**禁止放入全单选串关的场次**：
- 信号矛盾（SP方向 vs 欧指方向相反）
- 无欧指数据仅有SP支撑的挪甲/冰岛超
- 规则23(-2深盘) — 穿盘概率<20%

#### 2.2 类型B：双选+单选混合串关（如4双选5单选）

双选场次=2注覆盖，容错空间。

**双选选取规则**：
| 场景 | 双选理由 | 双选方案 |
|:----|:---------|:---------|
| 规则23(-2盘) | 穿盘难 | 让胜/让负(两端)或让负/让平(偏下盘) |
| +1/+2盘 | 让平率27.7%/71.7% | 让负/让平(客不败)或让胜/让平(主不败) |
| 挪甲0盘 | 极端波动 | 让胜/让负(两端) |
| 冰岛超0盘 | 走水率27.2% | 让负/让平或让胜/让平(含让平) |
| 信号矛盾 | 欧指与SP方向相反 | 含让平双选(受力面) |

**单选选取规则**：
- 巴西乙排名优势 → 单选让胜/让负(走排名方向)
- 爱甲SP极低(≤1.80) → 单选让负
- 世界杯深盘 → 单选让平(走水方向)
- 芬甲/芬超非极端 → 单选SP最低方向

## 四、铁律

1. **每场先做完整分析再组合**，禁止跳过单场分析直接进入组合
2. **双选+单选必出**，无例外（v5.8.0规则27-30）
3. **北单SP三赔最低≠最终方向**，需欧指验证一致性
4. **挪甲0盘禁止偏重**（极端波动），仅SP≤2.0的强信号可出手
5. **巴西乙排名前3可颠覆4b信号**（规则21）
6. **-2盘禁止偏单选穿盘**（规则23）
7. **信号矛盾时以59itou欧指为准**，非SP
