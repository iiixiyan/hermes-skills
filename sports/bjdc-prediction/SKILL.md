## 边界与铁纪

- ✅ **仅北单**（lotteryId=45），竞足问题请转 `football-prediction`
- ✅ **场次拆分：早场 vs 晚场分别处理**
  - **早场（12:55~19:00）**：J1联赛、K2联赛、芬超、芬甲、瑞典甲下午场等 → 在11:00前完成采集+预测
  - **晚场（22:00~06:00）**：挪超、瑞典超、芬超晚场、罗甲、意乙、法甲、爱超、冰岛超、智利甲、巴西乙等 → 在22:00前完成采集+预测
  - 两批次互不重叠，各管各的时间段
- ❌ 禁止北单分析中输出精确比分（如2-1、1-0）
- ❌ 禁止使用 λ 公式/xG 模型/竞足输出格式
- 🚨 **‼️ 禁止使用「三赔/SP最低=方向」反推逻辑** — SP最低≠市场共识方向。SP最低往往是庄家压低赔率引导投注（诱盘），而非真实方向。方向必须来自九维预测法（基本面）+实力盘定位法（盘口信号链）的独立判定。SP仅用于Edge验证（模型概率vs市场概率）和串关过滤（SP≥2.1确保预期收益）。当SP最低方向=基本面方向时验证有效，反之需要重新审查基本面判断。
- ✅ 逐场采集逐场分析：每场单独完成后立即输出。当用户明确请求范围（如"89-129"）时，允许用表格批量输出。
- ⚠️ **批量场景两阶段法（40+场比赛时强制使用）**：
  - **Stage 1（列表页筛查）**：从北单列表页 `kt.59itou.com/883/danchang/` 一次性提取全部比赛的核心数据：matchID、SP三赔、联赛、主客队、让球数、排名[Rank]。将列表页 innerText 解析为结构化数据。📌 40+场比赛的列表页数据提取约需1次 browser_console 调用（15秒），远快于逐场详情页采集。
  - **Stage 1 判定依据**：使用排名差 + 联赛特异性信号规则 + 偏重判定8级优先级（⚠️ 严禁SP三赔最低反推方向！SP仅用于Edge验证和串关过滤，方向来自排名+联赛+偏重规则），输出批量表格（含编号/联赛/主客队/让球/SP三赔/推荐方向/信心）。
  - **Stage 2（用户点选深挖）**：用户从批量表格中点选某场后，对该场做完整双模推导（含详情页欧指/亚指Tab采集 + 诱上阻上水位判定 + Edge计算）。
  - ⚠️ 批量表格输出时每场标注"需要完整推导请点选场次号"，引导用户互动。
- ⚠️ **批量场景用户铁律：一气呵成**：用户说"全部"或给一个范围(如89-129)后，所有场次必须在连续的回复中出完，禁止停等用户指令"继续"。每场用一行或一个区块呈现，表格格式即可。
  - 批量表格必须包含：编号/时间/主客队/联赛/让球/SP三赔/推荐方向/信心
  - 用户从表中点选某场要求详情时，再对该场做完整双模推导
- ❌ **禁止混用竞足数据到北单分析**：北单lotteryId=45，竞足lotteryId=90，两套数据严格隔离
⚠️ **Hermes浏览器Tab限制**：59itou北单详情页的欧指/亚指Tab在Hermes内置浏览器中不可切换（Vue SPA路由问题）。战绩Tab始终可靠。欧指不可用时，分析回退为：列表初盘三赔最低方向 + 战绩Tab数据（近10场/H2H/综合实力），信心降一星。

🆕 **北单跨平台欧指获取（世界杯/大赛跨平台比赛）**：当北单中的比赛同时存在于竞足（新浪API）平台时（如世界杯R32淘汰赛），可直接通过新浪API footballMatchOddsEuro 获取欧指8参数数据，无需依赖59itou详情页Tab切换。通过北单列表页的联赛名+队名+时间匹配Sina竞足 matchId，即可获取完整欧指信号（r1/c1/r2/c2/r3/c3）。此方式完全绕过59itou SPA Tab限制。2026-06-30实战验证：6场北单世界杯比赛全部成功获取Sina欧指数据。

## 流程（精简版8步） 🔴 CHECKPOINT

1. **数据采集** 🔴 逐场采集/批量筛查二选一
2. **模式一：八维预测法**（基本面主导）→ `references/09-prediction-methods.md`
3. **模式二：实力盘定位法**（盘口主导）→ `references/09-prediction-methods.md`
4. **泊松模型基础计算** — 市场校准λ+统计λ+双变量泊松+DC τ修正
5. **信号检查与Edge检测** — 因子检查+联赛阈值+Edge≥5%出击
6. **诱上阻上与盘口验证** — 降水不升盘=诱上，升盘中高水=阻上
7. **进阶工具交叉验证** — 必发交易量+凯利排雷+大小球协同
8. **双模交叉验证与输出** — 两模一致→单选，分歧→并列+偏重/放弃
9. **（按需）SP≥2.1过关组合构建** — 当用户要求"必赢"/"稳赢"时：
   - **约束**：每项SP≥2.1（用户刚需），总投入=2^双选数×2元
   - **双选覆盖策略**：主不败(3+1) / 客不败(1+0) / 分胜负(3+0)，选最可靠的
   - **联赛优先级**：挪超≥瑞典超>芬甲>冰岛超>意乙>爱超>爱甲（信号可靠性排序）
   - **组合公式**：4双选+5单选=16注32元；5双选+5单选=32注64元
   - ⚠️ **"必赢"是信心表述，非绝对保证** — 必须标注双选覆盖范围而非误导
   - 详见 `references/14-cross-source-validation.md`

> 详细8步含进球数预测：`references/12-goal-prediction.md`

### ⚡ 强制前置流程：数据核查 → 偏重判定（v5.5.1强制规则）

**任何预测必须按以下顺序执行，禁止跳过数据核查直接进入偏重判定：**

```
第一步：数据核查前置条件（5项）
  ↓ 全部通过 → 进入偏重判定
  ↓ 任一项触发 → 执行对应动作后，禁止做偏重单选，仅保留双选
第二步：偏重判定10级
  ↓ 命中即停
第三步：双选策略匹配
  ↓
第四步：输出
```

**🚨 常见违规案例（26069期发现）：**
- #44 克罗地亚: 让球方近4场全输盘→应禁止偏让胜, 但跳过核查直接偏让胜❌
- #42 克里西乌马: 排名#5让-1给排名#3→应禁用统计默认值, 但跳过核查套用"巴西乙-1盘75%"❌
- #46 刚果金: 极端一致信号+基本面同向→应判定为真防护, 但跳过核查用"造热陷阱"❌

**禁止仅按让球数分组做分析。必须逐场采集59itou战绩Tab数据（近10场、主客场、综合实力、排名、H2H），然后综合所有因素进行判断。**

### 偏重判定10级优先级（v4.9.4优化版）

> 基于#129场复盘修正：战意权重提升、伤停新增、H2H前提条件细化

按顺序检查，命中即停：

| 优先级 | 条件 | 偏重 |
|:---:|:---|:---|
| 1️⃣ | **核心球员伤停**（主力≥2人缺阵或核心射手/门将缺阵）| → 偏缺阵方反向 |
| 2️⃣ | **保级/争冠战意差≥2级**（非仅限于末5轮；全程适用，尤其客队保级vs主队无欲无求） | → 偏战意强方 |
| 3️⃣ | 因子4b触发（一致升主胜/客胜≥20家） | → 偏模式二 |
| 4️⃣ | 实力盘偏离≥1档 | → 偏模式二 |
| 5️⃣ | 联赛=瑞典超/挪超 | → 偏模式二 |
| 6️⃣ | H2H压倒性压制（≥4连胜或≥80%胜率 **且** 双方当前排名差≤5位，避免跨级压制误判） | → 偏模式一 |
| 7️⃣ | 联赛=爱超/爱甲/J1/丹甲/瑞典甲（信号完全不可用） | → 偏模式一 |
| ⚡ | **冰岛超0盘极端一致升主胜(≥25家)+降盘(≥5家)+高水(≥12家)：主队主场胜率≥55%→阻上实盘；弱队(<55%)→不防看衰** | → 强队偏模式一/弱队偏模式二 |
| ⚡ | **冰岛超+1盘极端一致信号→偏主场（+1盘75%走让平+让胜）** | → **双选让胜+让平偏模式一** |
| 8️⃣ | **极端水位背离**（高水≥15家且低水≤3家，同时基本面支持方与盘口矛盾） | → 偏盘口指向方 |
| 9️⃣ | 模式二无信号 | → 偏模式一 |
| 🔟 | 以上均不命中 | → 放弃/双选覆盖 |

**变更说明**：
- 原4️⃣(H2H压制)降为6️⃣，增加"排名差≤5"前提条件（H2H只有实力相当时才有参考意义）
- 原5️⃣(保级战意)从第5位升到第2位，且放宽至全程适用（不限于末5轮）
- 新增1️⃣伤停优先级（最高优先级，因核心球员缺阵直接改变实力对比）
- 新增8️⃣极端水位背离（#129场复盘发现：高水16家/低水2家+基本面主优→机构不防主胜的信号价值）
- 原6️⃣(联赛不可靠)保持不变为7️⃣
- 原7️⃣(模式二无信号)降为9️⃣
- 原8️⃣(放弃)变为🔟

## 复盘发现：一致升主胜信号的诱上/阻上区分规则（2026-05-29验证）

### 背景

- 博卡青年vs天主教大学（解放者杯）：胜升29家/降0家 + 亚指降盘16家/升盘0家 → 诱上型 → 实际让负 ✅

### 诱上/阻上/实盘力挺区分规则 🔴 CHECKPOINT（v4.9.6 全面修正版）

**盘理铁律（必须先背熟）：**
```
诱上：必须给甜头！热门方基本面好 + 浅盘/低水/降赔 = "稳如死狗"假象 → 走下盘
实盘力挺：必须设门槛！热门方基本面好 + 升盘/降水/降赔 = 庄家真怕打出 → 走上盘
不防/看衰：必须高赔利诱！热门方基本面好 + 升水/降盘 = 庄家不怕你买 → 走下盘
便宜化热门方：强队让深盘降至浅盘 = 降低门槛诱买强队 → 走弱队方向
强拉主队：基本面差的主队 + 强升盘/降主胜 = 利用盘口制造"有内幕"假象 → 走客队
明降暗阻：嘴上降赔（一致降主胜）+ 实际给极端高水 = 庄家嘴上怕实则不防 → 走下盘
```

**完整判定链：**
```python

# 热队 = 基本面占优（排名高/近况好/主场强/H2H压制）
# 冷队 = 基本面劣势

# Step 1: 看欧指方向

# 升主胜(4b) + 降盘 + 高水 → 不防/看衰（高赔利诱）
# 升主胜(4b) + 降盘 + 低水 → 诱上（低水给甜头）
```

1. 检测到「一致升主胜」时，**先判断热/冷队**：主队基本面好=热队，主队基本面差=冷队

## 操作规则补充（v5.2.17新增 — FIFA差陷阱/排名倒挂/信号同向）

15. ⚠️ **FIFA差陷阱**：FIFA差巨大(≥30)不代表让球方能打穿盘口。必须检查让球方近5场的**赢盘率/盘口表现**——基本面占优≠盘口能覆盖。案例：克罗地亚(#11)-1vs加纳(#73)FIFA差62位偏让胜，但克罗地亚近4场全部输盘，结果2-1让平❌
16. ⚠️ **排名反转让深盘=异常信号**：当让球方的联赛排名低于受让方时（如第5名让-1球给第3名），这是一个盘口异常信号，**不可直接套用联赛统计默认值**。必须核查：①是否主场加成过强 ②受让方客场战绩 ③综合实力差是否匹配盘口。案例：克里西乌马(巴乙5)-1vs圣贝尔纳多(巴乙3)「巴西乙-1盘75%」偏让胜，但排名低的队让深盘，实际1-0让平❌
17. ⚠️ **极端一致信号需核验基本面方向**：当极端一致升/降主胜(≥25家一致)且基本面(排名/综合实力/客场战绩)与信号方向一致时，判定为**真防护非造热陷阱**。仅当基本面与信号方向矛盾时才考虑造热陷阱。
18. ⚠️ **极端一致降主胜+基本面逆向→强拉主队（巴西乙专修），但需注意极端信号强度阈值**：巴西乙中「极端一致降主胜(≥25家)+基本面逆向(客队排名/近况/客场战绩更优)」= 强拉主队诱盘 → 走客队方向。案例：6/28#62福塔莱萨(3-1-6近况差)vs累西腓体育(5-3-2+客场6-3-1) — 极端降主胜26/1+升平22/1+升负26/1+强升盘7家，基本面客队优→强拉主队，偏让负。⚠️ **例外阈值**：当三线全极端(降主胜≥25+升平≥20+升负≥20+百家主胜降幅>5%+强升盘≥5家+H2H优势)同时满足，即使基本面逆向，也可能为真防护而非强拉主队。处理：双选两端(让胜+让负)，不偏重单选。
19. ⚠️ **明降暗阻+基本面碾压级强队→双选偏基本面方向**：当「极端一致降主胜(≥20家)+强升盘(≥8家)+极端高水(≥15家)」=明降暗阻模式触发时，若基本面为**碾压级优势**(近10场胜率≥80%+综合实力差≥30点+FIFA/排名差≥25位)，不可机械走下盘。因基本面优势过于极端可部分抵消明降暗阻的信号价值。处理：双选两端(让胜+让负/让胜+让平)，偏基本面方向。案例：6/28#66德国(-1)vs巴拉圭——极端降主胜24/1+强升盘13家+极端高水16家=明降暗阻，但德国近10场9胜1负进31失9+综合66:34+FIFA差31位碾压级优势→偏让胜(双选保险)。
20. ⚠️ **巴西乙4b升主胜+极端高水+排名差≥5位→偏模式一（基本面）**：巴西乙中「4b一致升主胜(≥20家)+极端高水(≥15/≤3)」触发不防/看衰信号时，若同时满足(排名差≥5位+综合实力≥50+主场），则基本面优势可部分抵消信号指向。处理：偏模式一(基本面)→偏让胜/让平。案例：6/28#60米纳斯吉拉(#11)1-0阿瓦伊(#18) — 4b升主胜21/6+极端高水18/3+排名差7+综合52:48+主场→实际让胜✅。反例：#63累西腓航海(#9)0-1戈亚斯(#12) — 排名差3<5不触发，仍走信号方向让负✅。

## 六、思维模型 — 第一性原理与对抗式审查

> 以下两种思维模型是贯穿所有北单预测与复盘的最高层方法论。每次分析前调用，作为认知框架的初始化。

### 6.1 概述

将第一性原理（回归本质）和对抗式审查（挑战假设）融入北单预测与复盘，能显著提升分析的深度与准确性：

- **第一性原理** —— 从零开始，用最基础的客观事实重建对比赛的理解，不依赖欧指信号表面或盘口叙事
- **对抗式审查** —— 主动扮演"魔鬼代言人"挑战自己的偏重判定、信号解读和方向选择

### 6.2 第一性原理（First Principles）—— 回归本质的预测

核心：不直接看欧指升降家数或亚指盘口水位，从最基础的比赛事实出发构建判断。

#### 6.2.1 赛前预测应用

**① 拆解到基本要素：**

| 基本要素 | 客观数据 | 对应北单数据源 |
|:--------|:---------|:--------------|
| 球队实力 | 联赛排名、FIFA排名、综合实力分(0-100) | 59itou排名Tab + 战绩Tab |
| 近期状态 | 近10场胜率/进球/失球、主客场战绩分开 | 59itou战绩Tab |
| 阵容完整度 | 核心球员是否缺阵、预计首发/阵型 | 59itou阵容Tab + 情报Tab |
| 战意 | 保级/争冠/争欧战资格，赛程密集度 | 排名Tab + 联赛积分榜 |
| H2H历史 | 历史交锋中盘口与赛果的关系 | 59itou历史Tab |
| 盘口本质 | 让球数反映的真实实力差 vs 预期差 | 列表页让球数 + 排名 |

**② 从要素重建分析：**

- **问题1：让球数合理吗？**
  排名差 vs 让球数：比如排名#1让-1给排名#10 → 合理；排名#5让-1给排名#3 → **盘口异常信号**，需要追问原因（主场加成过强？还是人为造热？）

- **问题2：欧指信号可信吗？**
  欧指一致升主胜 ≠ 必然走下盘。先问：**如果我没有欧指数据，仅凭排名/状态/伤停，我的方向判断是什么？**信号与基本面一致 → 可信；信号与基本面矛盾 → 造热/诱盘预警

- **问题3：这次盘口博弈的操纵基础是什么？**
  庄家在什么情境下需要诱？什么情境下需要阻？强队基本面好+低水浅盘 = 给甜头诱上；弱队基本面差+升盘 = 强拉主队诱客

**③ 第一性原理校验清单（每次偏重判定前自查）：**

```
□ 如果没有任何欧指信号（升降家数为0），仅凭排名/状态/伤停/H2H，我的方向是什么？
□ 当前欧指信号和基本盘推断一致吗？如果不一致，哪个更有第一性依据？
□ 这个"一致升主胜"信号，是市场真实看衰主队，还是庄家利用基本面向好造热？
□ 我的偏重判定是否过度依赖某个"联赛统计默认值"而没有回到本场比赛的基本事实？
```

#### 6.2.2 赛后复盘应用 — 从底层数据归因

**① 用基本数据代替主观评价：**
- ❌ "这场机构诱导成功/失败"
- ✅ "让球方排名#6近5场2胜1平2负，客场受让方排名#3近5场4胜1负——让球方基本面劣势，即使一致降主胜也不该做胆"

**② 从数据追溯根本原因：**
- 偏重方向错了：是因为偏重判定跳过了数据核查前置条件？还是信号解读错误（诱上误判为实盘）？
- 双选没覆盖：排除的方向为什么打出来了？排除的逻辑链在哪一步断了？

### 6.3 对抗式审查（Adversarial Review）—— 挑战假设的复盘

核心：主动扮演"魔鬼代言人"挑战偏重判定、诱上阻上识别和联赛信号可靠性判断。

#### 6.3.1 赛后复盘应用

模拟场景：复盘某场"一致升主胜+强降盘"的比赛，你判断为诱上走下盘，实际走上盘。启动对抗式审查：

**① 挑战数据解读：**
- 你的分析："一致升主胜≥20家+强降盘≥5家=诱上走下盘"
- 🔍 对抗式挑战：**"降盘后的水位数是多少？** 如果降盘后主队水位依然维持在低水(≤1.85)，这可能是阻上（挡买主队）而非诱上（给甜头）。低水才是诱上的必要条件——你确认了水位数据吗？"

**② 挑战事件归因：**
- 你的分析："这场爆冷是因为信号准确"
- 🔍 对抗式挑战：**"有没有可能是基本面本来就不支持让球方？** 排名倒挂（让球方排名<受让方）才是真正的原因，而一致升主胜只是刚好和基本面同向——不是信号"预测"了结果，而是结果本来就应该如此。"

**③ 挑战隐藏假设：**
- 你的分析："4b一致升主胜在这个联赛一贯有效"
- 🔍 对抗式挑战：**"这个联赛最近3期的信号准确率是多少？** 如果最近3期准确率在下降（如巴西乙从100%→67%→50%），你的"一贯有效"假设已经过时了。信号可靠性需要持续验证，不是固定的。"

#### 6.3.2 赛前预测应用 — 压力测试偏重方向

| 你的偏重 | 对抗式挑战 | 应对 |
|:---------|:----------|:-----|
| 偏模式一（基本面） | 基本面的数据是否完整？缺少伤停/H2H数据时基本面可信度打几折？ | 数据不完整时降级信心，加双选覆盖 |
| 偏模式二（盘口信号） | 这个信号在本联赛最近5期的准确率是多少？还是说你是印象流？ | 查references/05-league-templates.md或当期复盘数据 |
| 单选（偏让胜） | 如果这个让胜没打出，最可能走让平还是让负？ | 必须提前确认双选组合，让胜+让平还是让胜+让负 |

**对抗式审查输出规范（最终输出前强制执行）：**

```markdown
🛡️ 对抗式审查风险清单：
- [高/中/低] 风险1：[风险说明]
- [高/中/低] 风险2：[风险说明]
```

**对抗式审查四问（每次偏重判定前强制执行）：**

```
🔍 对抗式审查 —
1. 我的偏重判定最脆弱的前提假设是什么？（如果这个假设错了，方向会反转吗？）
2. 有没有我故意忽略的反面证据？（哪些信号/数据不支持我的方向？）
3. 如果我必须给这场比赛的对家（反方向）写分析理由，我能写出几条有力的论据？
4. 这个联赛最近3期的信号准确率趋势如何？是上升、下降还是稳定？
```

### 6.4 在现有流程中的集成点

| 现有步骤 | 第一性原理集成 | 对抗式审查集成 |
|:---------|:-------------|:--------------|
| 数据采集 | 只看原始innerText，不输入任何预判方向 | 每个数据源反问：这个数据可靠吗？有没有交叉验证？ |
| 偏重判定10级 | 先做第一性原理校验清单，再进入偏重判定 | 每级偏重触发后，对抗式挑战：是否有反证？ |
| 诱上阻上识别 | 回到盘口本质：庄家为什么要在这个场景诱/阻？ | 挑战自己的诱上/阻上判断：是否有其他解读？ |
| Edge检测 | Edge>5%→追问：这个Edge来自模型还是噪音？ | 如果Edge信号方向与基本面相悖，谁的优先级更高？ |
| 双选策略 | 双选覆盖的逻辑必须基于基本事实而非统计默认值 | "这个盘口的让平率24.8%是历史统计——本场比赛本身符合这个统计的前提条件吗？" |

### 6.5 铁律 — 两种思维模型的关系

```
第一性原理 → 构建（Build）：从原子级别理解比赛（无盘口信号时）
    ↓
对抗式审查 → 检验（Test）：审视构建的方向是否被信号干扰
    ↓
          → 弱信号 → 回到第一性原理重新拆解
          → 强信号 → 输出偏重判定/双选策略
```

两者循环迭代。单一使用任何一种都会产生盲区——只有第一性原理没有对抗式审查，会无视造热陷阱；只有对抗式审查没有第一性原理，会陷入"一切信号都是陷阱"的质疑主义，而错过真正的市场共识信号。
