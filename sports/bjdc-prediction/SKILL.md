## 边界与铁纪

- ✅ **仅北单**（lotteryId=45），竞足问题请转 `football-prediction`
- ✅ **场次拆分：早场 vs 晚场分别处理**
  - **早场（12:55~19:00）**：J1联赛、K2联赛、芬超、芬甲、瑞典甲下午场等 → 在11:00前完成采集+预测
  - **晚场（22:00~06:00）**：挪超、瑞典超、芬超晚场、罗甲、意乙、法甲、爱超、冰岛超、智利甲、巴西乙等 → 在22:00前完成采集+预测
  - 两批次互不重叠，各管各的时间段
- ❌ 禁止北单分析中输出精确比分（如2-1、1-0）
- ❌ 禁止使用 λ 公式/xG 模型/竞足输出格式
- 🚨 **‼️ 禁止使用「三赔/SP最低=方向」反推逻辑** — SP最低≠市场共识方向。SP最低往往是庄家压低赔率引导投注（诱盘），而非真实方向。方向必须来自九维预测法（基本面）+实力盘定位法（盘口信号链）的独立判定。SP仅用于Edge验证（模型概率vs市场概率）和串关过滤（SP≥2.1确保预期收益）。当SP最低方向=基本面方向时验证有效，反之需要重新审查基本面判断。
- ✅ 逐场采集逐场分析：每场单独完成后立即输出。当用户明确请求范围（如"89-129"）时，允许用表格批量输出。
- ⚠️ **批量场景两阶段法（40+场比赛时强制使用）**：
  - **Stage 1（列表页筛查）**：从北单列表页 `kt.59itou.com/883/danchang/` 一次性提取全部比赛的核心数据：matchID、SP三赔、联赛、主客队、让球数、排名[Rank]。将列表页 innerText 解析为结构化数据。📌 40+场比赛的列表页数据提取约需1次 browser_console 调用（15秒），远快于逐场详情页采集。
  - **Stage 1 判定依据**：使用排名差 + 联赛特异性信号规则 + 偏重判定8级优先级（⚠️ 严禁SP三赔最低反推方向！SP仅用于Edge验证和串关过滤，方向来自排名+联赛+偏重规则），输出批量表格（含编号/联赛/主客队/让球/SP三赔/推荐方向/信心）。
  - **Stage 2（用户点选深挖）**：用户从批量表格中点选某场后，对该场做完整双模推导（含详情页欧指/亚指Tab采集 + 诱上阻上水位判定 + Edge计算）。
  - ⚠️ 批量表格输出时每场标注"需要完整推导请点选场次号"，引导用户互动。
- ⚠️ **批量场景用户铁律：一气呵成**：用户说"全部"或给一个范围(如89-129)后，所有场次必须在连续的回复中出完，禁止停等用户指令"继续"。每场用一行或一个区块呈现，表格格式即可。
  - 批量表格必须包含：编号/时间/主客队/联赛/让球/SP三赔/推荐方向/信心
  - 用户从表中点选某场要求详情时，再对该场做完整双模推导
- ❌ **禁止混用竞足数据到北单分析**：北单lotteryId=45，竞足lotteryId=90，两套数据严格隔离
- ⚠️ **Hermes浏览器Tab限制**：59itou北单详情页的欧指/亚指Tab在Hermes内置浏览器中不可切换（Vue SPA路由问题）。战绩Tab始终可靠。欧指不可用时，分析回退为：列表初盘三赔最低方向 + 战绩Tab数据（近10场/H2H/综合实力），信心降一星。
- ✅ **终选前必须重新拉取欧指数据**

## 流程（精简版8步）

1. **数据采集** — 列表页→matchID+三赔→详情页6Tab（单场模式）或 列表页SP+排名+让球（批量Stage 1筛查模式）
2. **模式一：八维预测法**（基本面主导）→ `references/09-prediction-methods.md`
3. **模式二：实力盘定位法**（盘口主导）→ `references/09-prediction-methods.md`
4. **泊松模型基础计算** — 市场校准λ+统计λ+双变量泊松+DC τ修正
5. **信号检查与Edge检测** — 因子检查+联赛阈值+Edge≥5%出击
6. **诱上阻上与盘口验证** — 降水不升盘=诱上，升盘中高水=阻上
7. **进阶工具交叉验证** — 必发交易量+凯利排雷+大小球协同
8. **双模交叉验证与输出** — 两模一致→单选，分歧→并列+偏重/放弃
9. **（按需）SP≥2.1过关组合构建** — 当用户要求"必赢"/"稳赢"时：
   - **约束**：每项SP≥2.1（用户刚需），总投入=2^双选数×2元
   - **双选覆盖策略**：主不败(3+1) / 客不败(1+0) / 分胜负(3+0)，选最可靠的
   - **联赛优先级**：挪超≥瑞典超>芬甲>冰岛超>意乙>爱超>爱甲（信号可靠性排序）
   - **组合公式**：4双选+5单选=16注32元；5双选+5单选=32注64元
   - ⚠️ **"必赢"是信心表述，非绝对保证** — 必须标注双选覆盖范围而非误导
   - 详见 `references/14-cross-source-validation.md`

> 详细8步含进球数预测：`references/12-goal-prediction.md`

### 偏重判定10级优先级（v4.9.4优化版）

> 基于#129场复盘修正：战意权重提升、伤停新增、H2H前提条件细化

按顺序检查，命中即停：

| 优先级 | 条件 | 偏重 |
|:---:|:---|:---|
| 1️⃣ | **核心球员伤停**（主力≥2人缺阵或核心射手/门将缺阵）| → 偏缺阵方反向 |
| 2️⃣ | **保级/争冠战意差≥2级**（非仅限于末5轮；全程适用，尤其客队保级vs主队无欲无求） | → 偏战意强方 |
| 3️⃣ | 因子4b触发（一致升主胜/客胜≥20家） | → 偏模式二 |
| 4️⃣ | 实力盘偏离≥1档 | → 偏模式二 |
| 5️⃣ | 联赛=瑞典超/挪超 | → 偏模式二 |
| 6️⃣ | H2H压倒性压制（≥4连胜或≥80%胜率 **且** 双方当前排名差≤5位，避免跨级压制误判） | → 偏模式一 |
| 7️⃣ | 联赛=爱超/爱甲/J1/丹甲/瑞典甲（信号不可靠） | → 偏模式一 |
| 8️⃣ | **极端水位背离**（高水≥15家且低水≤3家，同时基本面支持方与盘口矛盾） | → 偏盘口指向方 |
| 9️⃣ | 模式二无信号 | → 偏模式一 |
| 🔟 | 以上均不命中 | → 放弃/双选覆盖 |

**变更说明**：
- 原4️⃣(H2H压制)降为6️⃣，增加"排名差≤5"前提条件（H2H只有实力相当时才有参考意义）
- 原5️⃣(保级战意)从第5位升到第2位，且放宽至全程适用（不限于末5轮）
- 新增1️⃣伤停优先级（最高优先级，因核心球员缺阵直接改变实力对比）
- 新增8️⃣极端水位背离（#129场复盘发现：高水16家/低水2家+基本面主优→机构不防主胜的信号价值）
- 原6️⃣(联赛不可靠)保持不变为7️⃣
- 原7️⃣(模式二无信号)降为9️⃣
- 原8️⃣(放弃)变为🔟

## 复盘发现：一致升主胜信号的诱上/阻上区分规则（2026-05-29验证）

### 背景
2026-05-29（周五）北单复盘发现：相同「一致升主胜」信号在不同比赛中指向截然不同的结果：
- 博卡青年vs天主教大学（解放者杯）：胜升29家/降0家 + 亚指降盘16家/升盘0家 → 诱上型 → 实际让负 ✅
- 波特诺山丘vs水晶体育（解放者杯）：胜升21家/降7家 + 亚指无降盘（主队仍让胜） → 阻上型 → 实际让胜 ✅

### 诱上/阻上/实盘力挺区分规则（v4.9.6 全面修正版）

**盘理铁律（必须先背熟）：**
```
诱上：必须给甜头！热门方基本面好 + 浅盘/低水/降赔 = "稳如死狗"假象 → 走下盘
实盘力挺：必须设门槛！热门方基本面好 + 升盘/降水/降赔 = 庄家真怕打出 → 走上盘
不防/看衰：必须高赔利诱！热门方基本面好 + 升水/降盘 = 庄家不怕你买 → 走下盘
便宜化热门方：强队让深盘降至浅盘 = 降低门槛诱买强队 → 走弱队方向
强拉主队：基本面差的主队 + 强升盘/降主胜 = 利用盘口制造"有内幕"假象 → 走客队
明降暗阻：嘴上降赔（一致降主胜）+ 实际给极端高水 = 庄家嘴上怕实则不防 → 走下盘
```

**完整判定链：**
```python
# Step 0: 判断热/冷队
# 热队 = 基本面占优（排名高/近况好/主场强/H2H压制）
# 冷队 = 基本面劣势

# Step 1: 看欧指方向
# 降主胜 + 强升盘 + 低水 → 实盘力挺（真怕主队打出）
# 升主胜(4b) + 降盘 + 高水 → 不防/看衰（高赔利诱）
# 升主胜(4b) + 降盘 + 低水 → 诱上（低水给甜头）
# 降客胜 + 降盘(便宜化客队) → 诱客队，走主队方向

# Step 2: 检查基本面与盘口是否一致
# 热队基本面好 + 实盘力挺信号 → 一致，可信
# 热队基本面好 + 不防/看衰信号 → 背离，信盘口
# 冷队(基本面差) + 强升盘/降主胜 → 强拉主队(诱)，走客队
# 热队(基本面好) + 降盘(降低门槛) → 便宜化热门方(诱)，走反方向
```

**诱上/阻上/实盘力挺判定表：**

| 类型 | 基本面 | 欧指特征 | 亚指特征 | 盘理解读 | 实际倾向 |
|:---|:------|:---------|:---------|:--------|:--------:|
| **诱上（给甜头）** | 热队基本面好+市场热度高 | 胜升≥20(4b) | 降盘≥5家+**低水** | 让热队"看起来更稳"，低水制造安心感 | **让负/下盘** |
| **不防/看衰（高赔利诱）** | 热队基本面好+市场热 | 胜升≥20(4b) | 降盘≥5家+**高水** | 高赔吸引博热队方向，庄家不怕赔付 | **让负/下盘** |
| **实盘力挺（真防）** | 热队基本面好 | 一致降主胜≥20 | **强升盘≥8家+低水** | 升盘增加门槛+低水减少赔付，真怕打出 | **让胜/上盘** |
| **强拉主队（诱）** | **冷队基本面差** | 降主胜 | **强升盘**  | 利用盘口制造"庄家有内幕"假象吸引主队资金 | **客不败** |
| **便宜化热门方（诱）** | 强队基本面碾压 | 降强队赔率 | **降盘(降低门槛)** | 强队从深盘降至浅盘，降低买入难度 | **弱队方向** |
| **明降暗阻** | 基本面占优方 | 一致降主胜≥20 | **极端高水(≥15高/≤3低)** | 嘴上降赔示好，实际高水不防，矛盾信号 | **走下盘** |
| **阻上拉下型** | 热队基本面**差/烂** | 主升极端+客降极端+平降极端 | 强降盘至浅盘 | 利用烂状态吓跑热队资金，制造客队陷阱 | **平局最受保护** |

### 操作规则
1. 检测到「一致升主胜」时，**立即检查基本面**：主队状态差还是好？
2. ⚠️ **关键区分**（v4.9.5新增）：
   - 主队基本面**好** + 4b升主胜 + 亚指降盘 → 诱上（利用基本面造热）
   - 主队基本面**差/烂** + 4b升主胜 + 亚指降盘 → **阻上拉下**（利用烂状态吓跑资金，降客胜+降平做热客队，亚指降盘设陷阱）
3. 检测到「极端一致降平（≥20家降）」时→ **平局是最被保护的结果**，尤其西乙等高平局联赛
4. 若一致降主胜(强>升) + 基本面验证主队状态好 → 让胜/上盘

### 操作规则
1. 检测到「一致升主胜」时，**先判断热/冷队**：主队基本面好=热队，主队基本面差=冷队
2. 若一致升主胜+主队是热队+亚指降盘+**高水** → **不防/看衰，走下盘**（高赔利诱）
3. 若一致升主胜+主队是热队+亚指降盘+**低水** → **诱上，走下盘**（低水给甜头）
4. 若一致降主胜+主队是热队+**强升盘+低水** → **实盘力挺真防，走上盘**（⚠️非"叠杀"）
5. 若一致降主胜+主队是热队+**强升盘+极端高水** → **明降暗阻，走下盘**
6. 若一致降主胜+主队基本面**差**+强升盘 → **强拉主队(诱)，走客队**
7. 若强队降盘(降低门槛) → **便宜化热门方，走弱队方向**
8. 若一致升主胜极端+主队基本面差+客降≥20+平降≥20+强降盘 → **阻上拉下，平局最受保护**
9. 芬甲/小联赛：一致降主胜信号可靠率高
10. ⚠️ **西乙让-1盘：永远把「让平」放在首位防范，慎博让胜**
11. ⚠️ **冰岛超降权0.3倍，深盘不可信**
