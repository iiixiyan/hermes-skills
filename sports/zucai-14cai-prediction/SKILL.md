# 足彩14场胜负彩综合预测体系（v1.0.0）

> ⚽ **足彩14场**是传统足球彩票的经典玩法——从指定14场比赛中选择胜平负结果，14场全中=一等奖，猜中13场=二等奖。

---

## ⚠️ 技能边界

**此技能仅适用于传统足彩14场胜负彩（14场胜平负），不适用于竞足单场比分或北单让球胜平负。**

| 维度 | 竞足 → football-prediction | 足彩14场 → 本技能 |
|:----|:--------------------------|:------------------|
| 输出内容 | 单场比分预测（λ计算、xG模型） | 14场胜平负打包 + 胆/双选分配 |
| 预测方式 | 逐场独立比分 | 14场联动筛选（胆优先、双选保底） |
| 核心策略 | 正常比分+爆冷+大比分三轨 | 胆+双选+全包三级分层 |
| 奖金模型 | 单关SP | 一等奖/二等奖金额预估 |
| 输出格式 | 〖竞足-XXXX〗 | 🏆 14场预测表（胆→双选→全包） |

**混淆禁止**：
- ❌ 足彩14场输出中不得输出精确比分预测（只输出胜平负）
- ❌ 不得使用λ公式/xG模型（14场只需3-way结果）
- ❌ 不得使用竞足输出格式头

## 三、三级分层分析法

### 3.1 核心概念

足彩14场的核心是将14场比赛按确定度分为三级：

| 等级 | 名称 | 含义 | 覆盖方式 | 每个选项的成本 |
|:---:|:----|:-----|:--------|:--------------|
| 🟢 **胆** | 单选 | 非常确信的结果 | 只选1个结果（胜/平/负） | 1注 |
| 🟡 **双选** | 双选 | 不太确定，排除一个结果 | 选2个结果（如胜+平排除负） | 2注 |
| 🔴 **全包** | 全选 | 完全无法判断 | 选全部3个结果 | 3注 |

**注数计算公式**：
```
总注数 = 2^双选数量 × 3^全包数量
总金额 = 总注数 × 2元
```

**示例**：
- 3胆 + 5双选 + 6单选 → 2^5 = 32注 = 64元
- 4胆 + 4双选 + 6单选 → 2^4 = 16注 = 32元
- 2胆 + 6双选 + 6单选 → 2^6 = 64注 = 128元
- 5胆 + 6双选 + 3单选 → 2^6 = 64注 = 128元（胆多双选也多=合理）

### 3.2 胆（Banker）选择标准

胆是14场预测的基石——选对了就大幅缩小范围，选错了全盘皆输。

**胆的选场标准**：
| 标准 | 说明 | 优先度 |
|:----|:-----|:------:|
| 实力碾压 | 主强客弱且排名差≥10位 | ⭐⭐⭐ |
| 主场龙 vs 客场虫 | 主队主场胜率高(≥60%) vs 客队客场胜率低(≤20%) | ⭐⭐⭐ |
| 赔率≤1.50 | 欧指主胜/客胜低赔，市场共识强 | ⭐⭐⭐ |
| H2H碾压 | 近5场相同对阵全胜 | ⭐⭐ |
| 战意明确 | 保级队vs已保级队、争冠队vs无欲无求 | ⭐⭐ |
| 伤停有利 | 核心无伤停+对手核心伤缺 | ⭐⭐ |

**禁止用作胆的场景**：
- ❌ 德比战（结果不可预测）
- ❌ 双方近况均差（菜鸡互啄）
- ❌ 友谊赛/杯赛小组末轮（战意不明）
- ❌ 让球深盘（-1.5以上，赢球输盘可能）
- ❌ 赔率≤1.20的超低赔（回报率低+爆冷风险不成正比）

**胆的验证步骤**：
1. 查基本面：排名差、近况、H2H
2. 查欧指：主胜是否一致降赔（至少15家以上降）
3. 查亚指：盘口是否支持（主让≥0.5且水位合理）
4. 查伤停：核心球员是否齐整
5. 查战意：有无额外动力（保级/争冠/争欧战）

### 3.3 双选策略

双选是14场盈利的关键——胆选得好+双选覆盖得好=中奖概率最大化。

**常用双选组合**：
| 类型 | 覆盖方向 | 适用场景 | 覆盖率 |
|:----|:---------|:---------|:------:|
| 主不败 | 胜+平 | 主队稍强但平局可能 | 66.7% |
| 客不败 | 平+负 | 客队更强或主队状态差 | 66.7% |
| 分胜负 | 胜+负 | 双方都善攻不善守、历史分胜负多 | 66.7% |
| 防平局 | 胜+平或平+负 | 排名接近、欧指平赔低 | 66.7% |

**双选优先级规则**：
1. 主实力稍优但客状态好 → 主不败（胜+平）
2. 客实力优但主主场强 → 客不败（平+负）
3. 双方实力接近 + 历史无平 → 分胜负（胜+负）
4. 排名接近 + 欧指平赔一致降 → 双选含平局
5. 强队做客但数据支撑不足 → 客不败（平+负）

**特殊情况**：
- 双方状态都差 → 优先双选含平局
- 联赛末轮战意差异大 → 偏战意方方向双选
- 杯赛单场淘汰 → 分胜负优先

### 3.4 全包策略

全包（选3个结果）只应在以下情况使用：
- 极端不确定性（德比、保级生死战、教练下课首场）
- 双方实力、状态、H2H完全均衡
- **注意**：每个全包使注数×3，成本暴涨，建议控制≤2个全包

## 四、逐场分析流程

### 第1步：确认14场场次
- 从59itou列表页提取全部14场对阵、联赛、开赛时间
- 整理成14场对阵表

### 第2步：逐场数据采集
对每一场比赛，采集详情页数据（按前面2.2节的8Tab要求）

### 第3步：逐场胜平负分析（简化版）

对于足彩14场，**不需要精确比分预测**，只需要判断3-way方向。分析方法：

**A. 方向判定矩阵**（每场打分后取最高分）

| 维度 | 主胜(3) | 平局(1) | 客胜(0) | 权重 |
|:----|:-------:|:-------:|:-------:|:----:|
| 联赛排名差 | +2~+5分 | 0分 | +2~+5分(客优) | ×2 |
| 近6场积分差 | +2~+3分 | +1分(平率>50%) | +2~+3分 | ×1.5 |
| 主客场表现 | +3分(主场强) | +1分(主场弱) | +2分(客场强) | ×1.5 |
| H2H优势 | +2~+3分 | +1分(连平) | +2~+3分 | ×1 |
| 欧指一致性 | +3分(一致降) | +2分(平赔降) | +3分(一致降) | ×1.5 |
| 伤停影响 | 核心缺阵-3分 | 双方均缺阵-1分 | 核心缺阵-3分 | ×2 |
| 战意差 | +3分(强战意) | 0分(无) | +3分(强战意) | ×1 |

**B. 简化判定法**（快速筛查时用）：
- 排名差≥8位 + 欧指主胜降≥15家 + 主队无核心伤停 → **胆（主胜）** 🟢
- 排名差≥8位 + 欧指客胜降≥15家 + 客队无核心伤停 → **胆（客胜）** 🟢
- 排名差3~7位 + 信号一致但不极端 → **双选（主不败/客不败）** 🟡
- 排名差≤2位 + 信号分歧 → **双选（含平局）** 🟡 或全包 🔴
- 欧指平赔一致降≥15家 → **双选含平局** 🟡

### 第4步：胆/双选/全包分配

1. **先定胆**：从14场中筛选出可做胆的场次（3~5个）
   - 必须有明确的实力/市场/基本面三重确认
   - 胆不能超过5个（太多=风险集中）

2. **再定双选**：剩余场次中，哪些需要双选（4~6个）
   - 不确定但方向可排除一个结果的
   - 优先覆盖「主不败」或「客不败」类型双选（方向明确）

3. **剩余为单选**：剩下的场次在胆确认后也作为单选输出

4. **全包极少量**：极端不确定时才用（≤2个）

### 第5步：注数计算与预算优化

| 预算范围 | 推荐结构 | 注数 | 金额 |
|:--------|:---------|:----|:----:|
| ≤32元 | 5胆+4双选+5单选 | 2^4=16 | 32元 |
| 32~64元 | 4胆+5双选+5单选 | 2^5=32 | 64元 |
| 64~128元 | 4胆+6双选+4单选 | 2^6=64 | 128元 |
| 128~256元 | 3胆+7双选+4单选 | 2^7=128 | 256元 |

**常用推荐**：
- 🟢 **稳单**：4胆+5双选+5单选 = 32注/64元（14场中13场保本概率高）
- 🟡 **均衡**：3胆+6双选+5单选 = 64注/128元（兼顾1等和2等）
- 🔴 **博单**：4胆+6双选+4单选 = 64注/128元（高命中率博一等奖）

### 第6步：奖金预期

**奖金估算公式**：
- 一等奖 ≈ 当期销量×65% / 中奖注数
- 二等奖 ≈ 一等奖金额×30%~50%
- 保本级：若14中13 → 二等奖金额 ≥ 投注金额

**奖金档次参考**（经验值）：
- 🏆 14场正确 → 一等奖，金额取决于当期销量和中奖人数
- 🥈 13场正确 → 二等奖，大约为一等奖的30%~50%
- ❌ ≤12场正确 → 无奖

## 六、奖金优化模型

### 6.1 SP值加权策略

当有各场次的SP赔率数据时，可做奖金优化：

**目标**：在保证覆盖率的前提下，最大化潜在奖金/投入比

**操作**：
1. 胆的SP需≥1.30（太低则做胆的价值小）
2. 双选方向中SP较高的结果需能覆盖2倍注数成本
3. 全包场次的SP分布需合理（任意结果SP>1.50）

### 6.2 缩水策略（高级）

对于512注以上的大方案，可按以下规则缩水：
- 同联赛多场比赛 → 去掉最不可能的组合
- 某队连续双选结果 → 保留核心方向
- 按1/3缩水率控制

## 七、复盘方法

### 7.1 逐场命中率分析
- 胆的命中率（胆错=全盘皆输，优先级最高）
- 双选覆盖率（双选是否覆盖了实际赛果）
- 总命中场次（14中N）

### 7.2 错误归因
- 胆错了：实力判断失误？伤停影响？还是意外爆冷？
- 双选没覆盖：排除的结果恰恰打出了？
- 定期记录错误模式 → 优化胆选择标准

### 7.3 复盘输出格式


## 九、常见问题与陷阱

### ⚠️ 常见陷阱

| 陷阱 | 说明 | 应对 |
|:----|:-----|:-----|
| 胆太多(>5) | 胆越多风险越集中，一个胆错就崩 | 控制在3~5个 |
| 全包太多(>2) | 每多一个全包注数×3，成本爆炸 | 尽量不用全包，或≤2个 |
| 忽视周中欧战 | 周中有欧战的强队联赛可能轮换 | 查赛程表确认周中消耗 |
| 友谊赛/国家队比赛做胆 | 友谊赛不确定性极高 | 禁止用国家队友谊赛做胆 |
| 联赛末轮战意误判 | 保级/争冠/欧战资格影响巨大 | 查积分榜确认各队战意 |
| 双选不含平局 | 很多比赛中平局概率被低估 | 排名接近+欧平平赔降→强制含平 |

### 🛟 异常处理

| 场景 | 处理方式 |
|:----|:---------|
| 14场列表页无数据（无可售期） | 反馈"当期无销售期"，等下一期 |
| 某场详情页采集不到 | 用列表页SP数据+基本面简化分析 |
| 多场同联赛同时开赛 | 按联赛特点批量处理（同一模板） |
| 比赛延期/取消 | 标记取消，减少一个胆或双选 |
