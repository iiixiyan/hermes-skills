# 竞彩足球比分预测（v4.5.0 重构版）

> 采用 Progressive Disclosure 架构。本文件仅放触发条件、核心流程、引用索引。详细方法论见 references/ 目录。

---

## ⚠️ 技能边界

**此技能仅用于竞彩足球（竞足）分析，不适用于北单（北京单场）。**

| 维度 | 竞足 → football-prediction | 北单 → bjdc-prediction |
|:----|:--------------------------|:-----------------------|
| 输出内容 | 比分预测（λ计算、xG模型、双轨比分） | 北单选项（让胜/让平/让负） |
| 核心模型 | 泊松分布λ加权、状态权重、攻防结构 | 盘口技术手册、诱/阻判定、信号速查 |
| 输出格式 | 〖竞足-XXXX〗 + 正常比分/异常比分 | 📊 北单推荐：让胜/让平/让负 |

**混淆禁止**：
- ❌ 北单分析中不得输出比分预测
- ❌ 北单分析不得使用football-prediction的λ公式和xG模型
- ❌ 北单分析不得使用〖竞足-XXXX〗格式头

## 二、核心流程

### 第1步：信息确认
确认赛事、队名、场次编号。缺失则主动询问。

### 第2步：数据采集
- 竞足列表 → `kt.59itou.com/627/jingcai/` 提取 matchID
- 北单列表 → `kt.59itou.com/883/danchang/`（仅用于matchID，不走本技能）
- 逐场详情页（lotteryId=90）6Tab innerText采集
- **文彩52强制交叉验证** → 见 references/ 已有文件

### 第3步：因子检查
遍历因子速查表（14因子），逐因子核对触发状态。详见 `references/01-factor-table.md`。

| 优先级 | 因子 | 触发条件（简） |
|:--|:--|:--|
| **P0** | 4b 一致升赔 | ≥20升/≤2降 → **执行前强制检查失效条件(杯赛决赛/中立场地/换帅)** → 异常比分须含反方向 |
| **P0** | 2 赛季末战意 | 末5轮+战意差 → 异常权重+2级 |
| **P0** | 14 弱队客场λ系数 | 客排名>主5名 或 跨级杯赛 → λ客×0.70~0.80 |
| **P1** | 12 一致降赔 | ≥20降/≤2升+基本面异常 → 造热预警 |
| **P1** | 13 双边分歧 | 一端≥20升+另端≥20降 → 平局+35%（⚠️解放者杯降级） |
| **P1** | 4b-EXT 边界升赔 | ≥20升/≤5降+幅度≥3% → 标记 |

### 第4步：模型计算
```
4.1 λ_stats  = (近3×0.4 + 近6×0.3 + 赛季×0.3) × 主客场系数
4.2 弱队客场λ修正（因子14+跨级系数）→ references/analysis-framework.md §4.2a
4.3 λ_market = 网格校准（argmin|P_model - P_market|²）
4.4 λ_transitive = 共同对手桥接法 → references/05-bridge-method.md
4.5 λ_final = α×λ_market + β×λ_stats + γ×λ_transitive
```

### 第5步：异常比分路径判定
在A1-A4（爆冷）和B1-B4（大比分）中匹配触发类型。详见 `references/02-abnormal-score-paths.md`。

### 第6步：双轨比分输出（含让球胜平负）

**让球数提取**：从详情页欧指/亚指Tab提取让球数。竞足列表页每场也有让球标记（如-1、+1等）。若无法提取→默认为0。

**让球胜平负推导规则**：
```python
# 机械计算：给定比分→对应的让球结果
# 用途：理解每个预测比分对应的让球选项
让球数 = N（竞足官方让球数，如-1=主队让1球，+1=客队让1球）
比分差 = 主队进球 - 客队进球

if 让球数 <= 0:  # 主队让球或平手
    if 比分差 > abs(让球数): 结果 = 让胜
    elif 比分差 == abs(让球数): 结果 = 让平
    else: 结果 = 让负
else:  # 客队让球(主队受让)
    if 比分差 >= 0: 结果 = 让胜
    elif 比分差 == -1: 结果 = 让平
    else: 结果 = 让负
```

⚠️ **关键区分：让球结果的「机械计算」vs「独立预测」**
- 上方的推导规则仅用于**把比分翻译成让球选项**（如2-0在-1盘下=让胜）
- 最终输出的「让球：[让胜/让平/让负]」必须是**独立预测**，严禁用比分反推
- 正确做法：用亚指盘口水位数据（升盘/降盘公司数、水位方向）独立判断让球方向
- 示例：让球-1+预测比分2-0→让胜，但亚指显示多家机构降盘+主队高水→穿盘存疑→最终判让平

示例：主队-1球
| 比分 | 净胜 |  让球结果 |
|:----|:---:|:---------:|
| 2-0  | +2  | ✅ 让胜 |
| 1-0  | +1  | ✅ 让平 |
| 1-1  | 0   | ✅ 让负 |
| 0-1  | -1  | ✅ 让负 |

**输出格式**：
```
〖竞足-XXXX〗 [主队] VS [客队]（[联赛]）⚽ 让球[±N]
📊 [近6轮数据摘要；伤停简述]
🔍 结构：[正路/失衡]（λ1=X.XX λ2=X.XX 差X.XX）
⚡ 关键：[影响走向的核心因素，≤35字]

📌 因子检查
- 4b一致升赔：[触发/未触发]（X家升/X家降）
- 12一致降赔：[触发/未触发]
- 13双边分歧：[触发/未触发]
- 14弱队客场λ：[触发/未触发]

🎯 正常比分：[比分A]/[比分B]
🎯 胜平负：[胜/平/负]
🎯 让球：[让胜/让平/让负]

🎯 异常比分：[比分A]/[比分B] 🏷️ [A1/B1等]

⭐ 信心：★★★☆☆
```

**推导规则**：
- **正常比分**：双选，λ模型选出最可能的2个比分覆盖
- **胜平负**：模型综合λ差+因子检查+市场信号独立判定（非从比分反推）。例：因子4b一致升主胜→即使预测2-1，胜平负也可能=负
- **让球**：模型综合盘口水位+欧指信号独立判定（非从比分+让球数换算）。例：让球-1+比分2-0，但亚指水位不支持穿盘→可能让平而非让胜
- **异常比分**：双选，含异常路径标签

## 四、输出铁规

### 核心格式（v4.4.1最终版）
```
🎯 正常比分：[比分A]/[比分B]     ← 双选，λ模型最可能的2个
🎯 胜平负：[胜/平/负]            ← 单独1个，独立推导
🎯 让球：[让胜/让平/让负]         ← 单独1个，独立推导

🎯 异常比分：[比分A]/[比分B] 🏷️ [A1/B1等]  ← 双选含标签
```

### 铁律
- ✅ **正常比分**：必须双选（2个比分），λ模型选最可能的2个覆盖
- ✅ **胜平负**：模型综合λ差+因子+市场信号**独立判定**，**严禁从比分反推**
  - 例：因子4b一致升主胜→即使预测2-1，胜平负也可能=负（冷门预警）
  - 例：λ差<0.30+因子12一致降赔→即使预测主胜，胜平负也可能=平
- ✅ **让球**：模型综合盘口水位+欧指信号**独立判定**，**严禁从比分+让球数反推**
  - 例：让球-1+比分2-0，但亚指水位不支持穿盘→让平而非让胜
  - 判断依据：亚指升盘公司数/降盘公司数、主队水位高水/低水公司数、盘口走势
- ✅ **异常比分**：必须双选，须标注类型标签（🔴A1-A4 / 🟠B1-B4）+ 触发条件
- ✅ 每场必须输出让球胜平负结果（让球数从59itou列表页(±N)提取）
- ❌ 禁止用比分反推胜平负或让球（必须基于因子独立判断）
- ❌ 禁止爆冷和大比分混在同一组
- ❌ 禁止"仅供参考"等免责套话
- ❌ 禁止跳过λ计算（近3/近6必须展开，赛季可简写）
- ❌ 禁止使用简化脚本/模板化输出
- ⚠️ 终选前必须重新拉取欧指数据（因子4b会漂移）
- ⚠️ **欧指Tab指数变化段缺失时，必须重新加载详情页**（browser_navigate+等待2s后再取）。连续2次取空→标记"欧指数据不完整"并使用百家平均直接判断升降方向（初赔vs即赔差价≥5%=有效升降）
- ⚠️ **因子4b执行前强制检查失效条件**：杯赛决赛❌、中立场地❌、近3场换帅❌。任一命中→4b降级为标记仅参考（不从P0执行）
- ⚠️ **价值回归识别规则**（v4.5.0新增）：当主胜≥20升/≤2降触发4b，但同时满足"主队近10场胜率≥70%"且"主胜初赔<2.0"且"平局降≥20家+客胜降≥20家"→判定为"市场价值回归"（初赔过低的市场修正），4b降级为P1参考，异常比分主队不胜方向可选不强制

### 串关输出铁规
- ❌ 禁止展开λ加权计算
- ❌ 禁止输出完整单场分析格式
- ❌ **禁止5串1中出现重复场次**（5场必须来自5场不同比赛）
- ✅ 用表格呈现场次+比分+信号+入选理由
- ✅ 必须附剔除场次及理由
