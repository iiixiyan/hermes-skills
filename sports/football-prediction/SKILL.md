## ⚠️ 技能边界

**此技能仅用于竞彩足球（竞足）分析，不适用于北单（北京单场）。**

| 维度 | 竞足 → football-prediction | 北单 → bjdc-prediction |
|:----|:--------------------------|:-----------------------|
| 输出内容 | 比分预测（λ计算、xG模型、三轨比分） | 北单选项（让胜/让平/让负） |
| 核心模型 | 泊松分布λ加权、状态权重、攻防结构 | 盘口技术手册、诱/阻判定、信号速查 |
| 输出格式 | 〖竞足-XXXX〗 + 正常比分/爆冷比分/大比分 | 📊 北单推荐：让胜/让平/让负 |

**混淆禁止**：
- ❌ 北单分析中不得输出比分预测
- ❌ 北单分析不得使用football-prediction的λ公式和xG模型
- ❌ 北单分析不得使用〖竞足-XXXX〗格式头

### 第4步：模型计算

**主引擎：`scripts/dixon_coles.py`**（Dixon-Coles τ修正 + 指数衰减加权 + 等级战绩xG代理 + Bayesian层次收缩 + 攻击×防守交叉乘积）

**⚠️ 核心公式（必须遵守！）：**

标准泊松模型要求 **比赛级λ = 进攻λ × 对方防守λ / 联赛平均**，而非仅用进攻λ。

```
λ₁（主队预期进球）= attack_home × defense_away / league_avg
λ₂（客队预期进球）= attack_away × defense_home / league_avg
```

其中 league_avg 从贝叶斯先验自动获取（如瑞超=1.35）。调用 `compute_match_lambdas()` 完成交叉乘积。

```
4.1 λ_stats = 指数衰减加权（scripts/dixon_coles.py §2）
    - 替代固定40/30/30，使用 exp_decay_weights(half_life=8)
    - 近1场权重最高，8场前比赛权重减半
    - 混合：近期加权90% + 赛季基准10%（防小样本过拟合）
    - 联赛季半衰期：
      · 日职/芬超 → half_life=6（赛季短，波动大）
      · 瑞超 → half_life=8（标准）
      · 五大联赛 → half_life=10（赛季长，样本充足）
      · 国际友谊赛 → half_life=4（间隔长，近期独家参考）
    → 详见 references/dixon-coles-integration.md §2

4.2 λ_xg = 等级战绩xG代理（scripts/dixon_coles.py §6）
    - 用59itou「对阵强队/弱队」实际进球平滑噪声，近似xG效果
    - ⚠️ 等级战绩数据来源：59itou战绩Tab（含跨赛季历史数据），使用前需确认时间跨度，避免包含球队降级/升级前的失真数据
    - 原理：对阵强队进球多=硬仗强→λ上修；对阵弱队失球多=防线差→λ_concede上修
    - 权重：赛季中场均50% + 对阵强队20% + 对阵弱队30%
    - 限制：最大偏离赛季均值±30%，样本<6场时降级回纯赛季
    - 防守端：对阵强队失球权重30%（抗压能力），对阵弱队失球20%（专注度）
    → 详见 references/dixon-coles-integration.md §6

4.3 λ修正（因子14弱队客场λ + 跨级系数 + 友谊赛收兵λ×0.85）

4.4 λ_bayesian = 贝叶斯层次收缩（scripts/dixon_coles.py §5）
    - **先验参数来源**：LEAGUE_PRIORS 字典基于历史赛季场均进球统计值，
      先验权重=等效观察场数，经验设定。瑞超(1.35,6)意味着6场先验观察。
    - 小样本（≤5场）→ 向联赛先验均值强收缩（防过拟合）
    - 大样本（≥15场）→ 信任数据，收缩弱
    - 输出：(后验λ, 置信区间68%/95%)
    - **分别收缩进攻λ和防守λ**：进攻λ向联赛均进收缩，防守λ向联赛均失收缩
    → 详见 references/dixon-coles-integration.md §5

4.4b ⚠️ 攻击×防守交叉乘积（关键步骤！— v4.9.1新增）
    - 调用 `compute_match_lambdas(attack_home, defense_home, attack_away, defense_away, league_avg)`
    - 输出：(λ₁, λ₂, λ_diff, balance)
    - λ_diff < 0.15 → 'balanced'（五五开）
    - 这是连接「球队独立λ」与「比赛级比分概率」的桥梁

4.5 λ_final = α×λ_bayesian_match + β×λ_market + γ×λ_transitive
    - λ_bayesian_match = 来自compute_match_lambdas的λ₁/λ₂（已含交叉乘积）
    - λ_market = 从欧指反推的市场λ（已含主场因素，故此处不再额外乘home_advantage）
    - λ_transitive = 共同对手桥接法（独立来源）
    - 默认 α=0.50, β=0.30, γ=0.20
    - 当样本量<5场时：α=0.60, β=0.25, γ=0.15（更依赖贝叶斯收缩）
    - **⚠️ HOME_ADVANTAGE 使用规则**：
      · 仅作用于 λ_bayesian（模型λ是中性数据，不含主场优势）
      · 不作用于 λ_market（欧指已隐含主场因素，重复乘会过度放大）
      · 不作用于 λ_transitive（桥接法基于实际赛果，含主场因素）
    - **⚠️ xG代理不重复计入**：λ_bayesian的输入已包含xG代理修正，
      融合时直接用 λ_bayesian，不再单独加γ×λ_xg项

4.6 比分概率分布 = Dixon-Coles τ修正泊松（scripts/dixon_coles.py §4）
    - 以融合后的 λ₁/λ₂ 为输入，调用 predict_match_scores()
    - τ因子自动修正低比分区域偏差（0-0/1-0/0-1/1-1）
    - 联赛特定ρ系数（日职-0.10, 瑞超-0.09, 芬超-0.08等）
    - 输出胜/平/负概率 + Top 4比分概率
    → 详见 references/dixon-coles-integration.md §1
```

### 第5步：异常比分路径判定（双轨分离）

**⚠️ 强制双选规则**：每场比赛必须输出 **爆冷比分2个 + 大比分2个**，共4个异常比分。用 `/` 分隔写在一行。严禁只给1个或漏掉某类。

**爆冷路径（A1-A4）**：弱队胜/平强队、主场溃败、保级队客场取分等。详见 `references/02-abnormal-score-paths.md §A`。

**大比分路径（B1-B4）**：进攻≥3球大胜、防线崩溃对攻战、进球数≥4等。详见 `references/02-abnormal-score-paths.md §B`。

### 第5.5步：反向检查 🔴 每步必须有作用

**⚠️ 用户铁律：管线中每一步都必须真实影响输出。承前启后，禁止噪音步骤。**

```
反向检查清单（每次预测完必须过一遍）：
□ 每个因子触发 → 实际修改了概率/λ/信心？还是只口头说了？
□ 指数衰减λ vs 赛季均值λ → 差异多少？差异<5%则该步骤无实质性贡献
□ xG代理λ vs 指数衰减λ → 是否真的修正了原始λ？
□ 贝叶斯收缩率 → 多少？<10%说明先验权重太低或样本足够大，该步作用有限
□ 因子4b触发后 → 是否真的有数值修改（平局×1.25、信心-1★等）？
□ λ₁和λ₂的差距来源 → 是来自攻击λ差，还是防守λ差，还是交叉乘积？
```

**口诀**：每一步的输出必须 ≠ 它的输入。如果相等，该步是噪音，应删除或重新设计。

**⚠️ 因子触发但无数值修正的处理规则**：当一个因子触发但检查后确认无需修改概率/λ/信心时，必须在输出中明确标注 `→ 信息标记，无需修正`，且附上原因说明（如"实力差距过大，非造热信号"）。禁止留下"触发但无动作"的模糊状态——要么修正，要么标注清楚为什么不需要修正。

### 第6步：双轨比分输出（含让球胜平负）

**让球数提取**：从详情页欧指/亚指Tab提取让球数。竞足列表页每场也有让球标记（如-1、+1等）。若无法提取→默认为0。

**让球胜平负推导规则**：详见 `references/analysis-framework.md §8`。关键区分：机械计算（比分→让球）vs 独立预测（盘口水位信号→让球），后者是正式输出依据。

**输出格式**：
```
〖竞足-XXXX〗 [主队] VS [客队]（[联赛]）⚽ 让球[±N]
📊 [近6轮数据摘要；伤停简述]
🔍 结构：[正路/失衡]（λ1=X.XX λ2=X.XX 差X.XX）
⚡ 关键：[影响走向的核心因素，≤35字]

📌 因子检查
- 4b一致升赔：[触发/未触发]（X家升/X家降）
- 12一致降赔：[触发/未触发]
- 13双边分歧：[触发/未触发]
- 14弱队客场λ：[触发/未触发]

🎯 正常比分：[比分A]/[比分B]
🎯 胜平负：[胜/平/负]  ← 推导依据
🎯 让球：[让胜/让平/让负]  ← 盘口信号依据

🔴 爆冷比分：[比分A]/[比分B] 🏷️ [A1-A4] [H2H历史印证]
🟠 大比分：[比分A]/[比分B] 🏷️ [B1-B4] [H2H历史印证]
⭐ 信心：★★★☆☆
⏱️ 数据时效：[采集时间HH:MM → 距今Xh]
```

### 输出示例（完整场次）

```
〖竞足-7008〗 AC奥卢 VS 雅罗（芬超 21:00）⚽让球-1
📊 AC奥卢(排3)近6场4胜0平2负；雅罗(排10)近6场2胜2平2负
🔍 λ₁=1.82 λ₂=0.95 diff=0.87 home_dominant
📌 4b升赔触发(25升/4降) | 14弱队λ触发(λ客×0.70)
🎯 正常：2-1/2-0 | SPF：胜 | 让球：让平/让负
🔴 爆冷：2-2/1-1 🟠 大分：3-1/3-2 ⭐★★★☆☆
```

> 来自5/31实战：**精确比分2-1命中✅，让平SP3.25打出。** 胜平负和让球独立推导，未从比分反推。

**推导规则**：
- **正常比分**：双选，λ模型选出最可能的2个比分覆盖
- **胜平负**：模型综合λ差+因子检查+市场信号**独立判定**，**严禁从比分反推**。例：因子4b一致升主胜→即使预测2-1，胜平负也可能=负
- **让球**：模型综合盘口水位+欧指信号**独立判定**，**严禁从比分+让球数反推**。例：让球-1+比分2-0，但亚指水位不支持穿盘→让平而非让胜
- **爆冷比分**：双选，含A类路径标签
- **大比分**：双选，含B类路径标签

### 第8步：比分深度复盘流程（⭐强制 — 每次复盘必须执行）

> **用户铁律：复盘必须以比分分析为唯一核心，比分命中率放在最前面。** 盘口信号、基本面验证是次要内容。
> 详见 `references/03-review-experience.md §复盘核心铁律` + `§系统偏差模式识别规程` + `§2026-06-01复盘补充`。

**比分深度复盘5维分析（缺一不可）：**

| # | 分析维度 | 方法 | 输出 |
|:-:|:--------|:-----|:-----|
| ① | **λ偏差量化** | 逐场计算"预测λ₁+λ₂ vs 实际总进球"偏差率 | 偏差矩阵，标出偏差率>30%场次 |
| ② | **半全场转化** | 按半场形态分组统计（0-0/客领先/主领先/对攻） | 半场→全场规律表 |
| ③ | **穿盘率分析** | 按让球数分组统计赢盘/走水/输盘 | 让-1穿盘率、让-2分化条件 |
| ④ | **SP价值漏斗** | 按SP分层（>5.0 / 2.0-5.0 / <2.0）统计命中 | 哪些预测最有价值 |
| ⑤ | **系统偏差识别** | 跨场次寻找共同偏差方向（如某联赛λ全部偏低） | 具体优化提案 ≥1条 |

**输出顺序（不可违反）：**
1. 🥇 比分命中率（精确/方向/偏差分布）
2. 🥇 λ偏差量化矩阵
3. 🥈 半全场转化 + 穿盘率 + SP价值（合并输出）
4. 🥉 系统偏差 + 优化方向

**公式：** `复盘输出 = 偏差矩阵 + 偏差归因 + 模式识别 + 优化提案`

## 附：批量预测模式（多场连续输出）

> 本附章作为一～五章的补充，适用于用户要求多场连续预测的场景。

**触发场景：** 用户要求"所有场次"、"继续"、或上下文隐含当天的完整预测覆盖。

### 铁律：一气呵成

```diff
- ❌ 做完一场停下来等用户说"继续"
+ ✅ 用户说了"继续"后 → 用同一个回复输出剩余所有场次完成预测
+ ✅ 用户说"全部预测" → 直接输出当天所有场次，禁止分批
+ ✅ 当回复长度受限时 → 直接追加后续回复，不等待用户指令
```

**规则背景**：用户习惯是"禁打断问'继续'要一气呵成"（见Memory）。一旦进入批量模式，所有场次必须在连续的回复中出完，中间不等待用户确认。

### 输出规格

批量模式下允许使用**半展开格式**以提高吞吐量，但必须保留以下核心字段不可省略：

| 字段 | 全量格式（单场） | 批量格式（多场） | 强制？ |
|:----|:--------------:|:---------------:|:------:|
| 场次/队名/联赛 | ✅ | ✅ | ✅ |
| 欧指关键信号 | 完整表格+分析 | 一句话信号摘要 | ✅ |
| xG代理效果 | 完整推导 | 仅输出修正前后对比数字 | ✅ |
| 交叉乘积λ | λ₁/λ₂/diff/balance | 同左 | ✅ |
| DC概率 | Top 6 | Top 4 | ✅ |
| 因子检查 | 完整逐项 | 仅触发的因子 | ✅ |
| 正常比分/胜平负/让球 | ✅ | ✅ | ✅ |
| 爆冷/大比分 | ✅ | ✅ | ✅ |
| 信心 | ✅ | ✅ | ✅ |
| 指数衰减详细计算 | ✅ | ❌ 省略（对精度无直接影响） | ❌ |
| 每步反向检查 | ✅ | ❌ 仅检查关键步骤（xG代理/交叉乘积/因子触发） | ❌ |
| 八维完整采集 | ✅ | ✅ 联赛场次必须完整，友谊赛可简化 | ✅/⚠️ |

**批量模式的格式模板（每场1个区块）：**

```
〖700X〗 [主] VS [客]（[联赛]）⚽[±N]
📊 [10字内数据摘要]
🔍 λ₁=X.XX λ₂=X.XX diff=X.XX [balance]
📌 [触发因子]: [一句话处理]
🎯 正常：[比分A]/[比分B]
🎯 SPF：[胜/平/负] | 让球：[让胜/让平/让负]
🔴 爆冷：[比分A]/[比分B] 🏷️ [A1-A4]
🟠 大分：[比分A]/[比分B] 🏷️ [B1-B4]
⭐ [X]★ | ⏱️ HH:MM
```

### 批量优先级规则

当剩余场次超过5场且时间紧迫时，按以下优先级处理：

```
🥇 联赛场次（瑞超/芬超/日职）→ 完整λ计算（含xG代理+贝叶斯+交叉乘积）
🥈 强弱分明的友谊赛（FIFA排名差>20）→ 简化计算（仅指数衰减+交叉乘积+友谊赛系数）
🥉 均衡友谊赛（FIFA排名差<20）→ 全量计算，但采集可以简化为仅欧指+战绩两Tab
```

### 跨批量的一致性声明

**禁止混用新旧λ计算方式**：在同一个批量输出中，所有场次的λ计算必须使用同一版本的管线。不允许一部分场次用v4.9.0（无交叉乘积），另一部分用v4.9.1（有交叉乘积）。如需切换版本，必须完成当前批次全部输出后进行。

## 四、输出铁规

### 核心格式
```
🎯 正常比分：[比分A]/[比分B]       # 双选
🎯 胜平负：[胜/平/负]  ← 推导依据  # 独立推导，严禁反推
🎯 让球：[让胜/让平/让负] ← 依据  # 独立推导，附盘口信号摘要

🔴 爆冷比分：[比分A]/[比分B] 🏷️ [A1-A4] [✅H2H历史印证]      # 双选（2个比分，用/分隔）
🟠 大比分：[比分A]/[比分B] 🏷️ [B1-B4] [✅H2H历史印证]      # 双选（2个比分，用/分隔）
⭐ 信心：[★~★★★★★]
⏱️ 数据时效：[采集时间HH:MM → 距今Xh]
```

### 铁律清单
- ✅ **正常比分**：必须双选（2个比分），λ模型选最可能的2个覆盖
- ✅ **胜平负**：模型综合λ差+因子+市场信号**独立判定**，**严禁从比分反推**
  - 例：因子4b一致升主胜→即使预测2-1，胜平负也可能=负（冷门预警）
  - 例：λ差<0.30+因子12一致降赔→即使预测主胜，胜平负也可能=平
- ✅ **让球**：模型综合盘口水位+欧指信号**独立判定**，**严禁从比分+让球数反推**
  - 判断依据：亚指升盘公司数/降盘公司数、主队水位高水/低水公司数、盘口走势
- ✅ **爆冷比分**：**必须双选（2个比分）**，用 `/` 分隔在一行，标注类型标签（🔴A1-A4）+ 触发条件 + H2H历史印证
- ✅ **大比分**：**必须双选（2个比分）**，用 `/` 分隔在一行，标注类型标签（🟠B1-B4）+ 触发条件 + H2H历史印证
- ✅ **爆冷比分的H2H历史比分不限制进球数**（允许3-2、4-2等大比分作为爆冷）
- ✅ 每场必须输出让球胜平负结果（让球数从59itou列表页(±N)提取）
- ❌ 禁止用比分反推胜平负或让球（必须基于因子独立判断）
- ❌ 禁止爆冷和大比分使用相同比分（必须差异化）
- ❌ 禁止"仅供参考"等免责套话
- ❌ 禁止跳过λ计算（近3/近6必须展开，赛季可简写）
- ❌ 禁止使用简化脚本/模板化输出
- 🚨 **‼️ 禁止使用「三赔/SP最低=方向」反推逻辑** — SP最低≠市场共识方向。SP最低往往是庄家压低赔率诱导投注（诱盘），而非真实方向。必须先通过八维预测法（基本面）+实力盘定位法（盘口信号链）独立确定方向，SP仅用于Edge验证（模型概率vs市场概率）和串关过滤（SP≥2.1确保预期收益）。
- ⚠️ **H2H连平/连败模式不可单独作为推荐依据**：H2H历史规律（如\"近6场5平\"）必须结合当前排名、近10场状态、伤停数据交叉验证。单独依赖H2H模式做推荐→用户反馈\"不稳\"⛔
- ⚠️ 终选前必须重新拉取欧指数据（因子4b会漂移）
- ⚠️ **欧指Tab指数变化段缺失时，必须重新加载详情页**（browser_navigate+等待2s后再取）
- ⚠️ **因子4b执行前强制检查失效条件**：杯赛决赛❌、中立场地❌、近3场换帅❌、**附加赛次回合·主队必须赢球方可晋级**❌。任一命中→4b降级为P1参考
- ⚠️ **价值回归识别规则**（v4.5.0）：当主胜≥20升/≤2降触发4b，但同时满足"主队近10场胜率≥70%"且"主胜初赔<2.0"且"平局降≥20家+客胜降≥20家"→判定为"市场价值回归"，4b降级为P1参考
- ⚠️ **友谊赛大小球缩减**：国际友谊赛λ应额外×0.85（收兵效应），或爆冷比分/大比分优先覆盖小球方向
- ⚠️ **挪超4b权重降级**：挪超联赛中因子4b从P0降级为P1+λ修正（2026-05-29验证：5/6场触发4b仅1场按预期方向走，其余被平局化解）。挪超4b触发时：①平局概率×1.25（定义：Dixon-Coles初始平局概率乘以1.25，上限不超过45%）②异常比分强制双选含平局 ③正路仅-1★而非-2★
- ⚠️ **瑞超4b权重降级**（2026-05-31验证）：参照挪超模式，瑞超4b一致升赔从P0降为P1参考。4b触发时：①平局概率×1.25（上限45%）②异常比分强制含平局 ③正路仅-1★
- ⚠️ **芬超平局信号失效**（2026-05-31验证）：芬超主客排名差≥3位时，因子13双边分歧和4b触发的平局信号降级为P2参考，优先走基本面方向（排名+实力差）。
- ⚠️ **因子12强弱悬殊比赛豁免**（2026-05-31新增—7003日本vs冰岛案例驱动）：当因子12触发（主胜≥20降/≤2升），但双方实力差距巨大（FIFA排名差>30或联赛排名差>10名）且主胜初赔<1.20时，因子12仅为信息标记，**无需数值修正**。原因：市场一致降赔是对合理实力差的正常反映，非造热信号。
- ⚠️ **友谊赛弱队状态权重降级**（2026-06-01新增—7011美国vs塞内加尔案例驱动）：当友谊赛客队具备"10+场不败"或"客场全胜"等统治级状态时，该状态信号**降级为P2参考**，不可作为方向判定的主力证据。原因：友谊赛对手质量参差不齐，客队不败记录可能源自对弱旅的碾压，而非真实统治力。降级处理：λ方向权重从β=0.30降至β=0.20，将差额0.10转移至λ_market（欧指市场信号）。
- ⚠️ **顶级强队友谊赛收兵系数上调**（2026-06-01新增—7012巴西6-2巴拿马案例驱动）：当主队FIFA排名<20且客队FIFA排名>30（FIFA差>25位）时，友谊赛收兵系数从×0.85上调至×0.90。原因：顶级强队在友谊赛中对弱旅的进攻压制力远强于一般强队，收兵效应不明显。巴西6-2巴拿马、德国4-0芬兰均印证此规律。此项与常规友谊赛收兵系数是叠加上下场关系：先判断是否顶级碾压→如是则×0.90，否则延续×0.85。|

### 🚫 反例与黑名单（禁止操作速查）

| # | 反模式 | 为什么不要做 | 替代做法 |
|:-:|:-------|:------------|:---------|
| 1 | **从比分反推胜平负或让球** | 胜平负和让球有独立的市场信号(因子4b/盘口水位)，从比分反推会丢失这些信号 | 胜平负→λ差+因子+市场信号独立判定；让球→盘口水位+欧指信号独立判定 |
| 2 | **SP最低=方向(诱盘反推)** | SP最低往往是庄家压低赔率诱导投注，而非市场共识方向 | 先八维预测法+实力盘定位法确定方向，SP仅用于Edge验证和串关过滤(SP≥2.1) |
| 3 | **H2H连平/连败单独做推荐** | H2H历史规律必须结合当前排名、近10场状态、伤停数据交叉验证 | 仅作为辅助参考，不单独作为推荐依据 |
| 4 | **爆冷和大比分使用相同比分** | 两个轨道应覆盖不同赛果方向，相同比分等于少了一个覆盖 | 爆冷走弱队爆冷/平局，大比分走大球/碾压，强制差异化 |
| 5 | **跳过λ计算或展开** | λ计算是比分预测的核心引擎，跳过等于随机猜 | 近3场(40%)+近6场(30%)+赛季(30%)必须展开 |
| 6 | **同一批次混用不同版本λ管线** | 不同版本λ算法不一致，导致预测结果不可比较 | 先完成当前批次全部输出，再切换版本 |
| 7 | **5串1中出现重复场次** | 重复场次违反串关独立性原则 | 5场必须来自5场不同比赛 |
| 8 | **并发采集多场比赛** | 并行采集导致数据混淆、Tab切换冲突 | 逐场采集，采完1场再采下一场，严禁delegate_task |
| 9 | **输出"仅供参考"等免责套话** | 免责话术降低输出可信度，用户明确禁止 | 直接输出比分/方向/让球，附推导依据 |
| 10 | **友谊赛弱队10+场不败做主力方向证据** | 跨大洲友谊赛状态可迁移性差，对手强度差异大 | 状态权重β从0.30降至0.20，差额转移至λ_market |
