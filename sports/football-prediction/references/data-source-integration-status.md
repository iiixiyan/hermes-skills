# 数据源集成状态三级分类（L1/L2/L3）

> 2026-06-20更新：L2已全自动，L3新增500彩票网+澳客网全自动采集

## L1 — API自动集成（✅ 回测已接入 + 每次预测自动调用）

**定义**：数据采集代码在预测/回测主管线中硬编码调用，每次运行自动采集，无需人工步骤。

**验证标准**：
- `worldcup-predict-all.py` 或 `wc_backtest_fundamental.py` 等主脚本中直接调用了采集函数
- 采集结果自动传入 `form_signal` 注入引擎
- 回测报告中有实测覆盖字段（如"✅ 28场100%"）

**数据源**：
- 新浪API `footballMatchOddsEuro`（欧赔53家百家平均+升/降家数）
- 新浪API `footballMatchOddsAsia`（亚盘17家盘口+水位）
- 新浪API `footballMatchDetail`（FIFA排名+天气+轮次）
- 新浪API `jczqMatches`（SPF/RQSPF/BF赛程列表+赛果）
- 59itou API `apic.jindianle.com`（综合实力分0-100）

## L2 — Playwright浏览器全自动（✅ --browser模式启用）

**定义**：通过Playwright+系统Chromium全自动采集，启动方式为`--browser`参数。脚本为`automated_l2.py`。

**验证标准**：
- 目标页URL格式已知且稳定（`info.titan007.com/analysis/{id}cn.htm`）
- 队名提取：`page.title()`提取 "主队 VS 客队(赛季世界杯)"
- 数据提取：`page.inner_text('body')`后正则解析
- 已验证全量扫描一个ID范围（如2906740-2906809）

**数据源**：titan007分析页
- 天气温度：`场地：...天气：...温度：...`
- 球员评分：`平均评分 x.x`
- 阵容伤停：`阵容情况`段（主队/客队伤病）
- 首发阵容：`号码\t姓名\t位置\t*\t评分`
- 杯赛积分排名：`杯赛积分排名`段
- 近期战绩：`近N场,胜W平D负L,胜率X%`
- 场均进球：`场均进球 x.x`
- H2H历史：`对赛往绩`段

## L3 — Playwright浏览器全自动（✅ --browser模式启用，仅当前/未来比赛有效）

**定义**：通过Playwright自动采集，但数据仅对当前/未来比赛有效（已完赛比赛在这些站返回"暂无数据"）。

**验证标准**：
- 从站点首页赛程表自动匹配队名→提取比赛ID
- 顺序索引法：赛程表中"VS"行的出现顺序与下方链接的顺序一致
- 已验证在当前比赛日数据完整

**数据源**：

### 500彩票网 shuju页 (`automated_l3.py` → `find_500_id()` + `parse_500()`)
- FIFA排名3期+积分变化（唯一提供排名趋势）
- 杯赛积分排名
- 预计阵容（阵型+首发11人+替补）
- 伤病+停赛名单（分开显示）
- 澳门心水推荐（专家文字分析+推介方向）
- 近期战绩10场（含赛事/比分/盘口/半场）
- 未来赛程+相隔天数
- 主客场战绩三分栏

### 澳客网 match详情页 (`automated_l3.py` → `find_okooo_id()` + `parse_okooo()`)
- 球队身价（两队总身价€，如"8.37亿€"）
- 比赛基本信息（轮次/小组/场次）

### 已排除：中国足彩网
原因：`news.zgzcw.com/jczq/zx_XXXXX.shtml`仅有新闻文章，无结构化数据表（赛季排名对比/40场走势/赔率方差等原始目标数据不存在）。`parse_zgzcw()`保留在新版`all_sources_collector.py`中兼容，不纳入自动管线。

## 维护规则

1. **诚实标注**：每个数据源的实际集成状态必须与代码管线一致。解析器函数写好了但不在回测/预测自动管线中的，标注「就绪待集成」而非「已集成」。
2. **L3→L2升级**：需经过≥10场不同比赛验证覆盖率，且处理空数据/异常格式的fallback。
3. **L1降级**：如果API返回空数据或404，降级为L2浏览器采集。
4. **新增数据源**：一律标记为L3，经过验证周期后再升级。
5. **Playwright浏览器初始化**：使用 `sync_playwright().start()` 而非 `__enter__()`，Chromium路径为 `/usr/bin/chromium-browser`。
