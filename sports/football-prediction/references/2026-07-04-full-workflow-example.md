# 2026-07-04 竞足临场预测 — 完整工作流示例

> 此文件记录了一次完整的11场竞足预测工作流，作为未来预测的模板参考。
> 展示了从数据采集→信号分析→规则链追踪→第一性原理评估→双选+单选输出的完整链路。

---

## 数据采集

使用新浪API三接口：

1. `jczqMatches?date=2026-07-04&isAll=1` — 获取当日竞足比赛列表+SPF/RQSPF
2. `footballMatchOddsEuro?matchId=X` — 获取53家欧赔，计算：
   - 百家平均初赔 vs 即赔（o1/o2/o3漂移）
   - 8参数(r1/c1/r2/c2/r3/c3)：升/降家数
3. `footballMatchOddsAsia?matchId=X` — 获取17家亚盘盘口+水位

## 每场分析模板

### 基本盘表格

| 项目 | 值 |
|:----|:----|
| 百家即赔 | o1 / o2 / o3 |
| 8参数 | r1 c1 | r2 c2 | r3 c3 |
| 竞彩SPF | h / d / a |
| RQSPF(让球) | 让胜 / 让平 / 让负 |
| 亚盘 | 升X降X 高X低X |

### 信号分析表

| 信号 | 阈值 | 判定 |
|:----|:----|:----:|
| xh(极端升主) | r1≥30+c1≤5 | ✅/❌ |
| hd(一致降主) | c1≥25+r1≤10 | ✅/❌ |
| r2h(极端升平) | r2≥25 | ✅/❌ |
| xa(极端升客) | r3≥25+c3≤5 | ✅/❌ |
| ad(一致降客) | c3≥25+r3≤10 | ✅/❌ |
| ds(极端降平) | c2≥25 | ✅/❌ |
| bl(骑墙) | c1≥40+r3≥40+o1<o3 | ✅/❌ |

### 规则链追踪

```
rd=X → fd=X → 信号检测 → 规则名 → (h_score, a_score) conf=N
```

### 第一性原理评估

- 检查信号矛盾（如xh+ad同时触发时的方向冲突）
- 检查造热风险（fd≤10淘汰赛中xa是否为真实信号）
- 天气修正（高温/低温/雨天对总进球的影响）
- 淘汰赛保守修正（R32/R16弱队死守意愿）
- 亚盘验证（升盘+低水=真实共识，升盘+高水=支撑不足）

### 最终输出

```
🎯 🎯 双选：X-Y / A-B
🎯 单选：X-Y
⭐ 信心：★★★☆☆ / ★★★☆☆
🏋️ 偏重依据：[信号名]
```
