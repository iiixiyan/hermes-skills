## P0-P10+P11 完整规则（执行顺序严格）

### P0: 东道主
**检测:** 队名包含"美国"或"卡塔尔"
- 美国 → **3-0 / 4-1**
- 卡塔尔 → **1-1 / 0-1**

### P0.5: 绝对碾压+强平信号
**检测:** `fifa≥60 AND f2c≥25(强平降)`
- **1-1 / 2-2**
- 案例: 伊朗2-2新西兰(fifa=65+f2c=32) ✅

### P1: 绝对碾压
**检测:** `fifa≥60` 或 `fifa≥40 AND o1<1.20`
- 分支A: `fifa≥70 AND o1<1.05` → **4-0 / 5-0**（极端碾压）
- 分支B: `fifa≥60 AND 主队FIFA名≤5` → **0-0 / 3-0**（超级强队防冷）
- 分支C: fifa≥60 → **3-0 / 5-0**
- 分支D: fifa≥40+o1<1.20 → **3-0 / 5-0**

### P2: 骑墙（同等极端降主胜+排除客胜）
**检测:** `f1c≥40 AND r3≥40 AND home_real_fav(o1<o3)`
- 含义：降主胜≥40且升客胜≥40 — 市场BOTH buy home AND sell away极端一致 → 转向防平
- **1-1 / 2-0**
- 案例: 比利时1-1埃及(f1c=49+r3=49) ✅, 墨西哥2-0南非(f1c=44+r3=46) ✅

### P3: 极端升主胜+平降（核心防平规则）
**检测:** `r1≥40(升主胜) AND f1c≤5(降主胜≤5) AND f2c≥15(平降)`
- 分支A: `fifa≤10` → **1-1 / 2-2**（均势防大比平）
- 分支B: **三向分歧** `f3c≥40(极强降客) AND 主队FIFA更强(f1<f2)` → **3-0 / 2-1**（市场全面买客→主可能零封）
- 分支C: 其他 → **2-1 / 3-1**

### P4: 真正客强
**检测:** `r1≥40 AND f1c≤5 AND f3c≥20(降客胜) AND away_real_fav(o3<o1)`
- 分支A: `is_ah_fav_away(主受让) AND o3>1.45(浅盘客)` → **2-0 / 1-1**（造热反走）
- 分支B: 其他（深盘客强）→ **0-3 / 1-4**

### P5: 造热（弱队被当热门）
**检测:** `h_excl=True AND home_real_fav(o1<o3) AND NOT home_stronger(f1<f2)`
- 含义：排名倒挂(弱队FIFA差) + 市场一致卖主买客 → 走反方向
- **2-0 / 1-1**
- 案例: 加纳1-0巴拿马(✅偏差1球)

### P6: 极端升主胜无平降
**检测:** `r1≥40 AND f1c≤5 AND f2c<15`
- 含义：市场出货主胜但不防平 → 主队仍有实力
- 分支: 主让球 → **2-1 / 3-1** | 其他 → **1-1 / 2-0**

### P7: 排除客胜（极端升客胜）
**检测:** `r3≥40(升客胜) AND f3c≤5(降客胜≤5)`
- 分支A: **假信号** `o3<1.60(客低赔) AND is_ah_fav_away(主受让)` → **0-1 / 1-2**
- 分支B: **均势** `fifa≤15`:
  - `home_real_fav`(主热门) → **2-1 / 1-1**
  - NOT home_real_fav(主冷门) → **1-0 / 2-1**
- 分支C: **有平降** `f2c≥15` → **2-1 / 1-1**
- 分支D: **主强** `fifa≥20` → **2-0 / 3-0**
- 分支E: 其他 → **2-1 / 1-1**

### P8: 强平降/双边分歧
**检测:** `f2c≥25(强平降) OR (r1≥20 AND r3≥20)(双边分歧)`
- 分支A: `fifa≥20`(差距大) → **1-1 / 0-0**
- 分支B: **主平衡+客极端卖** `abs(r1-f1c)≤5 AND r3≥35` → **2-0 / 3-0**
  - 案例: 英格兰4-2克罗地亚(abs(23-23)=0≤5, r3=40≥35) ✅方向
  - 不影响葡萄牙1-1刚果金(r3=33<35, 走分支A) ✅
- 分支C: 其他(均势) → **1-1 / 2-2**

### P9: 一致降主胜
**检测:** `f1c≥25 AND r1≤10`
- **2-0 / 3-0**

### P10: 一致降客胜
**检测:** `f3c≥25 AND r3≤10`
- 分支A: `fifa≥15` → **0-2 / 0-3**
- 分支B: 其他 → **0-2 / 1-2**

### P11: 弱平信号
**检测:** `f2c≥15`（无其他极端信号时）
- **1-1 / 2-0**

### 默认
- 主队赔率热门(o1<o3) → **2-0 / 2-1**
- 客队赔率热门 → **0-2 / 1-2**

## 信号变量速查

| 变量 | 含义 | 检测条件 |
|:----|:-----|:---------|
| `x_h_up` | 极端升主胜(市场卖主) | r1≥40 AND f1c≤5 |
| `x_a_up` | 极端升客胜(排除客胜) | r3≥40 AND f3c≤5 |
| `h_down` | 一致降主胜(市场买主) | f1c≥25 AND r1≤10 |
| `a_down` | 一致降客胜(市场买客) | f3c≥25 AND r3≤10 |
| `h_excl` | 排除主胜+降客胜 | r1≥40 AND f1c≤5 AND f3c≥20 |
| `d_fall` | 平降信号 | f2c≥15 |
| `d_strong` | 强平降信号 | f2c≥25 |
| `both_lock` | 骑墙 | f1c≥40 AND r3≥40 AND o1<o3 |
| `triple_split` | 三向分歧 | x_h_up AND d_fall AND f3c≥40 AND f1<f2 |
| `is_trap` | 造热 | h_excl AND o1<o3 AND NOT f1<f2 |
| `real_away` | 真正客强 | h_excl AND o3<o1 |
| `home_balanced_away_sold` | 主平衡+客极端卖 | abs(r1-f1c)≤5 AND r3≥35 AND r1≥20 |
