# 共同对手桥接法 — 实战案例

> football-prediction v3.9 §4.1 | 2026-05-24

## 案例1：7004 哈马比 vs AIK索尔纳（6桥·极可靠）

**共同对手**：天狼星/赫根/尤尔加登/埃尔夫斯堡/米亚尔比/北雪平

```
桥1·天狼星: A=1.15 B=3.45(极值)
桥2·赫根:   A=2.55 B=0.38
桥3·尤尔加登: A=0.25 B=1.00
桥4·埃尔夫斯堡: A=3.40 B=1.56
桥5·米亚尔比: A=1.70 B=0.34
桥6·北雪平: 跳过(零封)

中位数: A=1.70, B=0.38(极值3.45被过滤)
λ_trans: A=2.04, B=0.95 → A>>B
三λ融合: γ=0.25 → λ1=2.17 λ2=0.93 差1.24
```

**结论**：6桥一致A>B，桥接法极强确认主队碾压。

## 案例2：7008 马尔默 vs 韦斯特罗斯（5桥·多桥截断）

**共同对手**：索尔纳/哥德堡盖斯/天狼星/赫根/哈马比

```
桥1·索尔纳:     A=1.56 B=0.37 ← 唯一正常桥
桥2·哥德堡盖斯: A=5.87(截断) B=0
桥3·天狼星:     A=5.03(截断) B=2.33
桥4·赫根:       A=3.11(截断) B=2.83
桥5·哈马比:     A=5.87(截断) B=0

4/5桥截断 → γ从0.25降为0.15
仅桥1有效 → A=1.56, B=0.37
```

**教训**：双方都被强队碾压时，桥值爆炸。上限截断后仅1桥有效→降γ+降信心。方向仍正确（A>B），但数值精度降级。

## 案例3：7003 博尔顿 vs 斯托克波特（2桥·单桥极端）

**共同对手**：斯蒂文尼奇、维尔港

```
桥1·斯蒂文尼奇: A=1.85 B=0.31
桥2·维尔港:     A=0    B=∞(中位数过滤)

中位数: A=0.93, B=0.31
```

**教训**：维尔港桥A=0（被零封）被中位数过滤→仅1桥有效值。γ=0.15，方向正确。

## 关键经验

1. **6桥以上**=极可靠，γ=0.25，三λ融合最稳
2. **3-5桥**=可靠，γ=0.20-0.25
3. **1-2桥**=降权，γ=0.08-0.15
4. **多桥截断(>50%)**=降γ一档，方向仍可用但数值精度降级
5. **全桥截断**=桥接法不可用，γ=0
6. **单桥值>3.0**=截断处理，不影响方向判断
7. 主客场**必须**先校准到中立再桥接（主场×0.85进球，客场×1.15进球）
