### 与59itou的分工

| 59itou | 免费源 |
|:---|:---|
| 6Tab详情(阵容/战绩/欧指/亚指/排名/盈亏) | 身价 → Transfermarkt |
| 竞足/北单列表+matchID | 赔率历史 → 500.com/雷速 |
| 必发交易量 | xG数据 → Understat |
| 赛后复盘+SP | 对手实力 → Soccerway积分榜 |

> 优先用59itou获取原始innerText（一字不改），免费源作为交叉验证和补充维度。

> 基于微信文章「AI预测体育比赛」中「多模型融合提升稳定性」+「特征工程比算法选择更重要」的核心理念。

### 融合架构

```
59itou 6Tab原始数据
    ↓ feature_extractor.py (40+维结构化特征)
    ↓
┌──────────────────────────────────┐
│ Poisson λ加权 (主模型, 60%)      │ ← 现有模型
│ + XGBoost回归 (辅助模型, 40%)   │ ← 需≥20条复盘数据后激活
│ + 因子修正层 (13因子即时校准)    │ ← 强制执行
└──────────────────────────────────┘
    ↓
融合预测 (预期进球 + 胜平负概率)
```

### 使用方式

在分析每场比赛时，采集完6Tab数据后调用：

```python
from feature_extractor import extract_features, features_to_dict
from model_fusion import predict_with_fusion, get_fusion

# 1. 提取结构化特征

pred = predict_with_fusion(lambda_1, lambda_2, home_odds, draw_odds, away_odds)

fusion = get_fusion()
print(fusion.status())  # {'mode': 'poisson_only', 'training_samples': 1, ...}
```
