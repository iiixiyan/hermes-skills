## 三、关键发现与规则更新

### 发现1：防平信号体系验证（5/5 = 100%）

首轮中最可靠的信号组合是：
```
极端升主胜(≥40升/≤5降) + 平局一致降(≥20降) + 首轮慢热(×0.70~0.75)
= 防平信号，100%可靠
```

案例：
- 巴西vs摩洛哥：46升胜/3降 + 39降平 + 慢热×0.70 → 1-1 ✅
- 荷兰vs日本：42升胜/2降 + 35降平 + 慢热×0.70 → 2-2 ✅

### 发现2：东道主规则验证（1/1 = 100%）

卡塔尔1-1瑞士验证了东道主防冷规则。即使市场极端一致看好瑞士(负47降/0升)，东道主回归基本面依然有效。

### 发现3：因子19需增加首轮空转豁免（1/2 = 50%）

西班牙0-0佛得角暴露了因子19的一个盲区：
- 因子19说：主胜<1.20≥2家 → 信心上限60%，爆冷覆盖客胜
- 但西班牙(FIFA2)vs佛得角(FIFA67)并无"客胜"风险
- 实际0-0是"主队λ空转"——顶级强队控球但无法进球
- **修正**：当主队FIFA≤5且客队FIFA>60时，因子19的爆冷方向必须是"冷平/小球"(0-0/1-0)，而非"客胜"

### 发现4：4b排除客胜在首轮不可靠

海地vs苏格兰案例：负赔49升/0降极端排除客胜，但苏格兰1-0取胜。
- 与友谊赛的4b排除方向降级模式一致
- 已更新为世界杯首轮4b排除客胜降级P2参考

### 发现5：因子12造热信号

澳大利亚vs土耳其案例：负赔38降/6升一致看好土耳其(FIFA22>澳27)，但澳大利亚2-0胜。
- 当FIFA差<15且因子12触及时，可能为造热信号
- 已更新为世界杯首轮因子12造热警觉规则
