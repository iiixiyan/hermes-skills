### 3. 凯利公式配比（纯计算，无SQLite）

当需要推荐投注额时：

```python
def kelly_bet(p_model, odds, bankroll=10000, fraction=0.25):
    """
    p_model = 我们的预测概率
    odds = 市场赔率
    bankroll = 本金
    fraction = 凯利系数（保守0.25，激进0.5）
    """
    b = odds - 1
    p = p_model
    q = 1 - p
    f_star = (b * p - q) / b  # 完整凯利
    f = max(0, f_star * fraction)
    return round(bankroll * f, 0)

# 使用示例：
# bet = kelly_bet(p_model=0.45, odds=2.20, bankroll=10000, fraction=0.25)
# → 建议投注额
```

### 4. 命中判定规则

| 维度 | 条件 | 判定 |
|:----|:-----|:----:|
| 精确比分 | 预测比分 == 实际比分 | ✅ 精确命中 |
| 接近比分 | 差值=1球（如预测2-0→实际2-1） | ⚠️ 差1球 |
| 方向命中 | 胜平负方向正确 | ✅ |
| 让球命中 | 让球选项正确 | ✅ |
| 异常命中 | 异常比分类别覆盖了实际结果 | ✅ 冷门命中 |
