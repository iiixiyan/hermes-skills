# 数据源适配指南：59itou → football-prediction 参数映射

> 当配合 `59itou-data-fetch` 技能使用时，该指南说明如何将 59itou 各Tab的原始数据映射到分析框架的输入参数。

## 零、数据采集工作流（必读）

使用 `59itou-data-fetch` 技能获取数据时，按以下流程操作：

### 0.1 加载技能
```
skill_view(name='59itou-data-fetch')
```

### 0.2 获取matchID
1. 打开竞足列表页 → 获取目标比赛的基本信息（联赛、排名、赔率）
2. 打开北单列表页 → 用JS提取所有`.item[data-id]`
3. 用`联赛+排名`交叉匹配到目标比赛的matchID

### 0.3 逐场采集5个Tab
对每场比赛：
1. `browser_navigate` 进入详情页
2. 获取**战绩Tab**（默认）的完整innerText
3. JS原生`.click()`切换到**欧指Tab**，等待1-2秒后获取innerText
4. 切换到**阵容Tab**，获取innerText
5. 切换到**排名Tab**，获取innerText
6. 切换到**亚指Tab**，获取innerText

### 0.4 提取核心参数
从innerText中提取关键数据后，按下方各章节的映射填入框架。

## 一、框架需求 vs 59itou 可用数据

| 框架需求 | 59itou 数据源 | 映射方法 |
|---------|--------------|---------|
| 近6轮战绩 | 战绩Tab → 近10场战绩列表 | 从最近10场中取最近6场，主客分开统计 |
| 近3场权重 | 战绩Tab → 近10场战绩列表 | 取最近3场单独标注(权重40%) |
| 场均进球(替代xG) | 战绩Tab → "进X失X" | 用实际进球数代替xG(近6场均值和赛季均值) |
| 场均失球(替代xGA) | 战绩Tab → "进X失X" | 用实际失球数代替xGA(近6场均值和赛季均值) |
| 联赛排名 | 排名Tab → 赛季排名对比 | 直接读取 |
| 主客场差异 | 战绩Tab → "近10场主客战绩" | 提取主场/客场胜平负及进失球 |
| H2H历史交锋 | 战绩Tab → "两队交锋" | 近5次交锋比分及胜负趋势 |
| 综合实力 | 战绩Tab → "综合实力" (主XX-客XX) | 直接使用，但不用于λ计算 |
| 伤停信息 | 阵容Tab → 伤停数据 | 致命级/重要级/边缘级分类 |
| 身价对比 | 阵容Tab → "身价€" | 辅助判断实力差距 |
| 欧指/百家平均 | 欧指Tab → 百家平均初赔/即赔 | 用于对比模型概率和价值分析 |
| 亚指盘口 | 亚指Tab → 盘口/水位/赢盘率 | 辅助判断市场倾向 |
| 双赛体能 | 战绩Tab → "近期赛程" | 查看周中是否有比赛及间隔天数 |

## 二、缺少数据时的替代处理

### 2.1 xG/xGA 缺失
59itou 不提供 xG 数据。使用实际进球替代时：
- 近6场实际场均进球 → 替代 xG
- 近6场实际场均失球 → 替代 xGA
- 主客场系数参考：荷甲+15%，英超+15%~20%，德乙+10%~15%，日职+10%

### 2.2 战术相克数据缺失
59itou 不提供战术统计。通过以下方式推断：
- H2H 长期趋势：一方连续多年克制另一方 → 标记"血脉克制"
- 强弱队战绩：战绩Tab有"对阵强队/对阵弱队"数据可参考
- 阵型信息：阵容Tab有首发阵型(如4-3-3/3-4-2-1)可用于战术判断

### 2.3 天气数据缺失
通过联网搜索补全，或赛前1小时从 59itou 页面顶部公告栏获取。

## 三、λ 值计算参数提取

### 3.1 从战绩Tab提取
```
近10场战绩：
  主队：X胜 Y平 Z负 进A 失B
  客队：X胜 Y平 Z负 进C 失D
```

- 近6场进球 = 从最近6场比分中提取
- 赛季场均进球 = 联赛总进球/联赛已赛场次
- 主场场均进球 = 从"近10场主客战绩"中提取

### 3.2 从欧指Tab提取
```
百家平均：初赔 [主胜赔/平赔/客胜赔] 即赔 [主胜赔/平赔/客胜赔]
概率转换：主胜X% 平局Y% 客胜Z%
```

- 赔率变化：初赔→即赔的升降趋势
- 指数变化：上升家数 vs 降低家数

### 3.3 从亚指Tab提取
```
盘口：[主让X球/受让X球]  水位：[主水XX/客水XX]
赢盘率：主队XX% 客队XX%
```

- 盘口深度：判断市场对实力差的预期
- 赢盘率：判断球队实际表现与市场预期的偏差

## 四、输出参数检查清单

在输出预测前，逐项确认以下数据已获取：

- [ ] 主队近6轮战绩（含主场战绩）
- [ ] 客队近6轮战绩（含客场战绩）
- [ ] 主队近3场战绩（权重40%）
- [ ] 客队近3场战绩（权重40%）
- [ ] 主队赛季场均进球/失球
- [ ] 客队赛季场均进球/失球
- [ ] 主队主场场均进球/失球
- [ ] 客队客场场均进球/失球
- [ ] 近5次H2H比分及趋势
- [ ] 双方伤停情况（分类影响等级）
- [ ] 战意指数量化（1-5级）
- [ ] 双赛体能评估（周中比赛+间隔天数）
- [ ] 天气/场地条件（如有）
