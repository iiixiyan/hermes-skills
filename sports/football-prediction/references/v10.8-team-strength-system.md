# 队伍实力对比系统 v1.0 (v10.8)

> 集成于 `worldcup-predict-v10.py` 的 `compute_team_strength()` 和 `predict_with_basics()`
> 每场比赛自动运行(包括无form_signal时)

## 数据来源

| 数据 | 来源 | 权重 | 说明 |
|:----|:-----|:----|:-----|
| FIFA排名 | 新浪API footballMatchDetail | 30% | 排名越低越强, 线性映射到0-100 |
| 59itou综合实力分 | 59itou API `list[0/1].rank` | 25% | 0-100分, 需要传form_signal |
| 球员总身价 | 内置SQUAD_VALUES对照表 | 20% | 对数缩放: 10亿€→100分, 1亿€→50分 |
| 近期状态 | form_diff或近10场胜率 | 15% | 每胜=10分, 10胜=100分 |
| 球员评分 | titan007分析页平均评分 | 10% | 7.0=50分基准 |
| 伤停惩罚 | injury_impact_h/a | (-5分/人) | 核心伤停每人-5分 |

## 判定阈值

| 实力差 | 等级 | 进球修正 |
|:-----:|:----|:-------:|
| ≥25 | 碾压 | +1(不是+2!) |
| ≥8 | 明显优势 | +1 |
| ≥3 | 小幅优势 | 0 |
| ≥-3 | 接近 | 0 |
| ≥-8 | 小幅劣势 | 0 |
| ≥-25 | 明显劣势 | -1 |
| <-25 | 被碾压 | -2 |

> 历史验证: 碾压+2过激(巴西5-0海地估高为4-0), 已降为+1

## 冷门保护

当纯赔率规则链命中以下规则名时, 队伍实力对比**仅增强信心**而不改比分:
- P11- (市场过热/防平)
- R13 (双边分歧)
- R10A-/R10D- (客场/冷门标记)
- R10-Trap (陷阱)
- R16- (弱分歧)

## 基本盘参数对比分的实际影响

predict()中的温度/天气/中立使用λ系数修正(×0.90~1.05), 但5-10%的修正在取整后对比分零影响:
- λ=2.0×0.95=1.9 → 取整仍是2
- λ=2.0×0.90=1.8 → 取整还是2

**结论**: 基本盘修正必须是比分级直接修正(±1球), 而非λ百分比缩放。

## 身价数据源

预置SQUAD_VALUES对照表(50+支球队), 来源:
- Transfermarkt (权威标准)
- 百度百科 (已验证: 荷兰队总身价8.4亿€, 非估测的6.5亿€)

百度百科页面载入后搜索"总身价"关键词可提取数值。

## 回测流程

不再使用固定测试集。正确流程:
1. 赛前预测: 对未开赛比赛用当前引擎输出
2. 等赛果: 比赛完赛后获取实际比分  
3. 对比: 预测vs实际
4. 修复: 有偏差才分析根因修复
5. 复用: 修复后用于下一轮新比赛
