# 天天盈球 (ttyingqiu.com) 基本面数据源

> 发现时间: 2026-06-19 | 已验证: 40场世界杯比赛数据采集成功

## 用途

提供足球比赛的基本面数据（阵容、伤停、近10场战绩、身价、有利/不利因素分析），用于增强预测引擎。

## 数据能力

| 数据类型 | 可用性 | 说明 |
|:--------|:------|:-----|
| 首发阵容+阵型 | ✅ | 含球员评分、站位图 |
| 伤停信息 | ✅ | 具体球员+伤停解读 |
| 球队身价 | ✅ | 含双方对比 |
| 有利/不利因素 | ✅ | 每队5-8条详细分析文本 |
| 精选情报 | ✅ | 针对性战术/赛前分析 |
| 近10场战绩 | ✅ | 散文格式（非结构化） |
| 天气/场地 | ✅ | 部分比赛有 |
| H2H历史 | ✅ | 最近交锋记录 |

## URL格式

```
比赛列表页: https://www.ttyingqiu.com/jczq
           (左侧箭头翻到目标日期)
比赛详情页: https://www.ttyingqiu.com/live/zq/matchDetail/info/{matchId}
```

## matchId获取方法

1. 打开 `https://www.ttyingqiu.com/jczq`
2. 用日期导航(左侧箭头)翻到目标日期
3. 找到世界杯比赛行，点击"情"(情报)链接
4. URL中的数字即为 matchId

## 局限

- 散文中提取结构化数据需要NLP（无固定格式）
- 提前3天以上的比赛可能没有球队线索/伤停数据
- 比赛列表通过JS加载，curl不可用，必须用浏览器
