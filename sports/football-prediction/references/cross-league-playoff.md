# 跨级升降级附加赛特殊处理

> 2026-05-29 新增。基于5/28 葡超保级附加赛（卡萨皮亚2-0托伦斯）预测偏差复盘。

## 场景识别

**当遇到以下情况时，比赛属于跨级升降级附加赛（非杯赛、非普通联赛）：**

1. 主队和客队来自**不同级别的联赛**（如葡超vs葡甲、德甲vs德乙等）
2. 比赛性质为**升降级附加赛**（升级/保级决定战），通常有两回合
3. 赛事标识：葡超/西甲/意甲/德甲/法甲等联赛系统内的「保级附加赛」或「升级附加赛」
4. 赛事在prize page上标注为高级别联赛名（如「葡超」），但详情页中客队标注有不同联赛级别（如「葡甲3」）

## 与现有因子的关系

### 跨级杯赛系数（不适用）
现有的跨级杯赛系数 `# 跨级杯赛` 明确标注：
> **不适用场景**：联赛比赛（两队本就在同一级别）

升降级附加赛**不是杯赛**，但两队**不是同一级别**。因此跨级杯赛系数不能直接套用。

### 因子14（弱队客场λ系数 - 部分适用）
因子14条件：`客队排名比主队低 >= 5名`
- 不同联赛排名无法直接比较（葡超16 vs 葡甲3 不能简单说谁高谁低）
- 葡超整体水平高于葡甲，葡超16≈葡甲1-2实力
- **结论**：不适用，排名跨联赛不可比

### 因子2（赛季末战意差）
赛季末最后5轮 + 战意差：
- ✅ 适用：保级附加赛战意拉满
- 异常权重+2级强制触发

### 因子11（弱队主场进球下限）
`主弱客强 and λ_主 < 1.20 → λ_主 = max(λ_主, 1.0)`
- 跨级附加赛中，主队（通常是高级别联赛保级队）看似排名低但实际实力≈低级别顶级
- λ_主不应被过度压低为弱队
- 适用条件：与因子11方向相反，需特殊处理

## 核心问题：λ计算偏差

**跨级附加赛中λ计算的三大偏差来源：**

### 偏差1：主队赛季数据失真
- 主队整个赛季在高级别联赛中表现为保级队水平，进球少、失球多
- 但面对低级别对手时，实际攻防能力远超联赛数据反映的水平
- 案例：卡萨皮亚葡超赛季进5失13（近10场），但面对葡甲第3的托伦斯，实际攻防表现远优于此数据

### 偏差2：客队跨级竞争力高估
- 客队在低级别联赛中数据亮眼（近10场5胜4平1负，进14失7）
- 但面对高级别球队时，防守体系面临的质量压力倍增
- 案例：托伦斯葡甲出色，但面对葡超防线时全场仅0球

### 偏差3：赛制特殊性被模型忽略
- 附加赛两回合制，首回合0-0意味着主队必须赢球才保级
- 主队「背水一战」效应：主场观众+生死压力=进攻超常发挥
- 客队「保平争胜」心态：客场进球规则下平局即升级（首回合0-0），守平为主

## 修正方案（P0强制·2026-05-29新增）

### 步骤1：识别跨级附加赛
```python
if (
    (主队联赛 != 客队联赛)       # 不同联赛级别
    and 比赛是升降级附加赛        # 确认性质
    and not 杯赛决赛              # 排除杯赛
    and not 中立场地              # 排除中立场地
):
    跨级附加赛 = True
```

### 步骤2：主队λ上修（背水一战系数）
```python
if 跨级附加赛 and 主队是高级别联赛保级队:
    # 主队面对低级别对手时，进攻能力向上修正
    λ_主 *= 1.30   # 背水一战+跨级实力差
    # 案例：卡萨皮亚 λ_原=0.54 → 修正0.70
    # 实际主队进了2球，修正后仍偏低但方向正确
```

### 步骤3：客队λ下修（跨级防守压力）
```python
if 跨级附加赛 and 客队是低级别联赛升级队:
    # 低级别球队面对高级别对手，实际进攻能力打折
    λ_客 *= 0.75   # 跨级防守压力
    # 案例：托伦斯 λ_原=1.15 → 修正0.86
```

### 步骤4：赛制修正（两回合制）
```python
if 跨级附加赛 and 有两回合:
    if 首回合 == \"0-0\" 或 主队落后:\n        # 主队必须赢球才能保级/晋级
        主队进球信心 + 1级
        客队平局即晋级 → 守平为主，进攻投入减少
        λ_客 *= 0.90  # 客队进攻打折\n    elif 首回合客队取胜:
        主队背水一战系数 + 0.5级（更激进）
```

### 步骤5：因子4b信号重新评估
```python
if 跨级附加赛 and 因子4b触发:
    # 附加赛中4b信号可靠性低于联赛
    # 原因：附加赛市场流动性低，信号可能失真
    4b降级为P1参考（不从P0强制执行）
    异常比分可选包含反方向（不强制）
    # 案例：卡萨皮亚vs托伦斯，主胜26升/1降触发4b
    # 按P0强制应在异常比分中给出客胜方向(0-1/1-2)
    # 但实际主队2-0完胜，4b信号在附加赛场景下失效
```

## 验证案例

| 场次 | 原预测 | 修正后(跨级附加赛系数) | 实际 | 备注 |
|:----|:------|:--------------------:|:----:|:----:|
| 卡萨皮亚2-0托伦斯 | 1-1/0-0 | **2-0/1-0** ✅ | 2-0 | λ_主×1.30, λ_客×0.75×0.90 |

**2026-05-29追加确认（5/28比赛实际验证）**：
- 卡萨皮亚(葡超16)2-0托伦斯(葡甲3)：跨级附加赛第二回合(首回合0-0)
- 信号：因子4b(升主胜26/1降)触发→按跨级附加赛规则降级为P1✅
- 实际：主胜2-0打出，让球(-1)让胜
- **验证结果**：4b降级正确(主胜仍在P1级别允许范围内)，让球(-1)让胜也符合λ修正方向
- **所有5项修正全部有效**：λ主×1.30, λ客×0.75, 赛制修正λ客×0.90, 4b降级

## 不适用场景

- 🚫 普通联赛比赛（两队同级别）
- 🚫 杯赛（已有跨级杯赛系数处理）
- 🚫 同级别升降级附加赛（如德甲16vs德乙3 — 两队本质是同一级别的跨级，但相对差距小）
- 🚫 欧冠/欧联/欧协联等洲际俱乐部赛（已有独立规律）
