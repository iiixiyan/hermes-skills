# F40 + F42 交互规则 — Poisson验证报告

> 发现: 2026-07-08 竞足复盘 | 验证: 阿根廷#2 vs 埃及#24 (世界杯R16)

## 问题

F42的原始描述为"标准泊松Top6中第二位比分从3-0替换为3-1"。但此描述**仅在λ₂很低(0.20)时成立**。当F40同时触发(λ₂≥0.45下限保护), Poisson分布会发生重大变化。

## Poisson验证数据

**前提:** λ₁=1.57 (阿根廷), 两挡λ₂对比:

### 场景A: 无F40 (λ₂=0.20) — 3-0在位
```
1-0(26.7%)  2-0(21.0%)  0-0(17.0%)  3-0(11.0%)  1-1(5.3%)  4-0(4.3%)
                                         ↑ 第4位, 3-1不在Top6
```

### 场景B: 有F40 (λ₂=0.45) — 3-0大幅下降
```
1-0(20.8%)  2-0(16.3%)  0-0(13.3%)  1-1(9.4%)  3-0(8.6%)  2-1(7.4%)
                                                     ↑ 第5位, 3-1仅3.9%不在Top6
```

## 核心发现

| 属性 | F40关闭 | F40触发 |
|:----|:-------|:--------|
| λ₂ | 0.20 | 0.45 |
| 3-0排名 | 第4位 | 第5位 |
| 3-1概率 | ~2% | 3.9% |
| 3-1在Top6? | ❌ | ❌ |
| 弱队进球概率P(λ₂≥1) | 16.4% | **36.2%** |

**弱队进球概率从16.4%翻倍到36.2%** — 这就是F40的核心作用: 弱队从"几乎不可能进球"变为"有约1/3概率进球"。

## F42的正确理解

F42不是"替换固定位置", 而是：

1. **识别场景**: R16精英(fh≤5) vs 有能力弱队(fa∈[20,30]), fd∈[15,30], hd触发, 无r2h
2. **判断原因**: 精英强队领先2球后不全线死守(无r2h=市场不担心冷平), 弱队落后all-in反扑
3. **注入机制**: 将3-1强行注入到Top6中, **替换最小的合理零封比分**(通常是3-0, 如果3-0已在Top6; 否则可能是2-0)

### 为什么3-1注入比3-0好

| 比分 | 实际3-2的偏离 | 说明 |
|:----|:------------:|:-----|
| 3-0 | 偏差2球❌ | 埃及2球 vs 0球预测, 完全漏掉弱队进球 |
| 3-1 | 偏差1球✅ | 埃及2球 vs 1球预测, 方向正确仅差1球 |
| 2-0 | 偏差3球❌ | 阿根廷3球 vs 2球, 埃及2球 vs 0球 |

## 实现建议

当F40和F42同时触发时:

```python
# F40: 将λ₂提升到0.45
lambda_away = max(lambda_away, 0.45)

# F42: 检查3-1是否需要注入Top6
if f42_triggered:
    top6 = get_poisson_top6(lam_home, lam_away)
    # 检查3-1是否已在Top6
    if (3, 1) not in [s for s, _ in top6]:
        # 找到最小的零封比分(3-0或2-0)
        clean_sheets = [(g1, 0) for (g1, 0), p in top6]
        if clean_sheets:
            smallest = min(clean_sheets)  # 按(g1,0)比较
            # 用3-1替换最小零封比分
            top6 = [((3,1), None) if s == smallest else (s, p) for s, p in top6]
```
