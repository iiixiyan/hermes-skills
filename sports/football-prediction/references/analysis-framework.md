# 竞彩比分预测详细分析框架

> 本文件由 SKILL.md 在执行第 2-5 步时按需加载。

## 一、数据采集详细指标

### 1.1 联赛排名与积分形势
- 当前联赛排名、积分、胜平负场次
- 主客场分开统计：主场积分/客场积分、主场胜率/客场胜率
- 近期积分走势（近 6 轮积分曲线）

### 1.2 近 6 轮战绩
- 主队近 6 轮：总战绩 + 其中主场战绩
- 客队近 6 轮：总战绩 + 其中客场战绩
- 近 3 场额外标注（权重 40%）

### 1.3 进攻数据
- 场均射门数、射正率
- xG（期望进球）——近 6 场均值 + 赛季均值
- 反击进球占比、定位球进球占比
- 关键传球手助攻数据

### 1.4 防守数据
- 场均被射门数
- xGA（期望失球）——近 6 场均值 + 赛季均值
- 零封率
- 高压失误次数（导致射门的失误）

### 1.5 历史交锋
- 近 5 次对阵比分
- 让球盘口走势
- 战术相克特征（如某队面对高位逼抢时胜率骤降）

### 1.6 天气与场地
- 暴雨/高温/大风：影响技术流传导，利好防守方
- 人工草皮：影响球速和反弹，适应方占优
- 场地尺寸：小球场利好密集防守

## 二、伤停与战术影响

### 2.1 核心球员伤停评估
按影响等级分类：
- **致命级**：头号射手、组织核心、防线领袖、主力门将
- **重要级**：常规主力、战术支点
- **边缘级**：轮换球员、替补

评估替代者能力差距和战术调整可能性。

### 2.2 战术相克性分析
常见相克关系：
- 高位逼抢 → 克制 → 后场短传出球型
- 边路强攻 → 克制 → 三中卫翼卫回防慢
- 防守反击 → 克制 → 控球高压但防线身后空档大
- 密集防守 → 克制 → 缺乏远射/定位球能力的传控队

### 2.3 阵型变化预期
- 伤停导致的被迫变阵
- 针对性战术变阵的历史效果

## 三、战意与赛程体能

### 3.1 战意指数
按 1-5 级量化：
- 5 级：保级生死战、争冠天王山、德比战
- 4 级：欧战资格关键战、杯赛淘汰赛
- 3 级：争夺更好排名、杯赛小组赛
- 2 级：无欲无求但面对强队有表现欲
- 1 级：彻底无欲无求、已提前夺冠或降级

**触发异常权重的条件**：双方战意差 ≥ 2 级。

### 3.2 双赛体能
- 周中是否踢了欧战/杯赛
- 轮换幅度评估（首发更换几人）
- 客场旅行距离与休息天数
- 连续双赛的累积疲劳（第 3 场双赛体能下降更明显）

### 3.3 赛前动态
- 更衣室氛围（内讧传闻、将帅失和）
- 主帅赛前言论（是否有轮换暗示）
- 转会窗动态（核心球员被报价影响心态）

## 四、模型计算

### 4.1 基准预期进球
- 综合近 6 场 xG、赛季 xG、主客场系数，计算：
  - λ1（主队预期进球）
  - λ2（客队预期进球）
- 主客场系数参考：英超主场进球加成约 +15%~20%

### 4.2 状态权重
加权公式：

λ = (近3场均值 × 0.4) + (近6场均值 × 0.3) + (赛季均值 × 0.3)

### 4.2a 弱队客场λ修正（因子14·2026-05-26新增·P0强制）

**计算完基础λ后，按以下顺序应用修正：**

**步骤1：跨级杯赛修正（优先级最高）**
```python
if 跨级杯赛:
    if 低级别队是主队:
        λ_低 *= 0.75
        λ_高 *= 0.90
    else:
        λ_低 *= 0.65
        λ_高 *= 0.95
```

**步骤2：弱队客场λ系数（因子14）**
```python
if 客队排名比主队低 >= 5名:
    if 跨级杯赛:     # 跨级时已应用步骤1，此处不再重复
        λ_客 = λ_客  # 已修正
    elif 杯赛:
        λ_客 *= 0.70
    elif 联赛:
        λ_客 *= 0.80
```

**步骤3：解放者杯客场远征系数**
```python
if 联赛 == "解放者杯" and 跨国:
    if 飞行距离 > 2000km: λ_客 *= 0.65
    elif 飞行距离 > 1000km: λ_客 *= 0.75
    else: λ_客 *= 0.80
```

**步骤4：因子11弱队主场保底（应用步骤1-3后再检查）**
```python
if 主弱客强 and λ_主 < 1.20:
    λ_主 = max(λ_主, 1.0)
```

### 4.2b 跨级升降级附加赛λ修正（2026-05-29新增·P0强制）

**当识别为跨级升降级附加赛后（详见 `references/cross-league-playoff.md`），在步骤1-4之后应用以下修正：**

**步骤5：跨级附加赛识别**（在应用完步骤1-4后检查）
```python
if 主队联赛 != 客队联赛 and 比赛是升降级附加赛:
    # 背水一战系数
    if 主队是高级别保级队:
        λ_主 *= 1.30  # 背水一战+跨级实力差
    # 跨级防守压力
    if 客队是低级别升级队:
        λ_客 *= 0.75  # 低级别客场对高级别防守打折
    # 赛制修正（两回合制）
    if 首回合 == "0-0" and 主队是首回合客场:
        客队守平即晋级 → λ_客 *= 0.90
    # 因子4b降级
    if 因子4b触发:
        4b降级为P1参考（附加赛市场流动性低，信号可能失真）
```

### 4.3 攻防博弈结构判定
- **正路结构**：主队 xG > 客队 xGA 且 客队 xG < 主队 xGA
  → 实力差距清晰，大概率按预期走
- **失衡结构**：上述条件不满足
  → 存在不确定性，冷门概率提升

### 4.4 爆冷系数
满足以下任一条件触发 +1 级异常权重：
- 战意差 ≥ 2 级
- 核心射手/防线领袖伤停
- 保级队 vs 无欲无求强队
- 极端天气（暴雨/大风）
- 强队周中欧冠消耗 + 周末联赛间隔 < 72 小时

### 4.5 因子4b执行前强制检查（2026-05-28新增）

**在4b信号触发后、生成异常比分前，必须按以下顺序执行强制检查：**

**步骤1：失效条件检查**
```python
# 任一命中 → 4b降级为P1参考（不从P0执行）
if 比赛是杯赛决赛:
    4b_级别 = "P1"  # 决赛双方战意拉满+无主场优势，4b信号不可靠
    # 案例：3001 水晶宫vs巴列卡诺(欧协联决赛) 29升/0降但主胜打出
elif 比赛在中立场地:
    4b_级别 = "P1"  # 中立场地消除主场优势，4b是纯市场信号
    # 案例：同上（欧协联决赛为中立场地）
elif 主队近3场更换主教练:
    4b_级别 = "P1"  # 换帅效应（反弹/混乱）改变球队基本面
```

**步骤2：「价值回归」识别**（仅在4b触发主胜升赔方向时检查）
```python
# 场景：主胜升赔但实际是初赔过低的市场修正
if 主胜升 >= 20 and 主胜降 <= 2:  # 触发4b
    if 主队近10场胜率 >= 70% \
        and 主胜初赔 < 2.0 \
        and 平局降 >= 20 \
        and 客胜降 >= 20:
        判定 = "价值回归"  # 三方向都被压低=初赔过低的市场修正
        4b_级别 = "P1"    # 降级为参考，不从P0执行
        # 案例：3002 德尔瓦耶(近10场8胜,初赔1.83→2.02,29升/0降,平28降/客26降)
```

**步骤3：异常比分生成规则**
```python
if 4b_级别 == "P0":  # 通过检查
    异常比分必须包含反方向
elif 4b_级别 == "P1":  # 失效条件命中或价值回归
    异常比分可选包含反方向（不强制）
    正路信心只-1★（非P0的-2★）
```

## 五、异常因子挖掘

### 5.1 爆冷诱因
- 强队大幅轮换（联赛争冠无望 / 杯赛留力）
- 更衣室内讧或将帅失和
- 保级队背水一战 + 主场加成
- 历史交锋存在明显血脉克制
- 极端天气削弱技术流球队优势
- 关键球员赛前突发伤缺

### 5.2 极端赛果诱因
- 弱队防线核心缺阵 → 体系崩塌 → 大比分失利
- 强队需大量净胜球 → 放手狂攻 → 单方大球
- 双方均为互攻风格且无欲无求 → 对轰局
- 杯赛加时赛后的体能崩盘 → 尾声阶段连续失球
- 红牌/点球等意外事件改变比赛节奏

### 6.1 国际友谊赛特殊处理（2026-05-29追加）

### 6.1 国际友谊赛特殊处理（2026-05-29追加）

**特殊性**：
1. λ计算依赖的联赛数据不存在——球队没有"赛季均值"，只能用近几场国家队比赛历史数据
2. 排名差异 ≠ 实力差异——FIFA排名不反映即时状态
3. 让球穿盘率低——友谊赛中领先方常换人/收兵，1-0赢球输盘是典型模式
4. 综合实力分仍有效——阵容身价/平均年龄等数据可供参考

**处理建议**：
- 友谊赛信心不超过★★★☆☆（因联赛数据不可得）
- 因子12(一致降赔)在友谊赛中仍有效——一致降主胜30/0的强信号在主胜方向正确
- 让球预测偏向保守：主胜下让球倾向让平而非让胜（穿盘概率<50%）
- 异常比分中客胜方向的可信度低于联赛（友谊赛客队远征不会全力以赴）
- 因子2(战意)、因子9(争冠无望)等联赛专属因子不适用

**验证案例**：周四001 爱尔兰1-0卡塔尔(国际友谊赛)
- 因子12(一致降主胜30/0) → 主胜方向正确✅
- 让球(-1) → 1-0让平✅(穿盘率低预测正确)
- 无异常比分触发(客胜升30家=市场极度不看好客胜→异常客胜方向不可信)

### 6.2 国际友谊赛0-0/小球偏倚（2026-05-30新增）

**发现来源**：2026-05-29 5008 波黑0-0北马其顿

**核心规律**：国际友谊赛中，当双方实力接近（综合实力差≤10分）且让球方为-1浅盘时，0-0概率显著上升至约30-40%，远超联赛0-0概率(~10%)。

**原因**：
- 友谊赛双方都不会全力以赴
- 下半场大规模换人后进攻节奏被打断
- 双方主要目标为磨合阵容而非取胜

**修正式**：
```python
if 联赛 == "国际友谊" or 联赛 == "International Friendly":
    if 双方综合实力差 ≤ 10:  # 实力接近
        0-0概率 = 泊松P(0,0) × 2.5  # 0-0概率上调150%
        正常比分第1个应优先考虑0-0或1-1
    
    if 让球数 == -1 and 主胜初赔 > 1.70:
        让球优先: 让负 > 让平 > 让胜
        # 浅让友谊赛穿盘极难
```

**验证案例**：
| 场次 | λ1 | λ2 | 初赔 | 实际 | 预测改善 |
|:----|:--:|:--:|:----:|:---:|:--------|
| 5008 波黑0-0北马其顿 | 1.87 | 3.38 | 1.87 | 0-0 | 原预测1-0/2-0→应修正为0-0/1-1 |

## 八、让球胜平负推导（每次预测必须输出）

### 8.1 让球数提取
从详情页欧指Tab数据中提取让球数。竞足列表页通常以「主队(-1)」格式显示让球数。

**提取方式**（从欧指Tab innerText）：
```python
# 查找 "-X)" 或 "(+X)" 模式
import re
match = re.search(r'\(([+-]?\d+)\)', ouzhi_text)
if match:
    让球数 = int(match.group(1))
else:
    让球数 = 0  # 默认平手盘
```

### 8.2 让球结果计算

**让球数含义（竞足官方格式）**：
- `(-1)` = 主队让1球（主队需净胜2+才算让胜）
- `(-2)` = 主队让2球（主队需净胜3+才算让胜）
- `(0)` = 平手盘（胜=主胜，负=客胜）
- `(+1)` = 客队让1球（主队受让，不败即让胜）

**计算公式**：
```python
def calc_rangqiu(主队进球, 客队进球, 让球数):
    """
    让球数 = 竞足官方让球数
    示例：-1=主队让1球, +1=客队让1球(主队受让)
    """
    净胜 = 主队进球 - 客队进球
    
    # 调整后净胜 = 主队净胜 - 让球数绝对值
    abs_hcp = abs(让球数)
    
    if 让球数 <= 0:  
        # 主队让球(<=0) 或 平手(=0)
        # 让胜：主队净胜 > 让球数绝对值
        # 让平：主队净胜 = 让球数绝对值
        # 让负：主队净胜 < 让球数绝对值
        if 净胜 > abs_hcp:
            return "让胜"
        elif 净胜 == abs_hcp:
            return "让平"
        else:
            return "让负"
    else:  
        # 客队让球(>0)，主队受让
        # 让胜：主队赢 或 平（主队不败）
        # 让平：主队输1球
        # 让负：主队输2球及以上
        if 净胜 >= 0:
            return "让胜"
        elif 净胜 == -1:
            return "让平"
        else:  # 净胜 <= -2
            return "让负"
```

**验证示例**（基于5月26日实际数据）：
| 场次 | 让球 | 实际比分 | 净胜 | 让球结果 |
|:----|:---:|:-------:|:----:|:--------:|
| 2001 菲尔特2-0埃森 | -1 | 2-0 | +2 | ✅ 让胜 |
| 2002 圣埃蒂安0-0尼斯 | -1 | 0-0 | 0 | ✅ 让负 |
| 2003 拉努斯1-0米拉索尔 | -1 | 1-0 | +1 | ✅ 让平 |
| 2004 基多体大3-2时刻准备 | -1 | 3-2 | +1 | ✅ 让平 |
| 2005 弗拉门戈3-0库斯科 | -2 | 3-0 | +3 | ✅ 让胜 |
| 2006 蒙国民1-0科金博联 | -1 | 1-0 | +1 | ✅ 让平 |
| 2007 大学体育0-0托利马 | -1 | 0-0 | 0 | ✅ 让负 |

### 8.3 输出格式（每个结果只输出1个）

```
〖竞足-XXXX〗 [主队] VS [客队]（[联赛]）⚽ 让球[±N]
...
📌 因子检查...

🎯 正常比分：[比分A]/[比分B]
🎯 胜平负：[胜/平/负]
🎯 让球：[让胜/让平/让负]

🎯 异常比分：[比分A]/[比分B] 🏷️ [A1/B1等]

⭐ 信心：★★★☆☆
```

**推导规则**：
- **正常比分**：双选，λ模型选最可能的2个比分覆盖
- **胜平负**：模型综合λ差+因子检查+市场信号独立判定，**非从比分反推**
  - 例：λ差小+因子12一致降赔→即使预测2-0，胜平负也可能=平（冷平预警）
  - 例：因子4b一致升主胜→即使预测2-1，胜平负也可能=负（冷门预警）
- **让球**：模型综合盘口水位+欧指信号独立判定，**非从比分+让球数换算**
  - 例：让球-1+比分2-0，但亚指水位不支持穿盘→让平而非让胜
- **异常比分**：双选，含异常路径标签

**完整示例**（主队-1球，因子4b未触发，λ差0.8）：
```
🎯 正常比分：2-0/1-0
🎯 胜平负：胜
🎯 让球：让胜

🎯 异常比分：1-1/0-1 🏷️ A2·状态背离
```

让球结果对齐方式：
- 正常比分第1个的让球结果 → 主推让球方向
- 双选比分覆盖两个让球方向（如让胜/让平）
