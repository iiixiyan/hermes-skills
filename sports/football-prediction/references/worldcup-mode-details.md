## 一、总览：世界杯模式9步管线

```
第1步：确认比赛信息（阶段/排位/海拔/天气/旅行）

```

## 二、R系数计算（核心差异化参数）

### R_attack（进攻综合系数）

```
R_attack = R_伤病_进攻 × R_体能 × R_环境 × R_心理

R_伤病_进攻 = 0.4×(头号射手健康?1.0:0.65)
           + 0.3×(核心组织者健康?1.0:0.80)
           + 0.2×(边锋/边前卫健康?1.0:0.85)
           + 0.1×(更衣室?1.0:负面消息?0.90:1.0)
```

### R_defense（防守综合系数）

```
R_defense = R_伤病_防守 × R_体能 × R_环境 × R_心理

R_伤病_防守 = 0.5×(防线主力健康?1.0:0.80)
           + 0.5×(门将健康?1.0:0.80)
```

### R_体能

```
R_体能 = 赛程密集度系数 × 连续作战系数

赛程密集度：
  ≥5天间隔 → 1.0
  3~4天间隔 → 0.95
  ≤2天间隔 → 0.85

连续作战系数：
  小组赛第1轮 → 1.0
  小组赛第2轮 → 0.95
  小组赛第3轮 → 0.90
  加时赛后 → 0.85
```

### R_环境

```
R_环境 = 海拔系数 × 天气系数 × 旅行疲劳系数

海拔系数：
  0~500m → 1.0
  500~1500m → 低地国0.95, 高原国1.0
  1500~2500m → 低地国0.90, 高原国1.0
  >2500m → 低地国0.85, 高原国1.0
  高原适应：球队所在国平均海拔>1000m → 豁免

天气系数：
  高温>30°C → 0.95
  大雨/积水 → 总进球倾向缩小(>2.5球概率-10%)
  大风>20km/h → 小球倾向
  低温<5°C → 节奏放缓

旅行疲劳系数：
  <2000km → 1.0
  2000~5000km → 0.97
  5000~8000km → 0.95
  >8000km → 0.90
  时差>6h且到达<72h → 额外×0.95
```

### R_心理

```
R_心理 = 战意系数 × 大赛气质系数 × 更衣室系数

战意系数（从§五小组战略博弈获取）：
  战意极高(5/5)→1.10, 高(4/5)→1.0, 中(3/5)→0.90, 低(2/5)→0.80, 无(1/5)→0.70

大赛气质系数：
  传统强队/卫冕冠军 → 1.05
  首次参赛 → 0.95
  常规 → 1.0

更衣室系数：
  正常 → 1.0
  负面消息/不和 → 0.90
  严重内讧/将帅冲突 → 0.85
```

### λ修正公式

```
λ₁_final = λ₁_base × R_A_attack × (东道主修正?1.10~1.15:1.0)
λ₂_final = λ₂_base × R_B_attack

其中 λ_base 从59itou战绩Tab综合实力反推：
  λ_base = (50/(100-综合实力)) × 世界杯场均进球因子(~1.40)
```

## 三、🔀 双轨背离分析

### 轨道A：基本面得分

```
基本面得分 = 综合实力比(0~100) × 0.35
           + 阵容完整度(🟢=100/🟡=80/🟠=60/🔴=40) × 0.25
           + 伤病系数R(0~100) × 0.20
           + 环境适应度(0~100) × 0.10
           + 战意/心理(0~100) × 0.10

基本面倾向：得分差≥10→明显占优, ≥5→略占优, <5→均势
```

### 轨道B：市场隐含概率（Devig去抽水）

```
总概率 = 1/主胜赔 + 1/平赔 + 1/负赔
公平概率_主胜 = (1/主胜赔) / 总概率
公平概率_平 = (1/平赔) / 总概率
公平概率_负 = (1/负赔) / 总概率
```

### 碰撞判定矩阵

| 基本面 | 市场 | 判定 | 操作 |
|:------|:----|:----|:-----|
| 看好主队 | 看好主队 | ✅ 双轨共振 | 主方向坚定，信心维持 |
| 看好客队 | 看好客队 | ✅ 双轨共振 | 客方向坚定，信心维持 |
| 看好主队 | 看好客队 | ⚠️ 双轨背离 | 诱盘预警，异常比分必须含客胜 |
| 看好客队 | 看好主队 | ⚠️ 双轨背离 | 造热预警，异常比分含主胜 |
| 看好主队 | 平局高概率 | 🔶 半背离 | 防平，走主胜/平双选 |
| 均势 | 看好主队 | 🔶 市场过热 | 市场过度反应，走客队方向 |
| 均势 | 看好客队 | 🔶 市场偏好 | 参考但不主导，回归基本面 |

### 背离程度量化

```
背离系数 = |基本面倾向| / |市场倾向|

背离系数 > 1.5 → 深背离 → 冷门预警高
背离系数 1.2~1.5 → 中度背离 → 需防但不改主方向
背离系数 < 1.2 → 正常分歧 → 已消化
```

## 四、小组战略博弈（世界杯特有！）

### 出线形势判定

| 形势 | 条件 | 战意 | λ调整 |
|:----|:-----|:----|:------|
| 已确保小组头名 | 领先第2≥5分且剩1轮 | 低(2/5) | λ×0.70 |
| 确保出线但头名待定 | 领先第3≥4分 | 中(3/5) | λ×0.85 |
| 出线与淘汰之间 | 与第3分差≤2分 | 高(4/5) | λ维持 |
| 已淘汰 | 落后第2≥4分且剩1轮 | 无(1/5) | λ×0.80 |
| 生死战 | 必须赢/平才能出线 | 极高(5/5) | λ×1.10~1.15 |

### 默契球风险检查（小组赛第3轮）

```
□ 两队打平即可携手出线？ → 平局概率+30%；异常比分强制含1-1/0-0
□ 一场比赛结果直接影响另一场出线形势？ → 需关注同组另一场即时比分
```

### 战意差

```
战意差 = A队战意级别 - B队战意级别
战意差 ≥ 2级 → 高战意方倾向不败，低战意方倾向失败
战意差 = 0 → 正常对抗，回归基本面
```

## 五、环境因素详细评估

### 海拔适应度

- 比赛场地海拔高度从外部数据获取（网络搜索/赛事特征）
- 重点检查：低地国家（荷/英/丹等）在高海拔场地的影响

### 天气影响

- 高温(>30°C) → 双方R_环境×0.95
- 大雨/积水 → >2.5球概率-10%
- 大风(>20km/h) → 小球倾向
- 低温(<5°C) → 节奏放缓

### 裁判风格

- 出牌宽松(<3黄/场) → 身体对抗型球队受益
- 出牌严格(>5黄/场) → 技术型球队受益
- 偏向主队裁判 → 东道主额外加成

## 六、数据四重验证

| 验证 | 检查项 | 处理 |
|:----|:------|:-----|
| 真实性 | 来源权威（FIFA官网/59itou/Transfermarkt） | 非权威来源降级 |
| 有效性 | FIFA排名1~210、身价>0、综合实力0~100、场均进球0~5 | 无效→标注异常降权重 |
| 实时性 | 伤病≤3天、FIFA排名≤30天、身价≤90天、战绩≤7天 | 过期→降P2参考 |
| 合理性 | 数据之间自洽（高排名+低身价需解释等） | 冲突→标注可能原因 |

**验证结果影响：**
- 全部通过 → 🟢 正常预测
- 1项不通过 → 🟡 信心★★☆☆☆
- 2项不通过 → 🟠 信心★☆☆☆☆，仅方向预测无比分
- 3项+不通过 → 🔴 拒绝预测

## 七、淘汰赛路径影响

### 淘汰赛对阵镜像

```
已确保出线球队可能：
- 争头名→避开更强淘汰赛对手
- 故意第二→落入更有利半区（罕见但有先例）
```

### 黄牌累积风险

```
世界杯规则：累积2黄→停赛1场；1/4决赛后清零
核心球员已有1黄→可能避免吃牌（防守强度下降）
  → 防守λ×1.05~1.10
```

## 八、worldcup-prediction 与 football-prediction 隔离规则（用于双轨对比模式）

当双轨对比模式下**联赛管线**运行世界杯比赛时，以下参数必须豁免：

| 管线 | 可用 | 禁用 |
|:----|:----|:-----|
| 联赛模式(对比用) | λ泊松分布、4b/12/13因子(P1级别)、让球判断 | 友谊赛收兵系数、友谊赛半衰期3、A5弱队不败、友谊赛主队主场权重 |
| 世界杯模式(主用) | R系数、双轨背离、环境因素、小组战略、淘汰赛路径 | 14因子(降P2)、联赛半衰期、等级战绩xG代理 |

**世界杯比赛特殊边界条件：**
- 禁用友谊赛收兵系数（世界杯不收兵）
- 禁用友谊赛半衰期3（用标准half_life=8）
- 禁用A5实力悬殊弱队不败（世界杯实力差≠友谊赛）
- 13双边分歧在东道主比赛中降P2参考
- 亚指高水信号在东道主比赛中降P2参考
