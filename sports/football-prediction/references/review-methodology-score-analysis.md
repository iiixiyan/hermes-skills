> 2026-06-01 用户要求"重点对竞足比赛复盘比分情况"后系统化。每次复盘必须按以下6维执行。
> v2.0 2026-06-14 更新：新增偏差归因树(§6)、欧指亚指分歧比分处理规则、东道主因子对比分影响、比分命中分类体系。

## ① λ偏差量化矩阵

逐场计算 **预测λ₁+λ₂ vs 实际总进球**，定位λ的系统性高估/低估。

### 计算公式

```
偏差率 = (实际总进球 - 预测λ总和) / 预测λ总和 × 100%
```

### 阈值判定

| 偏差率 | 判定 | 行动 |
|:------|:-----|:-----|
| < 15% | 🟢 优秀 | λ预测精准，维持参数 |
| 15-30% | 🟡 可接受 | 检查联赛参数是否需要微调 |
| 30-50% | 🟠 轻度偏差 | 检查因子是否漏触发或系数过激 |
| 50-100% | 🔴 中度偏差 | 强制产出优化提案 |
| > 100% | 🆘 严重偏差 | 必须检查数据真实性+模型参数双重问题 |

### 比分命中分类体系

复盘时必须先对每场比赛的比分预测做**三分类**，而非笼统的"命中/未命中"：

| 分类 | 条件 | 占比(目标) |
|:----|:-----|:----------|
| 🎯 精确命中 | 预测比分与实际完全相同 | >20% |
| 🟡 方向命中 | 胜平负方向正确但比分不同 | >60% |
| 🔴 方向错误 | 胜平负方向错误（含欧指亚指信号误判） | <20% |

**方向命中的再细分**：
- 偏差1球（含预测2-1实际3-1、预测0-0实际1-1等）
- 偏差2+球（λ系统性高估/低估）
- 反方向命中（如预测客胜实际主胜但爆冷覆盖了）

**此分类必须在复盘最前面展现**，然后在偏差归因中逐场解释原因。

## ⑤ 系统偏差识别

跨场次寻找共同偏差方向。检查信号：
1. 某联赛的λ全部偏低/偏高？
2. 某赛事类型的某个因子持续误触发？
3. 主客场系数系统性偏差？
4. 所有偏差集中在同一方向（全部高估/全部低估）？

### 输出原则

每次复盘必须产出 ≥1 条可执行的优化提案。若无系统偏差，明确声明并说明检查了什么。

## ⑥ 偏差归因树（v2.0 新增 — 每场未命中比分必须追溯至底层原因）

复盘时每一场未命中的比分，不能停留在"比分错了"层面，必须按以下归因树追溯**到底层根因**：

### 归因树结构

```
比分未命中
├── 🅰 λ偏差（模型参数问题）
│   ├── A1 进攻λ高估 → 联赛参数/友谊赛系数/收兵系数需要调整
│   ├── A2 进攻λ低估 → 球队近期爆发/跨级实力被低估
│   ├── A3 防守λ高估 → 防线伤病/客场崩盘未反映在模型
│   └── A4 防守λ低估 → 对手攻击力被收兵系数过度压缩
├── 🅱 因子判断失误（因子触发或未触发的问题）
│   ├── B1 4b触发但应失效（杯赛决赛/中立场/换帅/东道主） → 强制检查失效条件
│   ├── B2 4b/12信号分歧（欧指看A方但亚指看B方）→ 正常比分应包含双方向
│   ├── B3 因子漏触发（指数变化段未加载导致遗漏）→ 数据采集完整性

├── 🅲 数据问题（采集/质量）
│   ├── C1 欧指指数变化段缺失 → 采集重载流程
│   ├── C2 战绩Tab数据缺失(<200字) → 降级λ计算
│   ├── C3 阵容Tab伤停信息缺失 → 伤停影响未被计入
│   └── C4 对比分影响：某个数据源加载了错误的比赛 → 数据真实性验证(§6.11)
├── 🅳 市场反转（欧指/亚指信号正确但市场方向反转）
│   ├── D1 世界杯东道主战 → 东道主加成导致极端市场信号降级
│   ├── D2 战意驱动反转（出线/保级/轮换）→ 战意差≥2级时走战意强方向
│   └── D3 深盘冷门（SP>5.0的客胜/冷平）→ 散户驱动反向，标记"不可控"
└── 🅴 其他
    ├── E1 极端天气/突发事件 → 标注但不追责模型
    ├── E2 红牌/点球等偶然事件 → 标注"方差噪音"
    └── E3 比赛延期/取消 → 标记为"未进行"
```

### 归因操作流程

**步骤1**：每场未命中 → 选1个归因编码（A1-A4/B1-B3/C1-C4/D1-D3/E1-E3）
**步骤2**：从归因编码反推优化方向
**步骤3**：统计本日归因分布 → 哪个编码出现最频繁就是系统偏差

### 6/13归因案例

| 场次 | 未命中原因 | 归因编码 | 优化方向 |
|:----|:----------|:--------|:---------|
| 6203 雅罗2-5赫尔辛基 | 赫尔辛基进攻λ低估300% | A2 | 芬超强队客场进攻λ修正 |
| 6204 瓦萨1-1库普斯 | ✅精确命中 | - | - |
| 6205 玛丽港0-3格尼斯坦 | 偏差1球，λ₂略低估 | A2 | 格尼斯坦近期状态权重可略提高 |
| 6206 拉赫蒂2-3塞伊奈 | 欧指23降主胜vs亚指13高水分歧，正常比分未覆盖客胜 | B2 | 欧指亚指分歧≥2项→正常比分覆盖双方向 |
| 6005 卡塔尔1-1瑞士 | 东道主战极端市场信号未充分降级 | D1 | 东道主战正常比分必须含东道主不败 |
| 6006 巴西1-1摩洛哥 | ✅精确命中 | - | - |

## ⑦ 欧指亚指分歧对比分的影响规则（v2.0新增）

当同一场比赛出现**欧指和亚指信号直接矛盾**时，**正常比分必须覆盖双方向**，不得仅跟随其中一方。

### 分歧判定标准

| 信号 | 欧指方向 | 亚指方向 | 判定 |
|:----|:--------|:--------|:----|
| 一致 | 主胜 | 低水/升盘 | ✅ 无分歧 |
| 弱分歧 | 主胜 | 高水但升盘数=降盘数 | ⚠️ 方向不变但下调穿盘信心 |
| **强分歧** | **主胜(23降主胜)** | **13家高水(看衰穿盘)** | **🔴 正常比分必须含客胜方向** |
| 反向分歧 | 客胜(22升客胜) | 降盘→主(盘口向主队移动) | 🔴 正常比分需含主队不败 |

### 实战案例（6/13）

**6206 拉赫蒂2-3塞伊奈**：
- 欧指：23降主胜/5升 → 强烈看主胜
- 亚指：主队13家高水/3家低水 → 看衰穿盘
- 当前处理：让平（降穿盘预期）✅
- 处理缺陷：**正常比分未覆盖客胜方向** → 仅给了1-0/2-1
- ✅ 修正：当期指亚指强分歧时，正常比分双选应各走一个方向，如：`2-1（主胜薄）/1-2（客胜防）`

### 强制执行规则

```
当以下条件同时满足时，强制触发「强分歧比分双覆盖」：
① 欧指看好A方（≥20家降A方或≥20家升B方）
② 亚指看好B方（A方高水家数≥低水家数2倍 或 A方降盘家数≥升盘家数2倍）
③ 两信号方向相反

→ 正常比分：A方向比分/B方向比分（各1个，双选）
→ 爆冷比分中必须覆盖亚指看好方向
→ 信心降1★
```

### 世界杯东道主因子

在世界杯比赛中，东道主战存在系统的比分修正规则：

```
东道主战市场信号降级清单：
□ 极端共识信号（一致降客胜/降盘）→ 信心降1★
□ 东道主拿分概率显著高于FIFA排名差建议
□ 正常比分必须包含东道主不败方向（1-1/1-0等）
□ 亚指高水信号降级为P2参考
```

### 6/13案例验证

**6005 卡塔尔1-1瑞士**（FIFA56 vs FIFA19）：
- 市场信号：28降客胜+17降盘，极端共识
- 实际结果：1-1平，东道主拿分
- 当前规则已添加（v4.9.20 §6.12），但复盘时必须检查该规则是否在执行中被遵守

### 出线/保级战意对λ的比分影响

当一支球队处于以下状态时，其进攻λ需要调整：

| 状态 | λ修正 | 比分影响 |
|:----|:------|:---------|
| 已锁定小组头名 | λ×0.70~0.80 | 比分预期降低，优先小球 |
| 已出局无战意 | λ×0.60~0.70 | 比分预期大幅降低 |
| 保级生死战(末5轮) | 防守λ×0.85（更紧） | 比分倾向1-0/1-1/0-0小比分 |
| 升级附加赛生死战 | 进攻λ×1.15~1.30 | 比分倾向大球/对攻 |

## ⑨ 比分偏差精细矩阵（v3.0新增 — 逐数字拆解误差）

> **核心思路：** λ偏差量化只回答了「总进球偏差」，但比分是两个数字的乘积。  
> 必须拆解到**主队进球偏差**和**客队进球偏差**两个独立维度，才能定位λ₁和λ₂哪个不准。

### 计算公式

```
主队进球偏差 = 实际主队进球 - 预测λ₁
客队进球偏差 = 实际客队进球 - 预测λ₂
进球分布偏差 = abs(主队进球偏差) - abs(客队进球偏差)
  → 正值=主队λ更不准，负值=客队λ更不准
主客同向偏差 = 主队进球偏差 × 客队进球偏差 > 0
  → 正数=双方同向高估/低估（联赛参数问题）
  → 负数=一高一低（交叉乘积/攻防匹配问题）
```

### 偏差矩阵示例（6/13实战）

| 场次 | 预测λ₁/λ₂ | 实际比分 | λ₁偏差 | λ₂偏差 | 分布偏差 | 同向性 | 诊断 |
|:----|:---------:|:--------:|:------:|:------:|:--------:|:------|:-----|
| 6203 雅罗2-5赫尔辛基 | 0.7/1.8 | 2-5 | +1.3 | **+3.2** | 客队不准 | 同向(+) | λ₂严重低估 |
| 6204 瓦萨1-1库普斯 | 1.4/1.3 | 1-1 | -0.4 | -0.3 | 均衡 | 同向(-) | 双方λ偏保守 |
| 6205 玛丽港0-3格尼斯坦 | 0.6/1.5 | 0-3 | -0.6 | **+1.5** | 客队不准 | 反向(-) | λ₁准但λ₂低估 |
| 6206 拉赫蒂2-3塞伊奈 | 1.6/1.1 | 2-3 | +0.4 | **+1.9** | 客队不准 | 同向(+) | λ₂严重低估+塞伊奈爆发 |
| 6005 卡塔尔1-1瑞士 | 0.4/2.2 | 1-1 | **+0.6** | -1.2 | 主队不准 | 反向(-) | 东道主效应λ₁过低 |
| 6006 巴西1-1摩洛哥 | 1.6/1.3 | 1-1 | -0.6 | -0.3 | 均衡 | 同向(-) | 双方λ偏保守 |

### 偏差诊断法则

| 模式 | 判断 | 行动 |
|:----|:-----|:-----|
| **λ₁与λ₂同向偏差**（均为正或均为负） | 联赛总进攻参数系统性偏高或偏低 | 调整联赛先验均值 |
| **λ₁与λ₂反向偏差**（一正一负） | 交叉乘积/攻防匹配问题 | 检查攻击×防守交叉乘积 |
| **客队偏差>主队偏差2倍** | 客队λ参数系统性问题 | 检查客场λ系数/弱队客场λ |
| **主队偏差>客队偏差2倍** | 主场优势系数问题 | 检查HOME_ADVANTAGE参数 |
| **主队进球偏差绝对值<0.3且客队>1.0** | 客队爆发不可预测 | 标记A2归因，检查客队近期状态 |

### 6/13偏差诊断结论

- **主要问题在λ₂（客队λ）**：5场方向正确的比赛中，客队λ偏差绝对值平均1.5，主队λ偏差平均0.6
- 交叉乘积基本合理（没有系统性反向偏差）
- 6005反向偏差（λ₁低估+λ₂高估）= 东道主特殊赛事，需降级处理

## ⑩ 比分偏差热力图（v3.0新增 — 跨场次寻找模式）

### 操作流程

每次复盘后，将所有偏差按以下维度填入热力图：

```
                 主队λ 高估 → → → 主队λ 低估
                   ↓  ← ← ←   ↓
                   ↓  过度保守   ↓
  客队λ 高估 →     A区           B区
                   (双高估)      (客队准主队低估)
                   ↓             ↓
  客队λ 准确 →     C区           D区
                   (主队高估)    (理想区)
                   ↓             ↓
  客队λ 低估 →     E区           F区
                   (主高客低)    (双低估→保守)
```

**6/13热力图分布：**
- F区（双低估—保守）：6204、6006（2场→λ模型偏保守）
- E区（主高客低）：6205（1场→客爆发）
- B区（客低估）：6203、6206（2场→客队λ系统性低估）

**结论：6/13偏差集中在F区（保守）和B区（客低估），无A区（激进）案例。**  
→ λ模型整体偏保守，尤其在客队进攻端

## ⑪ 可避免误差 vs 不可避免误差（v3.0新增）

### 分类标准

| 类别 | 定义 | 占比目标 |
|:----|:-----|:---------|
| ✅ 可避免误差 | 模型参数/因子判断/数据采集可以改进的误差 | 应逐步降至<50% |
| ⚠️ 半可控误差 | 特殊赛事（东道主/杯赛首轮）有规律但样本少 | 标注为已知盲区 |
| ❌ 不可避免误差 | 红牌/点球/极端天气/方差噪音 | 不追责，标注即可 |

### 6/13误差分类

| 场次 | 误差类型 | 分类 | 改善路径 |
|:----|:--------|:----|:---------|
