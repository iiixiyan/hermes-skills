# 比分深度复盘方法论（5维分析）

> 2026-06-01 用户要求"重点对竞足比赛复盘比分情况"后系统化。每次复盘必须按以下5维执行。

## ① λ偏差量化矩阵

逐场计算 **预测λ₁+λ₂ vs 实际总进球**，定位λ的系统性高估/低估。

### 计算公式

```
偏差率 = (实际总进球 - 预测λ总和) / 预测λ总和 × 100%
```

### 阈值判定

| 偏差率 | 判定 | 行动 |
|:------|:-----|:-----|
| < 20% | 可接受 | 无需修复 |
| 20-50% | 轻度偏差 | 检查联赛参数 |
| 50-100% | 中度偏差 | 检查因子失效条件 |
| > 100% | 严重偏差 | 必须产出优化提案 |

## ② 半全场转化规律

按半场形态分组统计，总结规律。

- 主队半场领先1球+：优先走"主胜"

## ③ 穿盘率分析

按让球数分组统计赢盘/走水/输盘。

### 关键发现（5/31）

| 让球 | 结果 | 规律 |
|:----|:-----|:-----|
| -1 | 3场全部走水(让平) | 让-1走水率100%，让平是高概率项 |
| -2 | 分化：日本输盘/瑞士穿盘/德国穿盘/巴西穿盘 | FIFA差决定穿盘能力 |
| +1(受让) | 赫根3-2赢盘 | 主场强队受让时赢盘率高 |

### 让-1走水规律

当同时满足以下条件时，让平作为默认选项：
- 主胜SP在1.50~1.70（非绝对碾压）
- 对手有一定防守能力（非鱼腩）
- 比赛非杯赛/友谊赛

## ⑤ 系统偏差识别

跨场次寻找共同偏差方向。检查信号：
1. 某联赛的λ全部偏低/偏高？
2. 某赛事类型的某个因子持续误触发？
3. 主客场系数系统性偏差？
4. 所有偏差集中在同一方向（全部高估/全部低估）？

### 输出原则

每次复盘必须产出 ≥1 条可执行的优化提案。若无系统偏差，明确声明并说明检查了什么。
