## P0-P11+P-TR 完整规则（执行顺序严格）

### P0: 东道主
**检测:** 队名包含"美国"或"卡塔尔"
- 美国 → **3-0 / 4-1**
- 卡塔尔 → **1-1 / 0-1**

### P0.5: 绝对碾压+强平信号
**检测:** `fifa≥60 AND f2c≥25(强平降)`
- **1-1 / 2-2**

### P1: 绝对碾压（三级分级）
**检测:** fifa差≥60 或 (fifa差≥40且o1<1.20)
- 分支A: `fifa≥70 AND o1<1.05` → **4-0 / 6-0**（极端碾压，如德国7-1库拉索）
- 分支B: `fifa≥60 AND 主FIFA≤5` → **0-0 / 3-0**（超级强队防冷，如西班牙0-0佛得角）
- 分支C: fifa≥60 → **3-0 / 5-0**
- 分支D: fifa≥40+o1<1.20 → **3-0 / 5-0**

### P1-R2: Round 2深盘高水控赔
**检测:** Round 2比赛 AND FIFA差>25 AND 主高水≥12 AND 主热门
- 含义: 深盘中高水非阻盘实为控赔降赔付
- **3-0 / 4-0**
- 案例: 瑞士4-1波黑(FIFA差44, 高水12) 🟡偏差1

### P2: 骑墙（同等极端降主胜+排除客胜）
**检测:** `f1c≥40 AND r3≥40 AND o1<o3`
- ⚠️ 东道主(加拿大)豁免此规则
- **1-1 / 2-0**

### P5: 造热（优先于P3/P4/P6）
**检测:** `h_excl=True AND o1<o3 AND NOT(f1<f2)`
- 含义: 排名倒挂(弱队FIFA差)+市场卖主买客→走反方向
- **2-0 / 1-1**
- ⚠️ **优先级高于P3/P4/P6**——当P3和P5同时触发时,P5胜出
- 案例: 加纳1-0巴拿马(FIFA73vs34, 49升主胜+48降客胜)

### P3: 极端升主胜+平降（核心防平规则）
**检测:** `r1≥40(升主胜) AND f1c≤5(降主胜≤5) AND f2c≥15(平降)`
- 分支A (三向分歧): `f3c≥40 AND 主FIFA更强` → **3-0 / 2-1**（市场全面买客→主零封）
  - ⚠️ 此分支优先于fh≤10检查
- 分支T (顶级强队): `主FIFA≤5` → **2-1 / 3-1**（不防平）
- 分支A (均势): `主FIFA≤10` → **1-1 / 2-2**
- 分支C: 其他 → **2-1 / 3-1**

### P4: 真正客强
**检测:** `h_excl=True AND o3<o1`
- 分支A: `o3>1.45(浅盘客)` → **2-0 / 1-1**（造热反走）
- 分支B: 其他(深盘客强) → **0-3 / 1-4**

### P9C: 加拿大东道主大胜
**检测:** 加拿大主场 AND Round 2+ AND 一致降主胜(c1≥25) AND r1≤5
- **4-0 / 5-0**
- 案例: 加拿大6-0卡塔尔(FIFA32vs49, 44降主胜)

### P6: 极端升主胜无平降
**检测:** `r1≥40 AND f1c≤5 AND f2c<15`
- 主热门(o1<o3) → **2-1 / 3-1**
- 其他 → **1-1 / 2-0**

### P7: 排除客胜（极端升客胜）
**检测:** `r3≥40 AND f3c≤5`
- 分支A: `o3<1.60(客低赔)` → **0-1 / 1-2**（假信号, 客仍赢）
- 分支X (主更强): `FIFA差≤15 AND 主FIFA更强`:
  - 有一致降主胜(c1≥25) → **4-0 / 3-1**（市场一致买主→大胜）
  - 无一致降主胜 → **3-0 / 2-1**
- 分支B1 (均势主热门): `FIFA差≤15 AND o1<o3` → **2-1 / 1-1**
- 分支B2 (均势主冷门): `FIFA差≤15 AND NOT(o1<o3)` → **1-0 / 2-1**
- 分支C (有平降): `f2c≥15` → **2-1 / 1-1**
- 分支D (大差距): `FIFA差≥20` → **2-0 / 3-0**
- 分支E: 其他 → **2-1 / 1-1**

### P8: 强平降/双边分歧
**检测:** `f2c≥25(强平) OR (r1≥20 AND r3≥20)(双边分歧)`
- 分支A: `FIFA差≥20 OR (主FIFA更差且差≥5)` → **1-1 / 0-0**
- 分支C-TOP (顶级强队高比): `主FIFA≤5 AND hb信号` → **3-1 / 4-2**
- 分支C: `hb(主平衡+客极端卖)` → **2-0 / 3-0**
- 分支B: 其他(均势) → **1-1 / 2-2**

### P9: 一致降主胜
**检测:** `f1c≥25 AND r1≤10`
- **2-0 / 3-0**

### P10: 一致降客胜
**检测:** `f3c≥25 AND r3≤10`
- 分支A: `FIFA差≥15 OR 客FIFA更强` → **0-2 / 0-3**
- 分支B: 其他 → **0-2 / 1-2**

### P11: 弱平信号
**检测:** `f2c≥15`（无其他极端信号时）
- **1-1 / 2-0**

### P-TR: 三向全升分歧
**检测:** `f1c≥25(降主) AND r2≥25(升平) AND r3≥25(升客) AND f2c≤10 AND f3c≤10`
- **1-1 / 1-0**
- 案例: 捷克1-1南非(36降主+39升平+37升客)

### 默认
- 主队赔率热门(o1<o3) → **2-0 / 2-1**
- 客队赔率热门 → **0-2 / 1-2**
