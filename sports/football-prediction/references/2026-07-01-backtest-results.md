## ESPN赛果提取方法

```javascript
// 导航到ESPN scoreboard
// https://www.espn.com/soccer/scoreboard/_/league/fifa.world/date/20260701
// 等待页面加载后运行：
document.body.innerText

// 赛果在DOM中的结构：
// [队名缩写][主队分][客队分][状态]
// 如：ENG2COD1FT → 英格兰2-1刚果金 FT
//     BEL3SEN2AET → 比利时3-2塞内加尔 AET
// 解析正则：/([A-Z]{3})(\d+)([A-Z]{3})(\d+)(FT|AET|HT)/
```
