## 博单比分选择方法论（2026-05-25）

针对用户要求"博单单选比分"场景，按以下流程执行：

### 前提条件
- 用户使用**索冷引擎 ColdSearch 比分精算**作为AI比分参考源（四维比分+SP赔率）
- 用户**博单偏好比分池**（单选高赔）：`4-1 / 3-2 / 2-2 / 2-3 / 3-3 / 1-4 / 0-4 / 0-2`
- 禁止双选，只选**1个比分**

### 选拔流程

```
Step 1：从用户提供的ColdSearch数据中，提取每场的四维比分+SP
Step 2：筛选出属于博单偏好池的比分候选
Step 3：对每个候选，检查其方向是否与ColdSearch最低赔方向一致
  → 一致：加分（方向有支撑）
  → 不一致：降权（AI主方向不支持）
Step 4：从合格候选中选SP最高且方向有支撑的那个
  → SP不宜超过30（太高的SP基本打不出）
  → SP不宜低于8（太低的SP不算博单）
  → 最优区间：SP 15~25
```

### 实战案例

```
ColdSearch数据：
  007 埃尔夫斯堡vs赫根：1:2(8), 2:3(22), 1:0(9.5), 2:1(9)

分析：
  1. 博单池匹配 → 2:3(22) ✅
  2. ColdSearch最低赔 = 1:2(8)，方向=客胜
  3. 2:3方向也是客胜 → 与主方向一致 ✅
  4. SP=22，在15-25最优区间 ✅
  5. 最终推荐：2:3 @ 22
```
