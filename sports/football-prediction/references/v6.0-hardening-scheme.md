## 3. 队伍实力对比系统（强制）
```python
实力分 = FIFA排名(30%) + 59itou实力分(25%) + 身价(20%) + 状态(15%) + 评分(10%)
```
- 实力差>20+阵容完整：lambda_强队*1.10 / lambda_弱队*0.90
- 实力差绝对值>40：lambda_强队*1.15
- 冷门保护：赔率已有冷门信号时不改比分
