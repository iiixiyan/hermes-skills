## 引擎偏差归因

### P11-Hot-B 路由追踪(Germany vs Paraguay)

```
P11-Hot-B(第674-688行):
  xa=True(r3=43,c3=3) ✅
  c1=38≥25 ✅
  fh=12≤15 ✅
  fd=25≥15 ✅
  o1=1.375<5.0 ✅
  → bl=True? c1=38<40 → False → 不进入造热完满跳过(第680行)
  → is_r1? rd=0 → False → 不走首轮防平(第684行)
  → Else → return (3, 0, P11-Hot-B, 4) ← ❌ 淘汰赛不应走此路径
```

### 根因

1. **引擎无R32淘汰赛专用规则**: 淘汰赛(rd=0)被当作未知轮次处理，所有P0/P1规则按小组赛逻辑执行
2. **hd+xa+r2_high三信号同时触发**: 在小组赛=真实方向共识，在淘汰赛=造热/过度自信
3. **巴拉圭防守韧性被低估**: 小组4场仅失4球(0澳/1美/0土)的防守能力未纳入λ计算
4. **德国末轮失利被乐观解读**: "会更专注"假设无数据支撑
