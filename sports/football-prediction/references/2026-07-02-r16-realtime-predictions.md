## 关键发现

### 未来比赛采集工作流

```python
# 1. 扫描日期获取未来比赛列表
# 2. 对每个matchId调用footballMatchDetail获取队名+联赛
# 3. 筛选league='世界杯'
# 4. 对世界杯比赛调用footballMatchOddsEuro获取8参数
# 5. 计算百家平均+8参→规则链匹配→输出预测
```

### 数据漂移对比模板
R16预测时应与小组赛数据做对比（如已存在预测）。哥伦比亚vs加纳的-16.5%主胜漂移是强烈的方向确认信号。

### 比赛日推算
- Sina `matchTime` 为UTC时间戳
- BJ时间 = timestamp + 8h
- 如1783101600→07-04 10:00 BJ
- 彩票销售日(date_key) ≈ 比赛实际日前1天
