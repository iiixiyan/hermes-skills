### 因子1：大比分低估修正（近3场权重提升）

**发现**：λ 模型系统性低估实际比分，多场比赛实际进球 > 预期（纽卡3-1、曼联3-2、巴萨3-1、大阪6-1）。
**原因**：赛季均值参数拉低了近期的进攻爆发信号。
**修正式**：对于近 3 场场均进球 > 赛季均值 × 1.3 的球队，将 λ 公式调整为：
```
λ = (近3场均值 × 0.50) + (近6场均值 × 0.25) + (赛季均值 × 0.25)
```
即**近 3 场权重从 40% 提升至 50%**，当出现「近期爆发」信号时使用。

### 因子2：赛季末段（最后5轮）战意差检查（强制规则）

**发现**：尤文0-2佛罗伦萨(SP10.00)、里尔0-2欧塞尔(SP7.60) 两大超级冷门均发生在赛季末段，主队「无欲无求」vs 客队「有明确目标」。
**强制检查**（赛季末最后5轮时，无论用户是否要求，必须检查）：
```python
if (当前轮次 >= 联赛总轮次 - 5) {
  1. 检查主队是否已无争冠/欧战/保级压力
  2. 检查客队是否有明确目标（保级生死线/欧战资格线）
  3. 检查主队周中比赛（杯赛/欧战）导致轮换
  4. 若「主队无欲无求」且「客队战意充沛」→ 强制触发异常权重 +2 级（非+1级）
}
```
**适用联赛**：五大联赛、荷乙、挪超、瑞超、芬超、葡超等有明确升降级制度的联赛。**次级联赛同一适用**——联赛末期（倒计时5轮），排名靠前但无欲无求vs保级队时，战意差驱动效果同样显著。

**加强规则**：

### 因子4：赛前SP赔率预判法

当需要快速判断一场比赛是否为「冷门候选」时：
- 检查竞足列表页的 胜/平/负 赛前赔率
- 将三个赔率转为隐含概率（1/赔率）并归一化

### 因子19：超低赔率爆冷预警（Ultra-Low Odds Cold Alert）v2

**发现来源**：soccer-lottery 开源项目方法论借鉴 + 竞足/北单实战长期观察。低赔主胜（<1.50）场次天然具备高爆冷风险，尤其当市场集中热捧导致初赔过度压低时，弱队不败回报率极高。

- 庄家无需进一步降赔即可平衡资金，因此因子4b（升赔检测）可能不触发
- 但弱队一方保持不败（冷平/客胜）的回报率天然高于常规场次
- 临场变数（轮换、早早进球、红牌）对低赔场次的方向冲击更大
- 短期样本中低赔主胜的冷门率约15-25%（统计值）

**条件**：
```python
if 百家平均主胜初赔 < 1.50:  # 市场最热选项为深度低赔
```

在 λ_total < 1.5 的超小比分局中，市场信号（欧赔升/降家数）的预测能力**显著下降**。原因是：
- 低λ比赛天然进球稀缺，一个意外进球就能完全改变方向（如拉赫蒂的1-0）
- 市场对低λ比赛的赔率调整更多反映**流动性不足**而非真实共识

**根因回溯**：在引擎执行顺序中，市场信号修正（Layer 1）先于主场地板检查，当市场信号指向客队方向时，因子11被跳过或覆盖了。正确做法：λ_total<1.5时，因子11必须在所有信号修正**之后**强制执行。

### 条件

```python
if λ_h + λ_a < 1.5:  # λ_total < 1.5，超小比分局
    if 主队是排名下风方（主排名 > 客排名）:  # 弱队主场

        λ_h_final = max(λ_h_final, 1.0)  # 弱队主场至少保底1球
        λ_a_final = min(λ_a_final, λ_h_final * 2.0)  # 客队进球不超过主队2倍
```

- **本质相同：两端资金极端化，中间方向(平局)形成真空 → 平局概率非线性上升**
