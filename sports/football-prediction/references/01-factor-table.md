### 因子1：大比分低估修正（近3场权重提升）

**发现**：λ 模型系统性低估实际比分，多场比赛实际进球 > 预期（纽卡3-1、曼联3-2、巴萨3-1、大阪6-1）。
**原因**：赛季均值参数拉低了近期的进攻爆发信号。
**修正式**：对于近 3 场场均进球 > 赛季均值 × 1.3 的球队，将 λ 公式调整为：
```
λ = (近3场均值 × 0.50) + (近6场均值 × 0.25) + (赛季均值 × 0.25)
```
即**近 3 场权重从 40% 提升至 50%**，当出现「近期爆发」信号时使用。

### 因子2：赛季末段（最后5轮）战意差检查（强制规则）

**发现**：尤文0-2佛罗伦萨(SP10.00)、里尔0-2欧塞尔(SP7.60) 两大超级冷门均发生在赛季末段，主队「无欲无求」vs 客队「有明确目标」。
**强制检查**（赛季末最后5轮时，无论用户是否要求，必须检查）：
```python
if (当前轮次 >= 联赛总轮次 - 5) {
  1. 检查主队是否已无争冠/欧战/保级压力
  2. 检查客队是否有明确目标（保级生死线/欧战资格线）
  3. 检查主队周中比赛（杯赛/欧战）导致轮换
  4. 若「主队无欲无求」且「客队战意充沛」→ 强制触发异常权重 +2 级（非+1级）
}
```
**适用联赛**：五大联赛、荷乙、挪超、瑞超、芬超、葡超等有明确升降级制度的联赛。**次级联赛同一适用**——联赛末期（倒计时5轮），排名靠前但无欲无求vs保级队时，战意差驱动效果同样显著。

**⚠️ 2026-05-20教训：威廉二世（荷乙第3）1-2福伦丹（保级队）**
预测方向完全打反（预测主胜，实际客胜SP3.15）。威廉二世虽排名第3，但荷乙无直接升级资格（仅附加赛），实际动力不足；福伦丹保级战意充沛。
**加强规则**：
```
if (联赛是荷乙/芬超/挪超等次级或北欧联赛 && 赛季末最后5轮) {
  // 这些联赛的排名含金量低于五大联赛
  // 排名前5但距离直升区差距≥6分 → 标记为「伪强队，战意存疑」
  // 排名后5且距离降级区≤3分 → 标记为「战意充沛的保级队」
  // 伪强队 vs 保级队 → 战意差权重升级（异常权重+2级，不可为+1级）
}
```

### 因子4：赛前SP赔率预判法

当需要快速判断一场比赛是否为「冷门候选」时：
- 检查竞足列表页的 胜/平/负 赛前赔率
- 将三个赔率转为隐含概率（1/赔率）并归一化
- 若市场最热选项概率 > 60% → 该结果大概率打出
- 若市场最热选项概率 < 40% → 高不确定性场次，异常比分概率显著上升
- **难点**：高SP冷门（SP>5.00）往往在赔率上已有信号但常被忽视——当某选项赔率初赔到即赔下降超过 15% 时，需高度警惕该方向

### 因子4b：「一致升赔」冷门识别信号（✅验证15/15）

**条件**：某选项 ≥20家升赔 且 ≤2家降赔。扩展：≥25家升赔即使涨幅<5%也有效。

**强制规则**：
1. 升赔方向的反方向必须出现在异常比分中
2. 正路信心 -1★（基础）；升赔方向=市场最热时额外-1★（共-2★）；叠加因子6时再-1★（最大-3★）
3. 叠加因子6时→冷门方向进入正常比分第2个

验证记录：挪超(2)瑞超(2)荷乙(1)葡超(1)芬超(1)意甲(1)德乙(1)日职(1)沙职(1)英超(3)瑞超(1)=15/15
**2026-05-24新增验证**：桑德兰vs切尔西(客27升/1降→客胜未出✅)、热刺vs埃弗顿(客30升/0降→客胜未出✅)、诺丁汉vs伯恩茅斯(主29升/0降→主胜未出✅)、马尔默vs韦斯特罗(主28升/1降→主胜未出✅)
**详细案例库**：`references/ouzi-signal-cases-20260524.md` — 含27场完整欧指信号分析

**失效条件**：休赛期末轮/中立场地/近3场换帅/杯赛决赛
**操作流程**：升赔家数→降赔家数→赔率涨幅→亚指降盘→判定
**⚠️ 注意**：此信号与因子6(排名倒挂)叠加时威力倍增。因子4b+因子6双触发 → 异常系数+2级。

### 因子4b-EXT：一致升赔边界扩展信号（🔶新引入·观测中）

**动机**：AC米兰1-2卡利亚里(SP11.80)当晚最大冷门，主胜21升/3降恰好在4b严格边界外(降=3>2)。诺丁汉1-1伯恩茅斯、马尔默2-3韦斯特罗等边界案例也反复出现。降赔数在3-5区间的"准4b"信号同样携带信息，但可靠性低于严格4b。

**条件**：某选项 ≥20家升赔 且 3家≤降赔≤5家（即接近但不满足严格4b的≤2降要求）
**附加条件**：初赔→即赔变化幅度 ≥ 3%（防止阈值放宽后被噪声触发）
**扩展条件**：升赔≥18且降赔≤2且幅度≥5%（当升赔数略低于20但幅度大时也视为有效）

**阈值**：
- 触发→异常比分中反方向出现：+1★权重（非严格4b的-1★力度）
- 同时满足附加条件（幅度≥3%）→ 提升至-1★（与严格4b同权）
- 叠加因子12时→升级为「双边分歧」→转因子13处理

**验证案例（🔶观测中，需更多数据）**：

**与严格4b的协同**：
- 严格4b（≤2降）→ P0最高优先级，正路-1★

### 因子5：平局信号 — 主胜即赔升水

**条件**：当多个指标同时指向平局时，平局概率显著上升：
1. 百家平均主胜即赔 > 初赔（主胜赔率上升）= 市场不看好主胜
2. 百家平均负赔即赔 < 初赔（客胜赔率下降）= 市场更看好客队不败
3. 双方综合实力差距 < 10分（如39-61以内）
4. λ1与λ2差异 < 0.50

**阈值**：匹配≥3条时，正常比分第1个优先考虑平局比分。

### 因子6：排名倒挂

**发现来源**：佐加顿斯（瑞超第5）2-3天狼星（瑞超第1）。天狼星排名第1但客场作战，市场仍开出佐加顿斯主胜低赔（2.16），主胜概率仅44% (<50%)，实际客队天狼星凭借更强的赛季状态客场取胜。

**条件**：
1. 客队联赛排名比主队高出 **3个或更多名次**
2. 市场最热选项（通常为主胜）概率 < 50%（即初赔 > 2.00）
3. 客队近6场状态与主队同等或更好

**阈值**：
- 匹配1条 → 标记为关注
- 匹配2条 → 异常比分中客胜方向的优先级提升
- 匹配3条 → 异常比分必须包含客胜方向的比分

### 因子7：H2H统治力覆盖（反向修正）

**发现来源**：切尔西（英超第10，近10场2胜1平7负）2-1热刺（英超第17）。切尔西状态极差，市场30家升主胜、0家降主胜，但凭借对热刺近10场8胜1平1负的绝对H2H统治力仍2-1取胜。

**条件**：
1. H2H近5场主队胜率 ≥ 60%
2. 百家平均主胜即赔升水 ≥ 8%（市场因近期状态过度看衰主队）
3. 客队近期状态并不显著优于主队（近6场积分差 < 4分），或客队同样状态不佳

**阈值**：
- 匹配2条 → 正常比分第1个应包含主胜方向
- 匹配3条 → 异常比分也必须反向考虑主胜选项

**适用场景**：德比战、历史交锋一边倒的宿敌对决。

### 因子8：「不败韧性」识别

**发现来源**：伯恩茅斯（英超第6，近10场4胜6平0负，平率60%）1-1曼城。

**条件**：
1. 主队近10场**输球场次 ≤ 3** 且 **平局场次 ≥ 5**（平率≥50%）
2. 或客队近10场客场**输球场次 ≤ 2** 且 **平局场次 ≥ 4**
3. 对手（强队）近期存在体能消耗（周中杯赛/欧战，间隔≤4天）或战意减损

**阈值**：
- 触发条件1 → 平局概率上调 30%（×1.30）
- 触发条件1+2 → 平局概率上调 50%（×1.50）
- 触发条件1+3 → 平局比分纳入异常比分第1个优先考虑

### 因子10：多因子联动冷平检测（赛季末段强制检查）

**发现来源**：2026-05-19 伯恩茅斯1-1曼城。触发因子2（战意差）+因子5（平局信号，部分匹配）+因子8（不败韧性）+因子9（争冠无望），四因子叠加导致冷平，而单独看每个因子都不完整。

**核心原则**：赛季末段（最后5轮），多个因子同时触发同一场比赛时，冷平概率**非线性上升**，不能按各因子独立计算。

**联动规则**（赛季末最后5轮时强制检查）：
```
if (赛季末最后5轮) {
  // 检查以下因子组合
  let activated_count = 0;
  
  if (因子2触发) activated_count++;     // 战意差
  if (因子8条件1触发) activated_count++; // 主队不败韧性(平率≥50%)
  if (因子9触发) activated_count++;     // 强队争冠无望
  if (因子5匹配≥2条) activated_count++; // 平局信号部分匹配
  
  if (activated_count ≥ 2) {
    平局概率上调 30%（×1.30）           // 2因子联动
  }
  if (activated_count ≥ 3) {
    平局概率上调 50%（×1.50）           // 3+因子联动
    异常比分第1个必须包含平局比分
  }
  if (激活因子包含 因子2+因子8+因子9 三者) {
    无论因子5是否完整匹配，平局权重直接+2级
    // 案例：伯恩茅斯1-1曼城 — 因子5仅匹配2/4条，但因2+8+9联动仍打出冷平
  }
}
```

**适用联赛**：五大联赛赛季末段（最后5轮），尤其适用于英超、意甲、法甲。

### 因子11：弱队主场进球下限修正（主场进球地板）

**发现来源**：2026-05-20复盘发现5场比分差1球，共同特征是弱队（受让方）在主场的实际进球比模型预测多1球。

**验证案例（2026-05-20，5场全部适用）**：
| 场次 | 原预测 | 实际 | 修正后 | 验证 |
|:---:|:------|:---:|:------|:---:|
| 3001 坦山猫1-3国际图尔 | 1-2 | 1-3 | 1-3 ✅ | 弱队主场进1球≥λ下限1.0 |
| 3002 斯达1-4博德闪耀 | 0-3 | 1-4 | 1-4 ✅ | 弱队主场进1球≥λ下限1.0 |
| 3003 库奥皮奥3-0雅罗 | 2-0 | 3-0 | — | 主队非弱队（强队），不适用 |
| 3004 盖斯2-0哈马比 | 2-1 | 2-0 | — | 主队非弱队（主胜方向正确），不适用 |
| 3008 弗赖堡0-3维拉 | 0-2 | 0-3 | 0-3 ✅ | 主队虽非弱队但受让方 |

**核心规律**：即便是联赛垫底的弱旅，在**主场**也有稳定的"进球地板"。原因是：
- 主场观众压力下，弱队至少会有1-2次有威胁进攻
- 弱队主场并非完全龟缩，定位球/反击偶有收获
- 客场强队领先后的保守心态（60分钟后收兵）给主队留出进球窗口

**条件**：
1. 主队联赛排名比客队低 **8个或更多名次**（即主弱客强格局）
2. 主队主场近6场场均进球 ≥ 0.50（再弱的队主场也有过半场次进球）
3. 模型计算的λ1（主队预期进球）< 1.20

**阈值**：
```python
if (条件1+条件2+条件3均满足) {
  // 主队(弱队)主场进球下限修正
  λ1 修正 = max(λ1原值, 1.0)  // 弱队主场至少保底1球
  
  // 修正后比分预测示例
  // 原预测 0-2 → 修正为 1-2（弱队主场进1球）
  // 原预测 0-3 → 修正为 1-3
  // 原预测 1-1 → 维持（已≥1球）或可上调至2-1
  // 原预测 0-0 → 修正为 1-1（主队主场进1球）
}
```

**不适用场景**：
- 德比战（双方都不会收兵，主队进球数可能更高而非更低）
- 杯赛淘汰赛（实力差距太大时可能0进球）
- 极端联赛如法甲（弱队主场进球率显著低于其他联赛）

### 因子12：「共识陷阱」（一致降赔 → 造热预警 · 🔶观测中4/4）

- A：初赔均衡→赛前被一致压低为深盘

- C：排名差≤3名 且 实力差≤10分

**阈值**：触发→正路-1★，平局+25%。A+B或A+C同时触发→正路-2★，平局+40%，异常比分必须含平局。

**失效条件**：实力差>30分 / 豪门主场（皇马/拜仁/曼城）/ 初赔即赔差<2%
