# SP赔率对齐与推荐策略

## 目的

在完成λ模型计算和双轨比分预测后，将预测结果与**官方竞彩SP赔率**对齐，生成可投注的行动建议。目标是**命中率优先**。

## 一、数据来源

用户通常以表格形式提供竞彩官方比分选项及SP赔率：

```
联赛 日期 场次
主队 VS 客队
比分A(SP) 比分B(SP) 比分C(SP) 比分D(SP)
```

每场4个比分选项，SP赔率越高越难打出。

## 二、预测对齐流程

### Step 1: 完成λ模型预测

按照 analysis-framework.md 正常执行：

- 正常比分2个（最符合实力逻辑的赛果）
- 异常比分2个（爆冷或极端赛果）

### Step 2: 与官方SP选项逐一比对

对每个预测比分，检查是否在官方4个选项中：

| 状态 | 含义 | 后续操作 |
|:----|:-----|:--------|
| ✅ 完全命中 | 预测比分在官方选项中 | 直接推荐该选项及其SP |
| ⚠️ 接近命中 | 预测比分与某选项差±1球 | 可微调后推荐（或同时展示原预测和官方选项） |
| ❌ 未命中 | 预测比分不在官方选项中 | 重新审视：差距不大可微调，差距大则放弃 |

### Step 3: 生成行动建议

**直接选取原则**（命中率优先）：

1. 官方的4个选项中如果有**平局比分**且SP≤6.0 → 平局是机构信号，必须重视
2. λ差值大的正路比赛 → 优先选SP中等（4-10）的选项，不追求最高SP
3. 爆冷比赛 → 优先选机构开出的冷门选项，而非自创极端比分

**微调规则**（当预测比分不在选项中时）：

| 原始预测 | 偏离选项 | 微调方式 | 示例 |
|:--------|:--------|:--------|:----|
| 3-1 | 有4-2,有3-0,无3-1 | 3-1→3-0（减1球） | 保留胜者方向，微调失球 |
| 2-1 | 有2-0,有1-1,无2-1 | 2-1→2-0（调失球） 或 2-1→1-1（调进球减1） | 保留胜/平方向 |
| 1-0 | 有2-0,无1-0 | 1-0→2-0（加1球） | 强化胜者方向 |
| 2-2 | 有1-2,有2-1,无2-2 | 2-2→1-1（各减1球） | 保留平局方向 |

**三原则**：
1. **保留方向**：不要翻转主客胜方向（如2-1→1-2是禁止的）
2. **单变量调整**：每次只改一支球队的进球数
3. **幅度≤1球**：从不在1球之外做调整

### Step 4: 输出格式

每场输出：

```
场次号 主队 VS 客队（联赛）

📊 预测 vs SP对照：
[正常比分] → 官方选项: 比分X(SP) ✅ 直接推荐
[异常比分] → 官方选项: 比分Y(SP) ⬆ 微调+1球推荐

🎯 推荐：比分X(SP) + 比分Y(SP) 双选
🎯 单选：比分X(SP) — [推荐理由，≤20字]
```

## 三、串关方案生成

### 3.1 场次筛选优先级

从多场已分析的比赛中挑选串关场次：

1. ⭐⭐⭐⭐⭐ λ差>1.5 + 欧指一致 + 亚指盘口稳定 → 最稳胆材
2. ⭐⭐⭐⭐ λ差1.0-1.5 + 正路结构 → 可选胆材
3. ⭐⭐⭐ 势均力敌但有明确方向 → 仅用于双选覆盖
4. ⚠️ 高不确定性 → 跳过

### 3.2 双选有效赔率计算

```
双选有效赔率 = 1 / (1/SP_A + 1/SP_B)
```

示例：2:0(4.2) + 3:0(7.6) = 1 / (1/4.2 + 1/7.6) = 2.70

### 3.3 串关结构

| 方案 | 结构 | 适合场景 |
|:----|:----|:--------|
| 4串1双选 | 4场×双选=16注 | 常规推荐 |
| 2串1爆冷 | 2场×单挑=1注 | 高SP搏冷 |
| 3串1单选+双选 | 2胆单选+1场双选=2注 | 胆材明确时 |

### 3.4 串关输出格式

```
## 🏆 推荐方案：X串1（双选/单选）

| 场次 | 对阵 | 推荐比分 | SP |
|:---:|:----|:--------|:--:|
| ① | ... | 比分A + 比分B | 有效SP≈X.XX |

💡 选入理由：
✅ ① — [≤25字]
✅ ② — [≤25字]

📈 最佳路径：A×B×C×D ≈ X倍
📊 保底覆盖：各场第一选项组合 ≈ X倍
```

### 3.5 5串1稳定推荐单结构（竞足每日预测 Cron 输出）

> 此结构为`cd0456f2c803`竞足每日预测任务（每日20:00执行）的尾部必输板块。所有逐场分析完成后，末尾统一输出。

**固定5串1结构：**

| 位置 | 类型 | 场次 | 说明 |
|:---:|:----|:---:|:-----|
| 🎯 第1场 | **进球数（3选项）** | 1场 | 大球盘口明确、双方不保守、近6场均进球≥1.5 |
| 🎯 第2场 | **双选（平 + 让平）** | 1场 | 实力接近、赢球至多1球、欧指平赔集中 |
| 🎯 第3场 | **双选（平 + 让平）** | 1场 | 同上条件 |
| 🎯 第4场 | **单选（稳胆）** | 1场 | 信心≥70%、SP≥1.60、无4b/12反向信号 |
| 🎯 第5场 | **单选（稳胆）** | 1场 | 同上条件 |

**选场原则：**
- 进球数场 → 选全场大2.5/3球盘口明确、双方攻击力充足的场次
- 双选场(平+让平) → 选实力均衡、最多赢1球差、欧指平赔集中的场次（覆盖平局+1球小胜）
- 单选场(稳胆) → 选方向最清晰、4b/12无反向信号、亚指盘口稳定的场次
- 5串1容错0 → 宁保守勿冒险，不选信号模糊/因子冲突的场次

**赔率标注：** 每场从欧指Tab百家即赔中提取即时SP

**避免的错误：**
- ❌ 进球数场选保守队（双方龟缩=小球概率大）
- ❌ 双选场选实力悬殊（赢2+球让平失效，只剩平局）
- ❌ 单选场选SP<1.60（赔率太低不值得串）
- ❌ 选4b/12信号冲突的场次进任何位置

### 3.6 让球高赔奖金提升策略（加分项）

> 当用户提出具体奖金目标（如"奖金500以上"），且标准胜平负赔率无法达标时，使用此策略。

**基本原则**：对高确信度的场次，用让球(-1)等价选项替代标准胜平负，系统性提升串关赔率。

**让球替代映射表**：

| 原始方向 | 让球(-1)等价选项 | 赔率提升倍数 | 适用条件 |
|:--------|:----------------|:-----------:|:--------|
| 胜(1.20-1.50) | (-1)平(3.00-3.80) | 2.0-3.0× | 实力碾压+主场优势，赢1球概率>60% |
| 胜(1.50-1.80) | (-1)胜(2.80-3.80) | 1.6-2.2× | 主队强且攻击力好，赢2球概率高 |
| 负(2.50-3.50) | (-1)负(1.40-1.90) | 0.5-0.7× | 市场极端倒向客队（30升0降等信号），赔率反而更低，不建议 |
| 负(3.00-4.50) | 标准负(3.00-4.50) | 原赔率使用 | 客队受让时让球负赔率反而更低，保持标准负 |

**串关奖金计算方法**：

```
总注数 = 各场选项数乘积
投资额 = 总注数 × 2元
最低奖金 = 2元 × (各场最低赔率乘积)   ← 保底回报
最高奖金 = 2元 × (各场最高赔率乘积)   ← 票面显示
```

**常见结构奖金测算**（参考值）：

| 串关结构 | 注数 | 档位 | 让球前 | 让球后(1-2场切换) |
|:--------|:---:|:----|:-----:|:----------------:|
| 5串1·2双+2单+3进球 | 12注 | 最低奖金 | ~90元 | ~200-350元 |
| 5串1·2双+2单+3进球 | 12注 | 最高奖金 | ~300元 | **~500-1010元** ✅ |
| 4串1·3双+1单 | 8注 | 最高奖金 | ~200元 | ~350-600元 |
| 4串1·全单 | 1注 | 单注 | ~60元 | ~180-350元 |

**实战步骤**：

1. 先用标准胜平负构建串关，计算最低/最高奖金
2. 若最高奖金 < 用户目标 → 对高确信度场次切换让球选项
3. 优先切换的主场强队（如基多体大1.24→(-1)平3.40）
4. 次选切换信号明确的方向（如30升0降→标准负3.40）
5. 重新计算，直到最高奖金 ≥ 用户目标

**禁止事项**：
- ❌ 为追求高赔牺牲命中率 — 只切换高确信度场次
- ❌ 对势均力敌的比赛使用让球
- ❌ 双选场次切换让球后选择变窄 — 双选仍应保留两个选项
- ❌ 让球(-2)及以上 — 过于激进，不适合串关策略

**切换优先级**（按可靠性排序）：
1. ⭐⭐⭐ 主胜1.20-1.45区间 → (-1)平（实力碾压确认）
2. ⭐⭐⭐ 主胜1.45-1.70区间 → (-1)胜（主场优势+攻击力确认）
3. ⭐⭐ 市场极端信号（因子4b）→ 标准负（不切换让球）
4. ⭐ 平局信号强 → 保持标准平局（让球平不匹配）

## 四、常见陷阱

1. ❌ 追求最高SP组合忽视命中率 — 优先保证至少中1注
2. ❌ 跨联赛混搭不考虑时差 — 凌晨比赛结果已知后再选后续场次
3. ❌ 使用自创的极端比分（如5-0、6-1）作为双选 — 官方未开出说明概率极低
4. ✅ 命中率优先 -> SP回报次之 -> 组合合理性最后
