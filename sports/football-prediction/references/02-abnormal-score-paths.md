## 异常比分双路径模型（v3.6 新增）

> **学术基础**：Dixon & Coles (1997) τ修正 → 低比分比独立泊松预测更多；Karlis & Ntzoufras (2003/2009) 双变量泊松 → 进球相关性ρ；零膨胀模型 → 0-0概率被独立泊松低估；Cox过程 → 进球强度随比赛状态变化；市场效率理论 → 因子4b/12/13。

### 核心区分：两套完全不同的生成机制

| 维度 | 🔴 爆冷路径（Underdog） | 🟠 大比分路径（Goal Explosion） |
|:--|:--|:--|
| **定义** | 弱队不败/取胜，预期方向被颠覆 | 总进球≥4，远超泊松λ预期 |
| **泊松表现** | λ差被反转（强队λ被压制） | 双λ同时高→方差膨胀 |
| **赔率信号** | 因子4b一致升赔 | 大小球升盘+赔率无明确方向 |
| **关键驱动** | 信息不对称+市场过度反应 | 进攻状态叠加+防守双漏洞 |
| **常见比分** | 1-0, 0-1, 1-1, 0-0 | 2-2, 3-2, 2-3, 3-3, 4-1 |
| **输出要求** | 必须标注「🔴 爆冷：X型」 | 必须标注「🟠 大比分：X型」 |

---

### 🔴 爆冷四型

#### A1·赔率共识反转（最可靠，验证11/11）

```
学术基础：市场效率理论 — 当一致升赔信号出现时，
市场共识方向被机构集体否定，共识方向大概率打不出。

触发条件：
  任意方向(主胜/平/客胜) ≥20家一致升赔 且 ≤2家降赔
  且升赔方向 = 市场最热方向（隐含概率>40%）

比分模板（若升主胜→走客胜/平局方向）：
  主队让球：0-1, 1-1, 0-2
  客队让球：1-0, 1-1, 2-1
  双方平手：0-1, 1-1

异常比分输出标签：🔴 爆冷A1·赔率反转
```

#### A2·近期状态背离（均值回归）

```
学术基础：行为经济学锚定效应 — 市场对近期极端状态
过度反应，连胜队被高估，连败队被低估。

触发条件：任一方近6场 全胜 或 全败

比分模板：
  连败队主场：1-0, 1-1, 2-1  （弱队反弹）
  连胜队客场：1-1, 0-0, 1-0  （强队受阻）

异常比分输出标签：🔴 爆冷A2·状态背离
```

#### A3·体能悬崖

```
学术基础：运动科学 — 72h内双赛+高消耗使实力差缩小30-40%。

触发条件：(间隔≤72h) + (上场消耗大/加时120分钟/飞行>3h)

比分模板：
  体能劣势方主场：0-0, 1-1
  体能劣势方客场：1-0, 2-1（对方赢球但不穿盘）

异常比分输出标签：🔴 爆冷A3·体能悬崖
```

#### A4·关键缺阵未被消化

```
学术基础：信息不对称理论 — 市场未能充分定价核心球员缺阵的影响。

触发条件：头号射手/主力门将/防线核心缺阵 + 赔率变化<5%

比分模板：
  缺阵方若为上盘(让球方)：0-1, 1-1, 0-0
  缺阵方若为下盘(受让方)：1-0, 2-0（对方的冷门被放大）

异常比分输出标签：🔴 爆冷A4·关键缺阵
```

### 🟠 大比分四型

#### B1·对攻结构（双λ膨胀）

```
学术基础：Karlis-Ntzoufras 双变量泊松 — 当两队λ同时高时，
进球正相关ρ>0，总进球方差远超独立泊松预测。

触发条件：λ1>1.5 AND λ2>1.2
  AND 双方近3场场均总进球>3.0

比分模板：2-2, 3-2, 2-3, 3-3

异常比分输出标签：🟠 大比分B1·对攻结构
```

#### B2·绝望攻势

```
学术基础：博弈论ALL-IN策略 — 落后方必须进球时，
全线压上导致防守真空，双方都容易进球。

触发条件：保级队vs已降级队 / 末轮必须赢N球

比分模板：3-1, 4-1, 2-3

异常比分输出标签：🟠 大比分B2·绝望攻势
```

#### B3·德比/死敌

```
学术基础：情绪经济学 — 德比战中红黄牌+失误率显著上升，
创造更多进球机会和定位球。

触发条件：同城德比/百年宿敌
  额外信号：历史交锋场均>3球

比分模板：2-2, 3-2, 2-3

异常比分输出标签：🟠 大比分B3·德比情绪
```

#### B4·防守双漏洞

```
学术基础：泊松叠加原理 — 双方防线都漏时，
进球事件非独立，总进球>两λ之和。

触发条件：双方近6场场均失球均>1.8

比分模板：2-1, 3-1, 2-2

异常比分输出标签：🟠 大比分B4·防守漏洞
```

### 异常比分输出铁规

**每场异常比分必须：**
1. 明确标注类型标签：`🔴 爆冷A1·赔率反转` 或 `🟠 大比分B1·对攻结构`
2. 至少有一个比分来自对应类型的比分模板
3. 逻辑行写明触发条件，不可空泛写"可能爆冷"

**输出格式升级：**
```
🔴 **第二组（异常比分）：[比分1] / [比分2]**
🏷️ 类型：[A1·赔率反转 / B1·对攻结构 ...]
👉 触发：[具体触发条件，≤30字]
```

### ⬆️ H2H历史常见比分参考（2026-05-28新增）

H2H历史交锋数据是推测异常比分的重要补充参考。当爆冷/大比分路径的模板比分与H2H常见比分高度重合时，可信度显著提升。

**H2H数据来源：** 详情页战绩Tab中"两队交锋"段落，从 `document.body.innerText` 中提取历史比分。

```python
# H2H常见比分提取
import re
from collections import Counter

def extract_h2h_common_scores(inner_text: str) -> list[str]:
    idx = inner_text.find("两队交锋")
    if idx < 0:
        return []
    h2h_section = inner_text[idx:idx+500]
    scores = re.findall(r'\b(\d+)-(\d+)\b', h2h_section)
    score_counter = Counter()
    for (h, a) in scores:
        if int(h) + int(a) <= 7:
            score_counter[f"{h}-{a}"] += 1
    return [s for s, _ in score_counter.most_common(3)]
```

**H2H应用规则：**
1. **优先级**：H2H高频比分（出现≥2次）> 模板默认比分 > H2H低频比分
2. **H2H确认**：异常比分推荐在H2H历史中出现过 → 标注 `✅ H2H历史印证`
3. **H2H补充**：H2H高频比分(top2)不在异常模板中，且类型不矛盾 → 额外列出一个 `📖 历史常见`
4. **H2H全覆盖**：H2H最高频2个比分与异常模板完全匹配 → 标记 `🎯 H2H高度吻合`
5. **无数据**：无H2H数据或出现0次 → 正常输出模板比分，不标注

**输出示例：**
```
🎯 异常比分：0-1/1-1 🏷️ A1·赔率反转 ✅ H2H历史印证
```
或（有H2H补充比分时）：
```
🎯 异常比分：0-1/1-1/📖0-0 🏷️ A1·赔率反转 ✅ H2H历史印证
```

**H2H补充比分类型约束：**
- ❌ 禁止H2H补充比分与异常类型矛盾（A型爆冷不能加4-4、B型大比分不能加0-0）

**禁止项：**
- ❌ 禁止爆冷和大比分混在同一组（不能1-0和4-1放一起）
- ❌ 禁止不标类型标签
- ❌ 禁止只用"可能爆冷"空泛表述
- ✅ 若同时触发爆冷型和大比分型 → 爆冷型优先（信息量更大）
- ✅ 若只触发一种 → 只输出那种，另一组正常比分不变
