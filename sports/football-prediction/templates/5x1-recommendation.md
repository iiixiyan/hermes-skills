# 竞足5串1推荐单输出模板

## 使用场景
竞足每日cron任务（cd0456f2c803）末尾统一输出时使用。

## 结构
1场进球数（3选项）+ 2场双选（平+让平）+ 2场单选（稳胆≥1.60）

## ⚠️ 铁律：5场必须来自5场不同比赛
- 进球数场(1场) + 双选场(2场) + 单选场(2场) = **5场完全不同的比赛**
- 同一场次编号出现2次=无效方案，必须换场
- 例：3005出现在进球数，就不能再出现在单选或双选里
- 推荐方案：从7场中剔除2场信心最低的，剩余5场各司其职

## 输出格式
```
══════════════════════════════════════
🏆 今日5串1稳定推荐单 🏆
══════════════════════════════════════

🎯 进球数（3选项）
〖竞足-XXXX〗 [主队] VS [客队]
总进球：A球 / B球 / C球 → [选场理由]

🎯 双选（平 + 让平）
〖竞足-XXXX〗 [主队] VS [客队]
双选：平 @X.XX / 让平 @X.XX → [选场理由]

〖竞足-XXXX〗 [主队] VS [客队]
双选：平 @X.XX / 让平 @X.XX → [选场理由]

🎯 单选（稳胆）
〖竞足-XXXX〗 [主队] VS [客队]
单选：[胜/平/负] @X.XX | 让球：[让选项] @X.XX → [选场理由]

〖竞足-XXXX〗 [主队] VS [客队]
单选：[胜/平/负] @X.XX | 让球：[让选项] @X.XX → [选场理由]

───────────────────────────
💰 理论奖金估算：约X倍
📊 选场逻辑：[简要说明]
```

## 选场原则
- **进球数场** → 大球盘口明确（全场大2.5/3盘）、双方不保守、近6场均有球
- **双选场(平+让平)** → 实力接近、赢球至多1球、欧指平赔集中（覆盖平局+1球小胜）
- **单选场(稳胆)** → 信心≥70%、SP≥1.60、无4b/12反向信号、方向最清晰
- **5串1容错0** → 宁保守勿冒险，不用信号模糊的场次

## 每场必须标注
- ✅ 即时SP赔率（从欧指Tab百家即赔提取）
- ✅ 胜平负方向 + 让球方向
- ✅ 选场理由（为什么选这场不选那场）

## 禁止
- ❌ 同一场次重复出现在两个不同位置
- ❌ 不选信号模糊/因子冲突的场次
- ❌ 不出现"仅供参考"等免责话
